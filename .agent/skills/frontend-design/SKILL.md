# Frontend Design Skill

프리미엄 프론트엔드 UI 디자인 가이드라인.

## Design Principles
- **Typography**: 독특한 폰트 선택 (Arial/Inter/Roboto 지양)
- **Color**: CSS 변수 활용, 일관된 색상 시스템
- **Motion**: 스태거드 애니메이션, 스크롤 트리거
- **Spatial**: 비대칭, 오버랩, 그리드 브레이킹

## Anti-Patterns (AVOID)
- 제네릭 폰트 (Inter, Roboto, Arial)
- 뻔한 퍼플 그라디언트
- 예측 가능한 레이아웃
