도형 1 — 피라미드

피라미드는 가운데 정렬된 삼각형입니다. 각 줄의 별 개수가 1, 3, 5, 7, 9처럼 홀수로 증가합니다.

Cpyramid.c
1#include <stdio.h>
2
3int main(void) {
4 printf(" *\n");
5 printf(" ***\n");
6 printf(" *****\n");
7 printf(" *******\n");
8 printf("*********\n");
9 return 0;
10}
실행 결과
줄(i)공백별규칙
141공백: 5-i, 별: 2i-1
233공백: 5-i, 별: 2i-1
325공백: 5-i, 별: 2i-1
417공백: 5-i, 별: 2i-1
509공백: 5-i, 별: 2i-1
피라미드 패턴 n줄 피라미드에서 i번째 줄:

- 공백: (n - i)개
- 별: (2i - 1)개
- 공백 + 별 = n - i + 2i - 1 = n + i - 1 (줄마다 다름)
- 총 너비: 2n - 1 (가장 넓은 줄 기준)

별이 홀수 개(1, 3, 5, ...)씩 증가하는 것이 핵심입니다.

도형 3 — 다이아몬드

다이아몬드는 피라미드 위에 역피라미드를 붙인 것입니다. 단, 가장 넓은 줄이 중복되지 않도록 합니다.

Cdiamond.c
1#include <stdio.h>
2
3int main(void) {
4 /* 상반부: 피라미드 */
5 printf(" *\n");
6 printf(" ***\n");
7 printf(" *****\n");
8 printf("*******\n");
9 /* 하반부: 역피라미드 (가장 넓은 줄 제외) */
10 printf(" *****\n");
11 printf(" ***\n");
12 printf(" *\n");
13 return 0;
14}

다이아몬드 = 피라미드(4줄) + 역피라미드(3줄, 첫 줄 제외)

복잡한 도형도 이미 알고 있는 단순한 패턴으로 분해하면 쉽게 작성할 수 있습니다.

10축 매핑

이번 유닛에서 배운 내용을 10개의 학습 축으로 연결합니다.

축 1: C언어 [필수] 이 유닛의 C 활용: printf로 피라미드, 다이아몬드, 하트 등 복합 도형 출력
깊이: LAYER 1 — 새 문법 없이 printf와 공백/별로 복잡한 패턴 생성
핵심: 피라미드 공식: 공백 (n-i)개, 별 (2i-1)개
축 2: 컴퓨팅 사고력 [필수] 사고 패턴: 분해(Decomposition)
복합 도형을 단순한 부분(피라미드, 역피라미드, 삼각형)으로 분해했습니다.

다이아몬드 = 피라미드 + 역피라미드
하트 = 불규칙 윗부분 + 역피라미드 아랫부분

분해는 모든 복잡한 문제 해결의 핵심 기법입니다.
축 3: 디버깅 [필수] 이 유닛의 에러:
1. 피라미드 첫 줄 공백 수 실수 (n-1개여야 함)
2. 다이아몬드 중간 줄 중복
3. 하트 윗부분 공백 실수
4. 피라미드(홀수증가)와 삼각형(1씩증가) 혼동
5. 줄 끝 불필요한 공백

검증: 출력을 기대 결과와 한 줄씩 비교
축 4: KOI — 정보올림피아드 [필수] 난이도: TIER 1-2 (출력 패턴)

- 피라미드 공식 (n-i)개 공백 + (2i-1)개 별은 암기
축 5: 코딩 습관 [선택] LAYER 1 — 도형 분석 습관:
- 코드 전에 종이에 표를 먼저 그립니다
