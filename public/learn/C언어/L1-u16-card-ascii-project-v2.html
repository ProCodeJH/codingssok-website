프로젝트 개요

이번 프로젝트는 Layer 1의 최종 프로젝트입니다. 지금까지 배운 모든 것을 활용하여 두 가지를 만듭니다.

프로젝트 구성 Part A: 나만의 명함
- 테두리가 있는 명함 형태
- 이름, 소속, 연락처, 좌우명 등 개인 정보
- 이스케이프 시퀀스 활용 (\t, \", \\)

Part B: ASCII 아트
- 자유 주제의 그림 (동물, 건물, 로고 등)
- 공백과 특수문자 조합
- 최소 10줄 이상

TIP: 칸 수 맞추기 명함의 모든 줄에서 | 의 위치가 같아야 상자가 깔끔합니다.

개발 흐름 — 전체 과정

1
1. 기획
명함에 넣을 정보와 ASCII 아트 주제를 정합니다.
2
2. 설계
종이에 명함 레이아웃과 ASCII 아트를 그립니다. 각 줄의 문자를 정확히 적습니다.
3
3. 뼈대 작성
#include, main, return 0을 먼저 쓰고 컴파일합니다.
4
4. 명함 구현
printf를 한 줄씩 추가하며 테두리와 정보를 채웁니다. 3~4줄마다 컴파일합니다.
5
5. ASCII 아트 구현
설계한 그림을 printf로 옮깁니다. 역슬래시와 큰따옴표에 주의합니다.
6
6. 테스트
gcc -Wall로 경고 0개를 확인하고, 출력이 설계와 일치하는지 비교합니다.
7
7. 개선
정렬을 다듬고, 주석을 추가하고, 코드 스타일을 점검합니다.

축 9: AI 연결 [선택] AI에게 ASCII 아트 만들기:
ChatGPT 등 AI에게 "고양이 ASCII 아트를 만들어줘"라고 요청하면 결과를 얻을 수 있습니다. 하지만 그것을 C 코드의 printf 안에 넣으려면 역슬래시와 큰따옴표를 모두 이스케이프해야 합니다.

이 이스케이프 작업은 AI가 만들어준 결과를 이해하고 수정하는 능력이 필요합니다.
