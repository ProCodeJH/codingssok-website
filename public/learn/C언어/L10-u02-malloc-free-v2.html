malloc과 free

malloc으로 힙 메모리를 할당하고 free로 반환하는 기본 패턴을 배우고, 동적 배열과 동적 구조체를 활용합니다.

이 유닛에서 배우는 것

  • malloc의 사용법과 반환값을 이해합니다.
  • free로 메모리를 반환하는 습관을 익힙니다.
  • 동적 배열을 할당하고 사용할 수 있습니다.
  • 동적 구조체를 할당하고 화살표 연산자로 접근할 수 있습니다.

DEBUG LAB — 에러를 찾으시오

에러:

수정:

malloc 5대 규칙
1. 반환값 NULL 검사: if (p == NULL) 처리
2. sizeof 사용: malloc(n * sizeof(int)) — 하드코딩 금지
3. 사용 후 free: 메모리 누수 방지
4. free 후 NULL 대입: p = NULL; — 댕글링 포인터 방지
5. 이중 free 금지: 같은 포인터를 두 번 free하면 미정의 동작

에러 체험

C
1// 에러 1: NULL 검사 없이 사용
2int *p = (int*)malloc(1000000000 * sizeof(int));
3p[0] = 10; // 할당 실패 시 NULL → 세그폴트!
4// 수정: if (p == NULL) { 에러 처리; }
5
6// 에러 2: free를 빠뜨림 (메모리 누수)
7void leak(void) {
8 int *p = (int*)malloc(100 * sizeof(int));
9 /* free(p); 없음 → 매 호출마다 메모리 누수 */
10}
11// 수정: 함수 끝에 free(p);
12
13// 에러 3: free 후 사용 (댕글링 포인터)
14int *p = (int*)malloc(sizeof(int));
15*p = 42;
16free(p);
17printf("%d\n", *p); // 이미 해제된 메모리 접근!
18// 수정: free(p); p = NULL;
19
20// 에러 4: 이중 free
21free(p);
22free(p); // 미정의 동작 → 크래시 가능
23// 수정: free(p); p = NULL; — NULL free는 안전

대회 기출

요약

핵심 정리
1. malloc(size): size 바이트 할당, void* 반환, 실패 시 NULL
2. free(p): 할당된 메모리 반환
3. 패턴: 할당 → NULL 검사 → 사용 → free → p=NULL
4. 동적 배열: int *arr = (int*)malloc(n * sizeof(int));
5. 동적 구조체: Student *sp = (Student*)malloc(sizeof(Student));
6. 5대 규칙: NULL 검사, sizeof 사용, free 필수, NULL 대입, 이중 free 금지

다음 유닛으로

다음: calloc, realloc — 더 편한 할당
malloc의 기본을 마스터했습니다. 다음 유닛에서는 0으로 초기화하는 calloc과, 크기를 변경하는 realloc을 배웁니다.
