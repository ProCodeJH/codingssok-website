에러:

수정:

컴파일 명령어
한 번에 컴파일:
gcc main.c student.c -o program

분리 컴파일 (대규모 프로젝트):
gcc -c main.c → main.o
gcc -c student.c → student.o
gcc main.o student.o -o program

장점: student.c만 수정하면 student.o만 재컴파일

대회 기출

요약

핵심 정리
1. 모듈 = .h(선언) + .c(정의)
2. .h: 함수 선언, typedef, 매크로, extern 변수
3. .c: 함수 본문, 전역 변수 정의, static 함수
4. 컴파일: gcc main.c module.c -o program
5. 분리 컴파일: gcc -c → gcc .o .o -o
6. 규칙: .c는 자신의 .h를 #include, 모든 .h에 헤더 가드

다음 유닛으로

다음: static과 extern 키워드
멀티파일 구조를 배웠습니다. 다음 유닛에서는 static과 extern 키워드로 변수와 함수의 가시성을 제어하는 방법을 배웁니다.
