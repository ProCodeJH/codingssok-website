мҡ”м•Ҫ

н•өмӢ¬ м •лҰ¬
1. fopen("нҢҢмқјлӘ…", "w"): м“°кё° лӘЁл“ңлЎң нҢҢмқј м—ҙкё° (кё°мЎҙ лӮҙмҡ© мӮӯм ң)

3. fprintf(fp, м„ңмӢқ, ...): нҢҢмқјм—җ м„ңмӢқнҷ”лҗң лҚ°мқҙн„° м“°кё°
4. fputs(л¬ёмһҗм—ҙ, fp): нҢҢмқјм—җ л¬ёмһҗм—ҙ м“°кё°
5. fputc(л¬ёмһҗ, fp): нҢҢмқјм—җ л¬ёмһҗ н•ҳлӮҳ м“°кё°
6. fclose(fp): нҢҢмқј лӢ«кё° (лІ„нҚј л№„мҡ°кё° + мһҗмӣҗ н•ҙм ң)
7. NULL мІҙнҒ¬ н•„мҲҳ: if (fp == NULL) { perror(...); return 1; }

лӢӨмқҢ: нҢҢмқј мқҪкё° вҖ” fscanf, fgets
нҢҢмқјм—җ лҚ°мқҙн„°лҘј м“°лҠ” л°©лІ•мқ„ л°°мӣ мҠөлӢҲлӢӨ. лӢӨмқҢ мң лӢӣм—җм„ңлҠ” нҢҢмқјм—җм„ң лҚ°мқҙн„°лҘј мқҪм–ҙмҳӨлҠ” fscanfмҷҖ fgetsлҘј н•ҷмҠөн•©лӢҲлӢӨ.

fprintfнҢҢмқј нҸ¬мқён„°м—җ м„ңмӢқнҷ”лҗң лҚ°мқҙн„°лҘј м“°лҠ” н•ЁмҲҳ. printfмқҳ нҢҢмқј лІ„м „мһ…лӢҲлӢӨ.
fcloseм—ҙлҰ° нҢҢмқјмқ„ лӢ«лҠ” н•ЁмҲҳ. лІ„нҚјлҘј л№„мҡ°кі  мӢңмҠӨн…ң мһҗмӣҗмқ„ н•ҙм ңн•©лӢҲлӢӨ.
fputsл¬ёмһҗм—ҙмқ„ нҢҢмқјм—җ м“°лҠ” н•ЁмҲҳ. к°ңн–ү л¬ёмһҗлҘј мһҗлҸҷмңјлЎң м¶”к°Җн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.
fputcл¬ёмһҗ н•ҳлӮҳлҘј нҢҢмқјм—җ м“°лҠ” н•ЁмҲҳ.
м“°кё° лӘЁл“ң ("w")нҢҢмқјмқ„ м“°кё° м „мҡ©мңјлЎң м—¬лҠ” лӘЁл“ң. нҢҢмқјмқҙ м—Ҷмңјл©ҙ мғҲлЎң мғқм„ұн•ҳкі , мһҲмңјл©ҙ лӮҙмҡ©мқ„ м§ҖмӣҒлӢҲлӢӨ.
