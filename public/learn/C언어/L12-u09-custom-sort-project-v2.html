커스텀 비교 정렬기 -- 프로젝트

qsort와 비교 함수를 활용하여 다양한 기준으로 데이터를 정렬하는 프로그램을 구현합니다.

이 유닛에서 배우는 것

  • 다양한 비교 함수를 작성하여 qsort에 전달할 수 있습니다.
  • 사용자가 선택한 기준으로 데이터를 정렬할 수 있습니다.
  • 다중 기준 정렬(1차 기준이 같으면 2차 기준 적용)을 구현할 수 있습니다.
  • 정렬 결과를 파일에 저장하고 불러올 수 있습니다.

프로젝트 개요

커스텀 비교 정렬기 (Custom Comparison Sorter)
qsort와 비교 함수를 활용하여 사용자가 원하는 기준으로 데이터를 정렬하는 프로그램을 구현합니다.

버전 1: 기본 -- 학생 성적 정렬 (이름순/점수순 선택)
버전 2: 확장 -- 다중 기준 + 오름/내림 선택 + 메뉴
버전 3: 완성 -- CSV 읽기 -> 사용자 선택 기준 정렬 -> CSV 저장 + 비교 함수 배열 활용

5#define MAX_STUDENTS 100
6#define NAME_LEN 30
7
8typedef struct {

13/* 이름 기준 오름차순 비교 함수 */
14int cmpByName(const void *a, const void *b) {
15 const Student *sa = (const Student *)a;
16 const Student *sb = (const Student *)b;
17 return strcmp(sa->name, sb->name);
18}

20/* 점수 기준 내림차순 비교 함수 */
21int cmpByScore(const void *a, const void *b) {
22 const Student *sa = (const Student *)a;
23 const Student *sb = (const Student *)b;
24 return sb->score - sa->score; /* 내림차순: b - a */
25}

35}

78 printStudents(students, count);
79 return 0;

실수 체험

실수 1: 비교 함수 반환값 오류
비교 함수는 반드시 음수, 0, 양수 세 가지 경우를 올바르게 반환해야 합니다.

/* 잘못된 패턴: 0과 1만 반환 */
int cmpWrong(const void *a, const void *b) {
const Student *sa = (const Student *)a;
const Student *sb = (const Student *)b;
return sa->score > sb->score; /* 0 또는 1만 반환! */
}

/* 올바른 패턴: 음수, 0, 양수 반환 */
int cmpCorrect(const void *a, const void *b) {
const Student *sa = (const Student *)a;
const Student *sb = (const Student *)b;
return sa->score - sb->score; /* 음수, 0, 양수 모두 가능 */
}

비교 함수가 0을 반환하지 않으면 같은 값의 정렬 순서가 불안정해지고, qsort가 무한 루프에 빠질 수도 있습니다.
실수 2: 전역 sortOrder 초기화 누락
정렬 방향을 전역 변수로 제어할 때 초기화하지 않으면 예상과 다른 결과가 나옵니다.

/* 잘못된 패턴: sortOrder 설정 전에 qsort 호출 */
int sortOrder; /* 초기화하지 않으면 쓰레기 값! */
qsort(arr, n, sizeof(Student), cmpByTotal);

/* 올바른 패턴: 반드시 초기값 지정 */
int sortOrder = 1; /* 기본값: 오름차순 */

전역 변수는 0으로 초기화되므로 sortOrder가 0이면 모든 비교 결과가 0이 되어 정렬이 되지 않습니다. 반드시 1 또는 -1로 설정해야 합니다.
