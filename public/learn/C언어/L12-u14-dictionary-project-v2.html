프로젝트 개요

영어 사전 (English Dictionary)
정렬과 이진 탐색을 종합 활용하여 영어 단어를 관리하고 빠르게 검색하는 사전 프로그램을 구현합니다.

버전 1: 기본 -- Word 구조체, 배열 입력, qsort 정렬, 이진 탐색
버전 2: 확장 -- 메뉴(추가/검색/목록/삭제), 파일 저장, 접두사 검색, 단어 수정
버전 3: 완성 -- CSV 가져오기, 검색 횟수 통계, 최근 검색 기록, 자동 정렬, 단어 퀴즈 모드

352
353 int imported = 0;

355 wordCount < MAX_WORDS) {
356 line[strcspn(line, "\n")] = '\0';
357
358 char eng[WORD_LEN], kor[MEANING_LEN];
359 char *sep = strchr(line, ',');
360 if (sep == NULL) continue;
361
362 *sep = '\0';
363 strncpy(eng, line, WORD_LEN - 1);
364 eng[WORD_LEN - 1] = '\0';
365 strncpy(kor, sep + 1, MEANING_LEN - 1);
366 kor[MEANING_LEN - 1] = '\0';

368 /* 중복 검사 */
369 if (binarySearch(eng) != -1) continue;
370
371 /* 정렬 유지하며 삽입 */
372 int pos = findInsertPos(eng);
373 for (int i = wordCount; i > pos; i--) {
374 dictionary[i] = dictionary[i - 1];
375 }
376 strcpy(dictionary[pos].english, eng);
377 strcpy(dictionary[pos].korean, kor);
378 dictionary[pos].searchCount = 0;
379 wordCount++;
380 imported++;
381 }
382
383 fclose(fp);
384 printf("%s에서 %d개의 단어를 가져왔습니다. (총 %d개)\n",

핵심 정리

3. 이진 탐색: 정렬된 배열에서 O(log n) 검색, 반드시 정렬 선행 필요
4. lower bound: 이진 탐색 변형, 삽입 위치 및 범위 검색에 활용
5. 접두사 검색: lower bound로 시작 위치를 찾고 연속 일치 항목 출력
6. 정렬 유지 삽입: findInsertPos + 요소 이동으로 정렬 상태 유지
7. 파일 기반 데이터 관리: 프로그램 종료 후에도 데이터 보존
8. 원형 버퍼: 고정 크기에서 최근 N건의 기록을 효율적으로 관리
9. 검색 통계: 검색 횟수를 기록하여 사용 패턴 분석
