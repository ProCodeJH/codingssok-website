특정 위치 삽입 (insertAt)
임의의 위치에 노드를 삽입하려면 다음 절차를 따릅니다.

1단계: 새 노드를 malloc으로 생성합니다.
2단계: 삽입할 위치의 이전 노드(prev)를 순회하여 찾습니다.
3단계: 새 노드의 next를 prev->next로 설정합니다.
4단계: prev->next를 새 노드로 변경합니다.

위치가 0(맨 앞)이면 head를 갱신하는 별도 처리가 필요합니다.
시간 복잡도: 탐색 O(n) + 삽입 O(1) = O(n)
C
1#include <stdio.h>
2#include <stdlib.h>
3
4typedef struct Node {
5 int data;
6 struct Node *next;
7} Node;
8
9/* 특정 위치(pos)에 노드 삽입 (0부터 시작) */
10Node *insert_at(Node *head, int pos, int data) {
11 Node *new_node = (Node *)malloc(sizeof(Node));
12 new_node->data = data;
13
14 /* 맨 앞 삽입 */
15 if (pos == 0) {
16 new_node->next = head;
17 return new_node; /* 새 head 반환 */
18 }
19
20 /* pos-1번째 노드(prev) 찾기 */
21 Node *prev = head;
22 for (int i = 0; i < pos - 1 && prev != NULL; i++) {
23 prev = prev->next;
24 }
25
26 if (prev == NULL) {
27 printf("위치 초과! 삽입 실패\n");
28 free(new_node);
29 return head;
30 }
31
32 /* prev 뒤에 삽입 */
33 new_node->next = prev->next;
34 prev->next = new_node;
35 return head;
36}
37
38void print_list(Node *head) {
39 Node *cur = head;
40 while (cur != NULL) {
41 printf("%d -> ", cur->data);
42 cur = cur->next;
43 }
44 printf("NULL\n");
45}
46
47int main(void) {
48 Node *head = NULL;
49
50 /* 리스트: NULL */
51 head = insert_at(head, 0, 10); /* 10 -> NULL */
52 head = insert_at(head, 1, 30); /* 10 -> 30 -> NULL */
53 head = insert_at(head, 1, 20); /* 10 -> 20 -> 30 -> NULL */
54 head = insert_at(head, 0, 5); /* 5 -> 10 -> 20 -> 30 -> NULL */
55
56 print_list(head);
57 return 0;
58}
값으로 노드 삭제 (deleteNode)
특정 값을 가진 노드를 삭제하려면 다음 절차를 따릅니다.

1단계: head 노드가 삭제 대상인지 확인합니다.
— head 삭제 시 head를 head->next로 변경 후 free합니다.
2단계: 이전 노드(prev)를 추적하며 삭제 대상을 탐색합니다.
3단계: prev->next를 삭제 대상의 next로 변경합니다.
4단계: 삭제 대상을 free합니다.

핵심: 삭제할 노드의 이전 노드를 반드시 알아야 합니다.
head 삭제와 중간 노드 삭제는 처리 방식이 다릅니다.
C
1/* 값(target)을 가진 첫 번째 노드를 삭제 */
2Node *delete_node(Node *head, int target) {
3 if (head == NULL) return NULL;
4
5 /* head가 삭제 대상인 경우 */
6 if (head->data == target) {
7 Node *temp = head;
8 head = head->next;
9 free(temp);
10 return head;
11 }
12
13 /* 중간/끝 노드 삭제: prev 추적 */
14 Node *prev = head;
15 while (prev->next != NULL && prev->next->data != target) {
16 prev = prev->next;
17 }
18
19 if (prev->next != NULL) {
20 Node *temp = prev->next;
21 prev->next = temp->next; /* 이전 노드와 다음 노드 연결 */
22 free(temp); /* 삭제 대상 해제 */
23 } else {
24 printf("%d을(를) 찾을 수 없습니다.\n", target);
25 }
26
27 return head;
28}
값으로 탐색 (search)
연결 리스트에서 특정 값을 찾으려면 head부터 순서대로 순회합니다.

현재 노드의 data가 target과 같으면 해당 노드 포인터를 반환합니다.
cur->next로 이동하며, NULL에 도달하면 탐색 실패입니다.

시간 복잡도: O(n) — 최악의 경우 전체 노드를 순회합니다.
배열의 인덱스 접근 O(1)과 달리, 연결 리스트는 임의 접근이 불가능합니다.
C
1/* 값(target)을 가진 노드를 탐색하여 포인터 반환 */
2Node *search(Node *head, int target) {
3 Node *cur = head;
4 int index = 0;
5
6 while (cur != NULL) {
7 if (cur->data == target) {
8 printf("%d을(를) 인덱스 %d에서 찾았습니다.\n", target, index);
9 return cur;
10 }
11 cur = cur->next;
12 index++;
13 }
14
15 printf("%d을(를) 찾을 수 없습니다.\n", target);
16 return NULL;
17}
전체 메모리 해제 (freeAll)
프로그램 종료 전에 연결 리스트의 모든 노드를 free해야 메모리 누수가 발생하지 않습니다.

절차:
1. 현재 노드(cur)를 head로 설정합니다.
2. cur이 NULL이 아닌 동안 반복합니다.
3. 다음 노드(next)를 미리 저장합니다.
4. cur을 free합니다.
5. cur을 next로 이동합니다.

주의: free(cur) 후에는 cur->next에 접근할 수 없으므로, 반드시 next를 먼저 저장해야 합니다.
C
1/* 연결 리스트의 모든 노드를 해제 */
2void free_all(Node *head) {
3 Node *cur = head;
4 while (cur != NULL) {
5 Node *next = cur->next; /* 다음 노드 미리 저장 */
6 free(cur); /* 현재 노드 해제 */
7 cur = next; /* 다음 노드로 이동 */
8 }
9}
10
11/* 사용 예시 */
12int main(void) {
13 Node *head = NULL;
14 head = insert_at(head, 0, 10);
15 head = insert_at(head, 1, 20);
16 head = insert_at(head, 2, 30);
17
18 print_list(head); /* 10 -> 20 -> 30 -> NULL */
19
20 head = delete_node(head, 20);
21 print_list(head); /* 10 -> 30 -> NULL */
22
23 search(head, 30); /* 30을(를) 인덱스 1에서 찾았습니다. */
24 search(head, 99); /* 99을(를) 찾을 수 없습니다. */
25
26 free_all(head);
27 head = NULL; /* dangling pointer 방지 */
28 return 0;
29}

대회 기출

요약

핵심 정리
1. insertAt: prev 노드 찾기 -> new->next = prev->next -> prev->next = new
2. deleteNode: prev 추적 -> prev->next = target->next -> free(target)
3. head 삭제: head = head->next 후 free (별도 처리 필수)
4. search: head부터 순회하며 data 비교 -> O(n)
5. freeAll: next를 미리 저장한 후 free -> cur = next 반복
6. 삽입/삭제 자체는 O(1)이나 위치 탐색에 O(n) 소요
7. free 후 포인터를 NULL로 설정하여 dangling pointer 방지
8. 배열 대비 장점: 삽입/삭제 시 원소 이동 불필요
