간단한 전화번호 해시

해시 테이블로 전화번호부를 구현하여 O(1) 검색을 체험합니다.

이 유닛에서 배우는 것

  • 해시 함수의 원리를 이해하고 직접 구현할 수 있습니다.
  • 해시 테이블에 데이터를 추가, 검색, 삭제할 수 있습니다.
  • 충돌 해결 기법(체이닝)을 구현하고 시각적으로 출력할 수 있습니다.
  • 동적 리사이징과 성능 측정을 통해 해시 테이블의 효율성을 검증할 수 있습니다.

프로젝트 개요

간단한 전화번호 해시 (Phone Book Hash Table)
해시 테이블을 활용하여 이름으로 전화번호를 O(1)에 검색하는 전화번호부 프로그램을 구현합니다.

버전 1: 기본 -- 이름을 키로 하는 해시 테이블 (추가, 검색, 전체 출력)
버전 2: 확장 -- 삭제, 수정, 충돌 통계, 체이닝 시각 출력
버전 3: 완성 -- 파일 저장/불러오기, 동적 리사이징(적재율 초과 시), 성능 측정(선형탐색 vs 해시 비교)
