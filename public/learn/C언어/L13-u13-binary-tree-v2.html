핵심 개념

트리 용어 정리
루트(Root): 트리의 최상위 노드 (부모가 없음)
부모(Parent): 자식 노드를 가진 노드
자식(Child): 부모에 연결된 하위 노드
리프(Leaf): 자식이 없는 노드 (단말 노드)
깊이(Depth): 루트에서 해당 노드까지의 간선 수 (루트의 깊이 = 0)
높이(Height): 해당 노드에서 가장 깊은 리프까지의 간선 수
형제(Sibling): 같은 부모를 가진 노드
차수(Degree): 한 노드의 자식 수
이진 트리 종류
완전 이진 트리(Complete): 마지막 레벨을 제외하고 모든 레벨이 꽉 차 있으며, 마지막 레벨은 왼쪽부터 채워진 트리
포화 이진 트리(Full/Perfect): 모든 레벨이 완전히 채워진 트리 (노드 수 = 2^h+1 - 1)
편향 이진 트리(Skewed): 모든 노드가 한쪽 자식만 가진 트리 (사실상 연결 리스트)

레벨 순회 (BFS) 개념

undefined

레벨 순회 알고리즘
1. 루트를 큐에 넣습니다.
2. 큐에서 노드를 꺼내 방문(출력)합니다.
3. 꺼낸 노드의 왼쪽 자식이 있으면 큐에 넣습니다.
4. 꺼낸 노드의 오른쪽 자식이 있으면 큐에 넣습니다.
5. 큐가 빌 때까지 2~4를 반복합니다.

에러 체험

에러 1: NULL 체크 누락 (base case 없음)
void preorder(TreeNode *node) {
printf("%d ", node->data); // node가 NULL이면 크래시!
preorder(node->left);
preorder(node->right);
}

-> 반드시 if (node == NULL) return;을 함수 첫 줄에 작성합니다.
-> 이것이 재귀의 종료 조건(base case)입니다.
에러 2: 순회 순서 혼동
// 중위 순회라고 했지만 실제로는 전위 순회
void inorder(TreeNode *node) {
if (node == NULL) return;
printf("%d ", node->data); // 출력이 먼저 -> 전위!
inorder(node->left);
inorder(node->right);
}

-> printf의 위치가 순회 종류를 결정합니다.
-> 전위: 맨 위 / 중위: 가운데 / 후위: 맨 아래
에러 3: 메모리 해제 순서 (전위로 해제)
void freeTree(TreeNode *node) {
if (node == NULL) return;
free(node); // 자기를 먼저 해제하면
freeTree(node->left); // 해제된 메모리 접근! (UAF)
freeTree(node->right);
}

-> 메모리 해제는 반드시 후위 순회 순서로 합니다.
-> 자식을 먼저 해제한 뒤 부모를 해제해야 합니다.

핵심 정리
1. 트리: 계층 구조. 루트에서 시작하여 부모-자식 관계로 연결
2. 이진 트리 노드: struct TreeNode { int data; TreeNode *left, *right; }
3. 전위(Preorder): 루트 → 왼쪽 → 오른쪽 (printf가 재귀 호출 위)
4. 중위(Inorder): 왼쪽 → 루트 → 오른쪽 (printf가 재귀 호출 사이)
5. 후위(Postorder): 왼쪽 → 오른쪽 → 루트 (printf가 재귀 호출 아래)
6. 메모리 해제는 반드시 후위 순회 순서 (자식 먼저, 부모 나중)
7. 레벨 순회(BFS): 큐를 사용하여 위에서 아래, 왼쪽에서 오른쪽으로 방문

용어설명
트리(Tree)계층 구조를 가진 비선형 자료구조. 루트에서 시작하여 부모-자식 관계로 노드가 연결됩니다.
이진 트리(Binary Tree)각 노드의 자식이 최대 2개(왼쪽, 오른쪽)인 트리입니다.
루트(Root)트리의 최상위 노드. 부모가 없는 유일한 노드입니다.
리프(Leaf)자식이 없는 노드. 단말 노드라고도 합니다.
전위 순회(Preorder)루트 → 왼쪽 → 오른쪽 순서로 방문하는 순회 방법입니다.
중위 순회(Inorder)왼쪽 → 루트 → 오른쪽 순서로 방문하는 순회 방법입니다.
후위 순회(Postorder)왼쪽 → 오른쪽 → 루트 순서로 방문하는 순회 방법입니다.
레벨 순회(Level-order)큐를 사용하여 같은 깊이의 노드를 왼쪽에서 오른쪽으로 방문하는 BFS 기반 순회입니다.
