프로젝트 개요

우선순위 기반 작업 스케줄러 (Priority Task Scheduler)
힙과 우선순위 큐를 활용하여 작업을 우선순위에 따라 관리하고 실행하는 스케줄러를 구현합니다.

버전 1: 기본 -- Task 구조체(이름, 우선순위), 최소 힙 기반 우선순위 큐, 작업 추가/실행
버전 2: 확장 -- 마감일(시간 개념), 우선순위 변경, 작업 목록 출력, 메뉴 시스템
버전 3: 완성 -- 파일 저장/불러오기, 카테고리별 관리, 실행 로그, 자료구조 종합 비교표, 통계

36/* 힙 아래로 내리기 (삭제 후 정렬) */

38 while (left(idx) <= heapSize) {
39 int smallest = idx;
40 int l = left(idx);
41 int r = right(idx);
42
43 if (l <= heapSize &&
44 heap[l].priority < heap[smallest].priority) {
45 smallest = l;
46 }
47 if (r <= heapSize &&
48 heap[r].priority < heap[smallest].priority) {
49 smallest = r;
50 }
51
52 if (smallest == idx) break;
53
54 swap(&heap[idx], &heap[smallest]);
55 idx = smallest;
56 }

최소 힙 동작 원리
최소 힙은 부모 노드가 항상 자식보다 작거나 같은 완전 이진 트리입니다.
배열 인덱스로 트리를 표현합니다 (1-based).

[1] 우선순위 1
/ \
[2] 우선순위 2 [3] 우선순위 3
/ \
[4] 우선순위 5 [5] 우선순위 2

부모: i / 2
왼쪽 자식: i * 2
오른쪽 자식: i * 2 + 1

삽입 (heapUp): 맨 끝에 넣고 부모와 비교하며 위로 올림
삭제 (heapDown): 루트를 꺼내고 맨 끝을 루트로 옮긴 뒤 아래로 내림

두 연산 모두 O(log n) -- 트리 높이만큼만 이동

버전 2: 확장 -- 마감일 + 우선순위 변경 + 메뉴

254 for (int i = 1; i <= heapSize; i++) {
255 if (heap[i].id == targetId) {
256 found = i;
257 break;
258 }
259 }

261 if (found == -1) {

272 heap[found].priority = newPriority;
273
274 if (newPriority < oldPriority)
275 heapUp(found);
276 else
277 heapDown(found);
278
279 printf("작업 #%d의 우선순위: %d -> %d\n",
280 targetId, oldPriority, newPriority);
