핵심 개념

_Bool 타입의 동작
_Bool은 C99에서 추가된 내장 타입입니다.

- 0을 대입하면 0(거짓)이 저장됩니다.
- 0이 아닌 값을 대입하면 1(참)로 변환되어 저장됩니다.
- 크기는 1바이트입니다.

_Bool b1 = 0; // b1 == 0
_Bool b2 = 1; // b2 == 1
_Bool b3 = 42; // b3 == 1 (0이 아닌 값 -> 1)
_Bool b4 = -5; // b4 == 1 (0이 아닌 값 -> 1)

에러 체험

에러 1: bool에 2를 대입하면?
bool b = 2;
printf("%d\n", b); // 출력: 1

→ _Bool과 bool은 0이 아닌 값을 모두 1로 변환합니다.
→ 2, 100, -1 등 어떤 값을 넣어도 1이 됩니다.
에러 2: if (flag == true) vs if (flag)
int flag = 2;
if (flag == true) // 거짓! (2 != 1)
printf("참\n"); // 실행 안 됨
if (flag) // 참! (0이 아닌 값)
printf("참\n"); // 실행됨

→ int 변수를 true와 직접 비교하면 의도치 않은 결과가 발생합니다.
→ if (flag)처럼 직접 조건으로 사용하는 것이 안전합니다.
에러 3: 비교 연산자 실수 (= vs ==)
bool done = false;
if (done = true) // 대입! 항상 참이 됨
printf("완료\n");

→ ==를 써야 하는 곳에 =를 쓰면 대입이 되어 항상 참입니다.
→ if (done == true) 또는 if (done)을 사용하십시오.
