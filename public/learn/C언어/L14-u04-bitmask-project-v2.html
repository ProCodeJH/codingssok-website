2진수 변환기 + 비트 마스크

비트 연산을 종합 활용하여 2진수 변환기와 비트 마스크 도구를 구현합니다.

이 유닛에서 배우는 것

  • 비트 연산으로 10진수와 2진수를 상호 변환할 수 있습니다.
  • 비트 플래그의 SET, CLEAR, TOGGLE, CHECK 연산을 구현할 수 있습니다.
  • 비트마스크를 이용한 집합 연산(합집합, 교집합, 차집합, 부분집합)을 구현할 수 있습니다.
  • 비트마스크 기반 권한 관리 시스템을 설계하고 구현할 수 있습니다.

2진수 변환기 + 비트 마스크
비트 연산자를 종합 활용하여 2진수 변환, 비트 플래그 관리, 비트마스크 집합 연산을 구현합니다. 비트 연산은 하드웨어 제어, 권한 관리, 상태 플래그 등 시스템 프로그래밍의 핵심 기법이며 알고리즘 대회에서도 자주 활용됩니다.

6/* 10진수 -> 2진수 문자열 변환 (비트 연산 사용) */
7void decToBin(unsigned int num, char *result, int bits) {
8 for (int i = bits - 1; i >= 0; i--) {
9 /* 최상위 비트부터 검사: 해당 비트가 1인지 확인 */
10 result[bits - 1 - i] = (num & (1 << i)) ? '1' : '0';
11 }
12 result[bits] = '\0';
13}
14
15/* 2진수 문자열 -> 10진수 변환 (비트 연산 사용) */
16unsigned int binToDec(const char *binStr) {
17 unsigned int result = 0;
18 int len = (int)strlen(binStr);
19
20 for (int i = 0; i < len; i++) {
21 if (binStr[i] == '1') {
22 result |= (1 << (len - 1 - i)); /* 해당 비트 위치에 1 설정 */
23 }
24 }
25 return result;
26}

10void printSet(unsigned char set, const char *names[], int n) {
11 printf("{ ");
12 int first = 1;
13 for (int i = 0; i < n; i++) {
14 if (set & (1 << i)) {

23/* 집합 비트 시각화 */
24void printBits(unsigned char set, int n) {
25 for (int i = n - 1; i >= 0; i--) {
26 printf("%d", (set >> i) & 1);
27 }
28}
29
30/* 원소 개수 (popcount) */
31int countBits(unsigned char set) {
32 int count = 0;
33 while (set) {
34 count += set & 1;
35 set >>= 1;
36 }
37 return count;
38}

40/* 합집합 */

122#define PERM_READ(1 << 0) /* 0b00000001: 읽기 */
123#define PERM_WRITE(1 << 1) /* 0b00000010: 쓰기 */
124#define PERM_EXECUTE(1 << 2) /* 0b00000100: 실행 */
125#define PERM_DELETE(1 << 3) /* 0b00001000: 삭제 */
126#define PERM_ADMIN(1 << 4) /* 0b00010000: 관리자 */
127#define PERM_SHARE(1 << 5) /* 0b00100000: 공유 */

152}

5if (flags & (1 << bit)) // 수정: 명시적으로 괄호 사용
6
7// 에러 2: CLEAR 연산에서 ~ 누락
8flags &= (1 << bit); // 해당 비트만 남기고 나머지를 모두 삭제!
9// 수정: flags &= ~(1 << bit); // 해당 비트만 0으로 만듦
10
11// 에러 3: 부분집합 판별 조건 오류
12if ((a | b) == a) // 이것은 b가 a의 부분집합인지 확인하는 것
13if ((a & b) == a) // 이것은 a가 b의 부분집합인지 확인하는 것
14// 주의: a가 b의 부분집합이면 a의 모든 비트가 b에도 있으므로
15// a & b == a가 성립합니다.
16
17// 에러 4: 비트 범위 초과 시프트
18int flags = 1 << 32; // 정의되지 않은 동작 (int가 32비트일 때)

요약

핵심 정리
1. 10진수 -> 2진수: 최상위 비트부터 (1 << i)로 마스크 검사하여 변환
2. 2진수 -> 10진수: 각 '1' 위치에 대해 result |= (1 << 위치)로 누적
3. SET: flags |= (1 << bit) -- 특정 비트를 1로 설정
4. CLEAR: flags &= ~(1 << bit) -- 특정 비트를 0으로 해제
5. TOGGLE: flags ^= (1 << bit) -- 특정 비트를 반전
6. CHECK: flags & (1 << bit) -- 특정 비트가 1인지 확인
7. 합집합: a | b, 교집합: a & b, 차집합: a & ~b
8. 부분집합 판별: (a & b) == a이면 a는 b의 부분집합
9. popcount: n &= (n - 1)을 반복하면 1인 비트 수만큼만 순회
10. 비트마스크 집합은 O(1) 연산이 가능하나 원소 수가 비트 크기로 제한됨
