커스텀 qsort 비교 함수

함수 포인터와 콜백을 활용하여 다양한 비교 함수로 qsort를 사용하는 프로젝트입니다.

이 유닛에서 배우는 것

  • 정수, 문자열, 구조체 각각에 맞는 비교 함수를 작성하고 qsort에 전달할 수 있습니다.
  • 다중 정렬 기준을 적용하는 비교 함수를 설계하고 구현할 수 있습니다.
  • 비교 함수 배열을 활용하여 사용자가 정렬 기준을 선택하는 시스템을 만들 수 있습니다.
  • void 포인터 기반 범용 정렬 프레임워크를 구현하고 성능을 측정할 수 있습니다.

qsort 비교 함수의 규칙
qsort의 비교 함수는 반드시 다음 시그니처를 따릅니다.

int compare(const void *a, const void *b);

반환값의 의미:
음수 (<0) : a가 b보다 앞에 위치
0 : a와 b의 순서가 같음
양수 (>0) : a가 b보다 뒤에 위치

void 포인터 캐스팅 순서:
정수: const void* -> const int* -> 역참조
문자열: const void* -> const char** -> 역참조
구조체: const void* -> const Student* -> 멤버 접근

주의: 정수 비교에서 return a - b는 오버플로우 위험이 있습니다.
INT_MAX - (-1) 등의 경우 오버플로우가 발생하므로,
안전한 비교는 if (x < y) return -1; 형태를 권장합니다.

요약

핵심 정리
1. qsort: stdlib.h의 범용 정렬 함수, 비교 함수 포인터를 인자로 받음
2. 비교 함수 시그니처: int (*)(const void *, const void *)
3. 반환값: 음수(a가 앞), 0(동일), 양수(a가 뒤)
4. 정수 비교: *(const int *)a -- 직접 역참조
5. 문자열 비교: *(const char **)a -- 이중 포인터 역참조
6. 구조체 비교: (const Student *)a -> 멤버 접근
7. 다중 기준: 1차 기준이 같으면 2차 기준으로 이동하는 체인 패턴
8. 함수 포인터 배열: 비교 함수를 배열에 저장하여 동적 선택 가능
9. 레지스트리 패턴: 비교 함수를 등록하고 이름으로 관리하는 구조
10. 정수 오버플로우 주의: a - b 대신 조건 분기로 안전하게 비교
