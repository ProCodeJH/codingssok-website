17 return dp[n];
18}

에러 체험

에러 1: 피보나치 초기값 누락
long long dp[n + 1];
dp[0] = 0;
/* dp[1] = 1; 빠뜨림! */
for (int i = 2; i <= n; i++)
dp[i] = dp[i-1] + dp[i-2]; // dp[1]이 쓰레기 값

-> dp[1] = 1 초기화를 반드시 작성해야 합니다.
-> 초기값 설정은 DP 구현의 첫 번째 단계입니다.
에러 2: 1로 만들기에서 나눗셈 조건 누락
for (int i = 2; i <= n; i++) {
dp[i] = dp[i-1] + 1;
dp[i] = min(dp[i], dp[i/2] + 1); // i가 홀수일 때 오류!
dp[i] = min(dp[i], dp[i/3] + 1); // i가 3의 배수 아닐 때 오류!
}

-> i % 2 == 0, i % 3 == 0 조건을 반드시 확인해야 합니다.
-> 나누어 떨어지지 않으면 해당 연산은 적용할 수 없습니다.
에러 3: 계단 오르기 dp[0] 초기값 실수
dp[0] = 0; // 잘못된 초기값!
dp[1] = 1;

-> dp[0]은 바닥에 서 있는 상태이므로 1이어야 합니다.
-> dp[0] = 0으로 설정하면 dp[2] = dp[1] + dp[0] = 1이 되어
2칸 계단의 답이 1(올바른 답: 2)이 됩니다.

핵심 정리
1. 피보나치 DP: 배열 O(n) -> 변수 2개 O(1) 공간 최적화
2. 계단 오르기: dp[n] = dp[n-1] + dp[n-2], 초기값 dp[0]=1, dp[1]=1
3. 2xN 타일링: dp[n] = dp[n-1] + dp[n-2], 초기값 dp[1]=1, dp[2]=2
4. 1로 만들기: dp[i] = min(dp[i-1], dp[i/2], dp[i/3]) + 1, 조건 분기 필수
5. 경로 역추적: path 배열에 이전 상태를 기록하여 최적 경로 복원
6. 핵심 패턴: 점화식 도출 -> 초기값 설정 -> 반복문으로 채우기 -> 결과 반환

선수 학습: 이 유닛(L15-U06)의 점화식 설계와 DP 배열 채우기
용어설명
점화식(Recurrence Relation)수열의 각 항을 이전 항들로 나타내는 관계식입니다. DP의 핵심은 이 점화식을 올바르게 도출하는 것입니다.
공간 최적화(Space Optimization)DP 배열 전체를 유지하지 않고 필요한 값만 변수로 관리하여 메모리 사용량을 줄이는 기법입니다.
경로 역추적(Path Reconstruction)DP로 최적값을 구한 후, 각 단계에서의 선택을 역으로 추적하여 실제 경로를 복원하는 기법입니다.
피보나치 수열(Fibonacci Sequence)F(0)=0, F(1)=1이고 F(n) = F(n-1) + F(n-2)인 수열입니다. 자연, 예술, 알고리즘에 광범위하게 등장합니다.
타일링(Tiling)주어진 영역을 정해진 모양의 타일로 빈틈없이 채우는 문제입니다. 조합론과 DP의 대표적 응용입니다.
최적 부분 구조(Optimal Substructure)큰 문제의 최적해가 작은 부분 문제의 최적해로 구성되는 성질입니다. DP가 적용 가능한 필수 조건입니다.
