DFS -- 깊이 우선 탐색

재귀와 스택을 이용한 깊이 우선 탐색(DFS)의 원리를 이해하고, 인접 행렬/리스트 기반 구현과 연결 요소, 사이클 감지 등의 응용을 학습합니다.

이 유닛에서 배우는 것

  • 깊이 우선 탐색(DFS)의 원리를 이해하고 탐색 순서를 손으로 추적할 수 있습니다.
  • 재귀 함수를 이용한 DFS를 인접 행렬과 인접 리스트로 각각 구현할 수 있습니다.
  • 스택 기반 반복 DFS를 구현하고 재귀 DFS와의 차이를 설명할 수 있습니다.
  • DFS를 활용하여 연결 요소 개수 세기와 사이클 감지 문제를 해결할 수 있습니다.

핵심 개념

DFS 원리: 스택 또는 재귀
DFS는 다음 규칙을 따릅니다.
1. 시작 정점을 방문 표시하고 처리합니다.
2. 현재 정점의 인접 정점 중 미방문 정점이 있으면, 그 정점으로 이동하여 1번을 반복합니다.
3. 미방문 인접 정점이 없으면 이전 정점으로 되돌아갑니다(백트래킹).
4. 모든 정점을 방문할 때까지 반복합니다.

재귀 호출 자체가 스택 구조이므로 재귀로 자연스럽게 구현됩니다.
명시적 스택을 사용하면 반복문으로도 동일한 탐색이 가능합니다.

재귀 DFS (인접 행렬)

C
1#include <stdio.h>
2#define MAX 100
3
4int adj[MAX][MAX]; /* 인접 행렬 */
5int visited[MAX]; /* 방문 배열 */
6int n; /* 정점 개수 */
7
8void dfs(int v) {
9 visited[v] = 1;
10 printf("%d ", v);
11 for (int i = 0; i < n; i++) {
12 if (adj[v][i] == 1 && !visited[i]) {
13 dfs(i);
14 }
15 }
16}
17
18int main(void) {
19 int m; /* 간선 개수 */
20 scanf("%d %d", &n, &m);
21 for (int i = 0; i < m; i++) {
22 int u, v;
23 scanf("%d %d", &u, &v);
24 adj[u][v] = 1;
25 adj[v][u] = 1; /* 무방향 그래프 */
26 }
27 printf("DFS 방문 순서: ");
28 dfs(0);
29 printf("\n");
30 return 0;
31}

2#include <stdlib.h>
3#define MAX 100
4
5typedef struct Node {
6 int vertex;
7 struct Node *next;
8} Node;
9
10Node *graph[MAX]; /* 인접 리스트 */
11int visited[MAX];
12int n;
13
14void addEdge(int u, int v) {
15 Node *newNode = (Node *)malloc(sizeof(Node));
16 newNode->vertex = v;
17 newNode->next = graph[u];
18 graph[u] = newNode;
19}
20
21void dfs(int v) {
22 visited[v] = 1;
23 printf("%d ", v);
24 Node *cur = graph[v];
25 while (cur != NULL) {
26 if (!visited[cur->vertex]) {
27 dfs(cur->vertex);
28 }
29 cur = cur->next;
30 }
31}

스택 기반 반복 DFS

C
1#include <stdio.h>
2#define MAX 100
3
4int adj[MAX][MAX];
5int visited[MAX];
6int n;
7
8void dfs_iterative(int start) {
9 int stack[MAX];
10 int top = -1;
11 stack[++top] = start; /* push */
12
13 while (top >= 0) {
14 int v = stack[top--]; /* pop */
15 if (visited[v]) continue;
16 visited[v] = 1;
17 printf("%d ", v);
18 /* 번호가 큰 정점부터 push하면 작은 번호가 먼저 pop됨 */
19 for (int i = n - 1; i >= 0; i--) {
20 if (adj[v][i] == 1 && !visited[i]) {
21 stack[++top] = i;
22 }
23 }
24 }
25}

1#include <stdio.h>
2#define MAX 100
3
4int adj[MAX][MAX];
5int visited[MAX];
6int n;
7int hasCycle = 0;
8
9void dfs(int v, int parent) {
10 visited[v] = 1;
11 for (int i = 0; i < n; i++) {
12 if (adj[v][i] == 1) {
13 if (!visited[i]) {
14 dfs(i, v);
15 } else if (i != parent) {
16 hasCycle = 1; /* 부모가 아닌 방문 정점 발견 */
17 }
18 }
19 }
20}

27 scanf("%d %d", &u, &v);
28 adj[u][v] = 1;
29 adj[v][u] = 1;
30 }
31 dfs(0, -1);
32 if (hasCycle)
33 printf("사이클이 존재합니다.\n");
34 else

요약

핵심 정리
1. DFS는 한 경로를 끝까지 탐색한 뒤 되돌아오는 깊이 우선 전략입니다.
2. 재귀 호출 자체가 스택이므로 재귀로 간결하게 구현됩니다.
3. 인접 행렬 DFS: O(V^2), 인접 리스트 DFS: O(V + E)
4. 명시적 스택으로 반복 DFS를 구현하면 재귀 깊이 제한을 피할 수 있습니다.
5. visited 배열로 방문 여부를 반드시 체크해야 무한루프를 방지합니다.
6. 응용: 연결 요소 개수, 사이클 감지, 경로 탐색, 이분 그래프 판별 등

다음 유닛: BFS -- 너비 우선 탐색
DFS가 한 길을 끝까지 파고드는 전략이라면, BFS는 시작점에서 가까운 정점부터 단계별로 탐색하는 전략입니다. 큐 자료구조를 활용하며, 가중치가 없는 그래프에서 최단 거리를 보장합니다. DFS와 BFS를 모두 익히면 그래프 문제의 대부분을 해결할 수 있습니다.
용어설명
DFS(Depth-First Search)그래프에서 한 경로를 끝까지 탐색한 후 되돌아와 다른 경로를 탐색하는 깊이 우선 탐색 알고리즘입니다.
백트래킹(Backtracking)더 이상 진행할 수 없을 때 가장 최근의 분기점으로 되돌아가는 기법입니다.
연결 요소(Connected Component)무방향 그래프에서 서로 경로로 연결된 정점들의 최대 부분 집합입니다.
사이클(Cycle)그래프에서 한 정점에서 출발하여 자기 자신으로 되돌아오는 경로입니다.
인접 행렬(Adjacency Matrix)정점 간 연결 여부를 2차원 배열로 표현하는 방식입니다. adj[i][j] = 1이면 i와 j가 연결되어 있습니다.
인접 리스트(Adjacency List)각 정점의 이웃 정점을 연결 리스트로 저장하는 방식입니다. 희소 그래프에서 효율적입니다.
