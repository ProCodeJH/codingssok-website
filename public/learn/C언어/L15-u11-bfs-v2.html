핵심 개념

BFS 원리
BFS는 큐(Queue)를 사용하여 레벨 단위로 그래프를 탐색합니다.

동작 과정:
1. 시작 정점을 큐에 넣고(enqueue) 방문 표시
2. 큐에서 정점을 하나 꺼냄(dequeue)
3. 꺼낸 정점의 인접 정점 중 방문하지 않은 정점을 모두 큐에 넣고 방문 표시
4. 큐가 빌 때까지 2-3 반복

예시 (시작: 정점 0)

0 -- 1 -- 3
| | |
2 4 5

큐 변화:
[0] -> 0 꺼냄 -> 인접 1, 2 추가
[1, 2] -> 1 꺼냄 -> 인접 3, 4 추가
[2, 3, 4] -> 2 꺼냄 -> 새 인접 없음
[3, 4] -> 3 꺼냄 -> 인접 5 추가
[4, 5] -> 4 꺼냄 -> 새 인접 없음
[5] -> 5 꺼냄 -> 새 인접 없음
[] -> 큐 비었음, 탐색 종료

방문 순서: 0 -> 1 -> 2 -> 3 -> 4 -> 5

19void addEdge(int u, int v) {
20 Node* newNode = (Node*)malloc(sizeof(Node));

22 newNode->next = adjList[u];
23 adjList[u] = newNode;
24

BFS의 핵심 특성: 최단 거리 보장

가중치 없는 그래프에서의 최단 거리
BFS는 시작 정점에서 각 정점까지의 최단 거리(간선 수)를 보장합니다.

이유:
- BFS는 거리 0인 정점(시작점)을 먼저 처리합니다.
- 거리 0인 정점의 인접 정점은 모두 거리 1입니다.
- 거리 1인 정점의 인접 정점 중 미방문인 것은 모두 거리 2입니다.
- 이 과정이 반복되므로, 각 정점은 처음 방문될 때가 최단 거리입니다.

주의: 이 성질은 모든 간선의 가중치가 동일한(또는 가중치가 없는) 그래프에서만 성립합니다.
가중치가 다른 그래프에서는 다익스트라 알고리즘을 사용해야 합니다.

BFS로 최단 거리 구하기

C
1#include <stdio.h>
2#define MAX 100
3
4int graph[MAX][MAX];
5int visited[MAX];
6int dist[MAX]; // 시작점으로부터의 최단 거리
7int queue[MAX];
8int front = 0, rear = 0;
9
10void enqueue(int v) { queue[rear++] = v; }
11int dequeue(void) { return queue[front++]; }
12int isEmpty(void) { return front == rear; }
13
14void bfsDistance(int start, int n) {
15 for (int i = 0; i < n; i++)
16 dist[i] = -1; // -1은 미방문
17
18 dist[start] = 0;
19 visited[start] = 1;
20 enqueue(start);
21
22 while (!isEmpty()) {
23 int cur = dequeue();
24
25 for (int i = 0; i < n; i++) {
26 if (graph[cur][i] == 1 && !visited[i]) {
27 visited[i] = 1;
28 dist[i] = dist[cur] + 1;
29 enqueue(i);
30 }
31 }
32 }
33}
34

40 graph[1][3] = graph[3][1] = 1;
41 graph[1][4] = graph[4][1] = 1;
42 graph[2][5] = graph[5][2] = 1;
43 graph[3][6] = graph[6][3] = 1;
44
45 bfsDistance(0, n);
46
47 printf("정점 | 최단 거리\n");
48 printf("-----|----------\n");
49 for (int i = 0; i < n; i++)
50 printf(" %d | %d\n", i, dist[i]);
51
52 return 0;
53}

C
1#include <stdio.h>
2#define ROWS 5
3#define COLS 5
4#define MAX_Q(ROWS * COLS)
5
6int maze[ROWS][COLS] = {
7 {0, 0, 1, 0, 0},
8 {1, 0, 1, 0, 1},
9 {0, 0, 0, 0, 0},
10 {0, 1, 1, 1, 0},
11 {0, 0, 0, 0, 0}
12};
13// 0 = 이동 가능, 1 = 벽
14
15int dist[ROWS][COLS];
16int dx[] = {-1, 1, 0, 0};
17int dy[] = {0, 0, -1, 1};
18
19typedef struct { int x, y; } Point;
20Point queue[MAX_Q];
21int front = 0, rear = 0;
22
23void enqueue(Point p) { queue[rear++] = p; }
24Point dequeue(void) { return queue[front++]; }
25int isEmpty(void) { return front == rear; }
26
27void bfsMaze(int sx, int sy) {
28 for (int i = 0; i < ROWS; i++)
29 for (int j = 0; j < COLS; j++)
30 dist[i][j] = -1;
31
32 dist[sx][sy] = 0;
33 enqueue((Point){sx, sy});
34
35 while (!isEmpty()) {
36 Point cur = dequeue();
37
38 for (int d = 0; d < 4; d++) {
39 int nx = cur.x + dx[d];
40 int ny = cur.y + dy[d];
41
42 // 범위 확인
43 if (nx < 0 || nx >= ROWS || ny < 0 || ny >= COLS)
44 continue;
45 // 벽이거나 이미 방문한 칸이면 건너뜀
46 if (maze[nx][ny] == 1 || dist[nx][ny] != -1)
47 continue;
48
49 dist[nx][ny] = dist[cur.x][cur.y] + 1;
50 enqueue((Point){nx, ny});
51 }
52 }
53}

요약

핵심 정리
1. BFS는 큐를 사용하여 시작 정점에서 가까운 정점부터 레벨 단위로 탐색합니다.
2. 핵심 패턴: 시작점 enqueue -> while(!isEmpty) -> dequeue -> 인접 정점 enqueue
3. 인접 행렬 BFS: O(V^2), 인접 리스트 BFS: O(V + E)
4. 가중치 없는 그래프에서 BFS는 최단 거리를 보장합니다.
5. dist 배열로 최단 거리를 기록하면 visited 배열을 대체할 수 있습니다.
6. 2D 격자 BFS: dx, dy 배열로 4방향 이동, 범위 검사 필수
7. 방문 표시는 큐에 넣는 시점에 즉시 해야 중복 방문을 방지합니다.
