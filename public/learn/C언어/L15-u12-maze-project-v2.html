미로 탈출 (BFS)

BFS를 활용하여 2D 격자 미로에서 최단 경로를 찾고, 경로를 역추적하여 시각화하며, DFS와 비교하는 프로젝트입니다.

이 유닛에서 배우는 것

  • 2D 격자 미로를 배열로 표현하고 BFS로 최단 경로의 길이를 구할 수 있습니다.
  • 부모 좌표 배열을 이용하여 경로를 역추적하고 실제 이동 경로를 출력할 수 있습니다.
  • 파일에서 미로 데이터를 읽어 다양한 크기의 미로를 처리할 수 있습니다.
  • DFS와 BFS의 탐색 결과를 비교하고 각각의 장단점을 설명할 수 있습니다.

BFS 탐색 순서와 최단 경로 보장
BFS는 시작점에서 거리 1인 칸을 모두 방문한 뒤, 거리 2인 칸을 방문하는 식으로
레이어(층) 단위로 확장합니다.

거리 0: S (시작점)
거리 1: S의 상하좌우 이웃 중 통로
거리 2: 거리 1 칸들의 이웃 중 미방문 통로
거리 3: ...

따라서 어떤 칸에 처음 도달한 순간의 거리가 곧 최단 거리입니다.
이것이 BFS가 최단 경로를 보장하는 핵심 원리입니다.

시간 복잡도: O(ROWS x COLS) -- 모든 칸을 최대 한 번 방문
공간 복잡도: O(ROWS x COLS) -- 큐, 방문 배열, 거리 배열

209 memset(dfsVisited, 0, sizeof(dfsVisited));
210 memset(dfsDist, -1, sizeof(dfsDist));
211 dfsTop = 0;
212 dfsVisitCount = 0;
213
214 dfsPush(sr, sc);
215 dfsVisited[sr][sc] = 1;
216 dfsDist[sr][sc] = 0;
217 dfsVisitCount++;
218
219 int found = -1;
220
221 while (!dfsIsEmpty()) {
222 Pos cur = dfsPop();
223
224 if (cur.r == er && cur.c == ec) {
225 found = dfsDist[er][ec];
226 break;
227 }
228
229 for (int d = 0; d < 4; d++) {
230 int nr = cur.r + dx[d];
231 int nc = cur.c + dy[d];
232
233 if (nr < 0 || nr >= rows || nc < 0 || nc >= cols)
234 continue;
235
236 if (maze[nr][nc] == 0 && !dfsVisited[nr][nc]) {
237 dfsVisited[nr][nc] = 1;
238 dfsDist[nr][nc] = dfsDist[cur.r][cur.c] + 1;
239 dfsPush(nr, nc);
240 dfsVisitCount++;
241 }
242 }
243 }
244
245 return found;
246}

249void tracePath(int sr, int sc, int er, int ec) {
250 pathLen = 0;
251 int r = er, c = ec;
252
253 while (r != sr || c != sc) {
254 path[pathLen].r = r;
255 path[pathLen].c = c;
256 pathLen++;
257 int pr = parentR[r][c];
258 int pc = parentC[r][c];
259 r = pr;
260 c = pc;
261 }
262 path[pathLen].r = sr;
263 path[pathLen].c = sc;
264 pathLen++;
265
266 /* 뒤집기 */
267 for (int i = 0; i < pathLen / 2; i++) {
268 Pos temp = path[i];
269 path[i] = path[pathLen - 1 - i];
270 path[pathLen - 1 - i] = temp;
271 }
272}

요약

핵심 정리
1. 2D 격자 미로: int maze[R][C], 0=통로, 1=벽
2. 방향 배열: dx[]={-1,1,0,0}, dy[]={0,0,-1,1}로 상하좌우 이동
3. BFS 큐: Pos 구조체 배열, front/rear로 관리
4. visited 배열: enqueue 시점에 1로 설정하여 중복 방문 방지
5. dist 배열: 시작점부터의 최단 거리, dist[nr][nc] = dist[cur] + 1
6. 최단 경로 보장: BFS는 레이어별 탐색으로 처음 도달 = 최단 거리
7. 경로 역추적: parentR/parentC에 이전 좌표 기록, 도착점에서 역순 추적
8. DFS vs BFS: DFS는 깊이 우선(스택), BFS는 너비 우선(큐)
9. DFS는 최단 경로를 보장하지 않음 -- 경로 존재 확인에 적합
10. 시간 복잡도: O(R x C) -- 모든 칸을 최대 한 번 방문
