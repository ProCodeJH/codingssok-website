네비게이션 프로젝트 (Navigation / Shortest Path)
다익스트라와 플로이드-워셜 알고리즘을 활용하여 도시 간 최단 경로를 찾는 네비게이션 시스템을 구현합니다.

버전 1: 기본 -- 도시 간 최단 거리 (다익스트라, 인접 행렬)
버전 2: 확장 -- 모든 도시 간 최단 거리 (플로이드-워셜) + 경로 출력
버전 3: 완성 -- 파일에서 지도 읽기 + 경유지 추가 + 최적 경로 시각화

버전 1: 기본 -- 도시 간 최단 거리 (다익스트라, 인접 행렬)

1#include <stdio.h>
2#include <string.h>

4#define MAX_CITIES 10
5#define INF 99999
6#define NAME_LEN 20

8int graph[MAX_CITIES][MAX_CITIES]; /* 인접 행렬 (가중치) */
9char cityName[MAX_CITIES][NAME_LEN];
10int numCities;
11
12/* 도시 이름으로 인덱스 찾기 */
13int findCity(const char *name) {
14 for (int i = 0; i < numCities; i++) {
15 if (strcmp(cityName[i], name) == 0)
16 return i;
17 }

19}
20
21/* 인접 행렬 초기화 */
22void initGraph(void) {
23 for (int i = 0; i < MAX_CITIES; i++)
24 for (int j = 0; j < MAX_CITIES; j++)
25 graph[i][j] = (i == j) ? 0 : INF;
26}
27
28/* 도로 추가 (양방향) */
29void addRoad(int from, int to, int weight) {
30 graph[from][to] = weight;
31 graph[to][from] = weight;
32}

34/* 다익스트라 알고리즘 */
35void dijkstra(int start, int dist[], int prev[]) {
36 int visited[MAX_CITIES] = {0};
37
38 for (int i = 0; i < numCities; i++) {
39 dist[i] = graph[start][i];
40 prev[i] = (graph[start][i] != INF && i != start)
41 ? start : -1;
42 }
43 dist[start] = 0;
44 visited[start] = 1;
45
46 for (int count = 1; count < numCities; count++) {
47 /* 미방문 정점 중 최소 거리 선택 */
48 int minDist = INF, u = -1;
49 for (int i = 0; i < numCities; i++) {
50 if (!visited[i] && dist[i] < minDist) {
51 minDist = dist[i];
52 u = i;
53 }
54 }
55
56 if (u == -1) break; /* 연결되지 않은 정점 */
57 visited[u] = 1;
58
59 /* 인접 정점 거리 갱신 */
60 for (int v = 0; v < numCities; v++) {
61 if (!visited[v] && graph[u][v] != INF) {
62 int newDist = dist[u] + graph[u][v];
63 if (newDist < dist[v]) {
64 dist[v] = newDist;
65 prev[v] = u;
66 }
67 }
68 }
69 }
70}

72/* 경로 출력 (재귀) */
73void printPath(int prev[], int dest) {
74 if (prev[dest] == -1) {
75 printf("%s", cityName[dest]);
76 return;
77 }
78 printPath(prev, prev[dest]);
79 printf(" -> %s", cityName[dest]);
80}

82/* 전체 결과 출력 */
83void printResult(int start, int dist[], int prev[]) {
84 printf("\n출발 도시: %s\n", cityName[start]);
85 printf("%-12s %8s %s\n", "도착 도시", "거리", "경로");
86 printf("------------------------------------------\n");
87
88 for (int i = 0; i < numCities; i++) {
89 if (i == start) continue;
90 printf("%-12s ", cityName[i]);
91 if (dist[i] == INF) {
92 printf("%8s 경로 없음\n", "INF");
93 } else {
94 printf("%8d ", dist[i]);
95 printPath(prev, i);
96 printf("\n");
97 }
98 }
99 printf("------------------------------------------\n");
100}

대회 기출

요약

핵심 정리
1. 네비게이션: 도시(정점) + 도로(간선, 가중치) = 가중치 그래프
2. 인접 행렬: graph[i][j]에 가중치 저장, 연결 없으면 INF
3. 다익스트라: 한 정점에서 모든 정점까지 최단 거리, O(V^2)
4. 다익스트라 핵심: 미방문 중 최소 거리 정점 선택 -> 인접 정점 갱신
5. 플로이드-워셜: 모든 쌍 최단 거리, O(V^3), 3중 for(k, i, j)
6. 플로이드-워셜 핵심: dist[i][j] = min(dist[i][j], dist[i][k]+dist[k][j])
7. k가 가장 바깥: 경유 정점을 단계적으로 확장하는 동적 프로그래밍
8. 경로 추적: prev 배열(역방향) 또는 next 배열(순방향)
9. 경유지 경로: 구간별 최단 거리를 합산하여 총 거리 계산
10. INF 연산 주의: 덧셈 전에 INF 여부를 반드시 확인

다음: KMP 문자열 알고리즘
네비게이션 프로젝트로 다익스트라와 플로이드-워셜 최단 경로 알고리즘을 종합 활용했습니다. 다음 유닛에서는 문자열 검색 분야로 넘어가, KMP(Knuth-Morris-Pratt) 알고리즘을 학습합니다. 접두사 함수(failure function)를 이용하여 텍스트에서 패턴을 O(n+m) 시간에 찾는 효율적인 문자열 매칭 기법을 배웁니다.
