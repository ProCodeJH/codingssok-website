왜 변수가 필요한가

Layer 1에서는 printf로 고정된 문자열만 출력했습니다. "Hello, World!"처럼 프로그램을 실행할 때마다 항상 같은 결과가 나왔습니다.

하지만 실제 프로그램은 다릅니다. 계산기는 사용자가 입력한 숫자에 따라 결과가 달라집니다. 게임은 점수가 계속 변합니다. 이렇게 변하는 값을 저장하려면 변수가 필요합니다.

에러: 값이 바뀔 때마다 코드를 수정해야 합니다

수정: 변수 값만 바꾸면 됩니다

변수란 무엇인가

변수(variable)는 값을 저장하는 메모리 공간에 이름을 붙인 것입니다.

편의점에서 물건을 보관할 때 사물함을 사용합니다. 각 사물함에는 번호가 붙어 있고, 물건을 넣거나 꺼낼 수 있습니다. 변수도 마찬가지입니다.

1
① 사물함 = 메모리 공간
컴퓨터 메모리에 값을 저장할 공간이 할당됩니다.
2
② 번호표 = 변수 이름
그 공간에 이름(score, age 등)을 붙여서 구별합니다.
3
③ 물건 = 값
공간 안에 숫자, 문자 등의 값을 넣습니다.
4
④ 교체 가능
기존 물건을 꺼내고 새 물건을 넣을 수 있습니다. 변수의 값도 바꿀 수 있습니다.
변수의 정의 변수(variable) = 값을 저장하는 메모리 공간 + 이름

variable의 어원은 라틴어 variabilis(변할 수 있는)입니다. 이름 그대로, 저장된 값이 변할 수 있는 공간입니다.

변수 선언

변수를 사용하려면 먼저 선언(declaration)해야 합니다. 컴퓨터에게 "이런 종류의 값을 저장할 공간을 마련하라"고 지시하는 것입니다.

Cdeclare.c
1#include <stdio.h>
2
3int main(void) {
4 int age; /* 정수를 저장할 변수 age 선언 */
5 age = 17; /* age에 17을 저장(대입) */
6 printf("나이: %d세\n", age);
7 return 0;
8}
실행 결과
나이: 17세

4번 줄의 int age;를 분석합니다.

요소의미
int자료형(type) — 정수(integer)를 저장하겠다는 선언
age변수 이름 — 이 공간을 age라고 부르겠다는 선언
;문장의 끝 — 선언도 하나의 문장입니다
선언의 형식 자료형 변수이름;

int score;
int count;
int temperature;

이번 유닛에서는 정수를 저장하는 int만 사용합니다. 다른 자료형(float, char 등)은 이후 유닛에서 학습합니다.

대입 연산자 =

변수에 값을 저장하는 것을 대입(assignment)이라고 합니다. = 기호를 사용합니다.

Cassign.c
1int score;
2score = 95; /* score에 95를 대입 */
3score = 88; /* score에 88을 대입 (기존 95는 사라짐) */
= 는 "같다"가 아닙니다 수학에서 = 는 "같다"를 의미하지만, C에서 = 는 "오른쪽 값을 왼쪽 변수에 저장하라"는 뜻입니다.

score = 95; → score에 95를 넣어라
score = score + 10; → score의 현재 값에 10을 더한 결과를 score에 다시 넣어라

"같다"를 비교하는 연산자는 == 입니다. (이후 유닛에서 학습)

선언과 초기화를 동시에

변수를 선언하면서 동시에 값을 넣을 수 있습니다. 이것을 초기화(initialization)라고 합니다.

Cinit.c
1#include <stdio.h>
2
3int main(void) {
4 int age = 17; /* 선언 + 초기화 */
5 int score = 95; /* 선언 + 초기화 */
6 int year = 2026; /* 선언 + 초기화 */
7
8 printf("나이: %d세\n", age);
9 printf("점수: %d점\n", score);
10 printf("연도: %d년\n", year);
11 return 0;
12}
실행 결과
나이: 17세 점수: 95점 연도: 2026년
선언 vs 초기화 선언: int age; → 공간만 만듦 (값은 쓰레기 값)
초기화: int age = 17; → 공간을 만들면서 값도 넣음
대입: age = 20; → 이미 만들어진 공간에 새 값을 넣음

초기화하지 않은 변수를 사용하면 예측할 수 없는 값(쓰레기 값)이 출력됩니다. 반드시 초기화 후 사용하십시오.

printf와 %d — 변수 값 출력

printf에서 %d는 "이 자리에 정수 값을 넣어라"는 표시입니다. 서식 지정자(format specifier)라고 부릅니다.

Cformat.c
1#include <stdio.h>
2
3int main(void) {
4 int a = 10;
5 int b = 20;
6
7 printf("a = %d\n", a); /* %d → 10 */
8 printf("b = %d\n", b); /* %d → 20 */
9 printf("a + b = %d\n", a + b); /* %d → 30 */
10 return 0;
11}
실행 결과
a = 10 b = 20 a + b = 30

%d를 여러 개 사용하면 쉼표로 구분하여 값을 순서대로 넣습니다.

Cmulti_d.c
1int x = 3;
2int y = 7;
3printf("%d + %d = %d\n", x, y, x + y);
실행 결과
3 + 7 = 10
%d 개수와 값의 개수를 맞추십시오 %d가 3개이면 쉼표 뒤에 값도 3개를 써야 합니다. 개수가 맞지 않으면 쓰레기 값이 출력되거나 프로그램이 비정상 동작합니다.

잘못된 예: printf("%d %d", x); → %d가 2개인데 값은 1개

변수 값 변경

변수의 값은 언제든지 바꿀 수 있습니다. 새로운 값을 대입하면 기존 값은 사라집니다.

Cchange.c
1#include <stdio.h>
2
3int main(void) {
4 int count = 0;
5 printf("처음: %d\n", count);
6
7 count = 1;
8 printf("변경 후: %d\n", count);
9
10 count = count + 1;
11 printf("1 증가: %d\n", count);
12
13 count = count + 1;
14 printf("1 증가: %d\n", count);
15 return 0;
16}
실행 결과
처음: 0 변경 후: 1 1 증가: 2 1 증가: 3
줄실행 내용count 값
4int count = 0;0
7count = 1;1
10count = count + 1; → count = 1 + 1;2
13count = count + 1; → count = 2 + 1;3
트레이스 테이블 위 표처럼 변수의 값이 변하는 과정을 한 줄씩 추적하는 것을 트레이스(trace)라고 합니다. 코드의 동작을 이해하는 가장 확실한 방법입니다.

용어 정리

용어설명
변수(variable)값을 저장하는 메모리 공간에 이름을 붙인 것입니다. 저장된 값을 바꿀 수 있습니다.
선언(declaration)컴파일러에게 변수의 존재와 자료형을 알리는 것입니다. 예: int age;
초기화(initialization)변수를 선언하면서 처음 값을 넣는 것입니다. 예: int age = 17;
대입(assignment)변수에 값을 저장하는 것입니다. = 연산자를 사용합니다. 예: age = 20;
자료형(data type)변수에 저장할 수 있는 값의 종류입니다. int는 정수, float는 실수를 저장합니다.
서식 지정자(format specifier)printf에서 값이 들어갈 위치와 출력 형식을 지정합니다. %d는 정수를 의미합니다.
쓰레기 값(garbage value)초기화하지 않은 변수에 들어 있는 예측 불가능한 값입니다.

변수는 추상화의 첫 걸음입니다. 구체적인 값(95, 88) 대신 이름(score)으로 대표합니다.

score = 88; → 한 곳만 바꾸면 전체가 바뀜

이것이 프로그래밍에서 추상화가 중요한 이유입니다.

4. %d 개수 불일치 → 쓰레기 값 출력

디버깅 원칙: 변수는 선언 → 초기화 → 사용 순서를 지키십시오.

대회에서 변수를 사용하는 이유:
- 입력값을 저장하기 위해 (다음 유닛에서 scanf 학습)
- 계산 중간 결과를 저장하기 위해
- 결과를 정해진 형식으로 출력하기 위해

대회에서는 변수 이름을 간결하게 짓는 것이 일반적입니다 (n, m, x, y 등).

- 의미 있는 이름을 사용합니다 (a, b보다 price, quantity)
- 한 줄에 변수 하나씩 선언합니다
- 관련 있는 변수는 가까이 선언합니다
축 7: 실생활 연결 [선택] 변수는 일상에도 있습니다:
- 편의점 사물함: 칸(메모리) + 번호(변수 이름) + 물건(값)
- 시험 성적표: 과목명(변수 이름) + 점수(값)
- 통장 잔고: 잔고(변수)가 입출금(대입)에 따라 변합니다

변수가 "변할 수 있는 값"이라는 뜻인 것처럼, 현실에서도 상태가 변하는 값은 모두 변수로 표현할 수 있습니다.
