printf 서식 지정자 %d %f %c %s

printf의 다양한 서식 지정자를 학습하고, 정수·실수·문자·문자열을 출력하는 방법을 익힙니다.

이 유닛에서 배우는 것

  • %d, %f, %c, %s 서식 지정자의 용도를 설명할 수 있습니다.
  • 자릿수 지정(%5d, %.2f)을 사용하여 출력 형식을 제어할 수 있습니다.
  • %o, %x로 8진수·16진수를 출력할 수 있습니다.
  • 서식 지정자와 값의 자료형이 일치해야 하는 이유를 이해합니다.

서식 지정자란

서식 지정자(format specifier)는 printf의 문자열 안에서 % 기호로 시작하며, "이 위치에 값을 넣어라"고 지시합니다. 값의 종류에 따라 다른 서식 지정자를 사용합니다.

서식 지정자자료형의미예시
%dint정수(decimal)printf("%d", 42); → 42
%ffloat, double실수(floating point)printf("%f", 3.14); → 3.140000
%cchar문자 1개(character)printf("%c", 'A'); → A
%schar[]문자열(string)printf("%s", "Hi"); → Hi
%uunsigned int부호 없는 정수printf("%u", 100); → 100
%ldlong긴 정수printf("%ld", 3000000000L); → 3000000000
%lldlong long매우 긴 정수printf("%lld", 9000000000LL); → 9000000000
%oint8진수(octal)printf("%o", 10); → 12
%xint16진수(hexadecimal)printf("%x", 255); → ff
%%없음% 문자 자체printf("100%%"); → 100%
가장 많이 쓰는 4가지 %d — 정수 (int)
%f — 실수 (float, double)
%c — 문자 (char)
%s — 문자열 (char[])

이 4가지만 확실히 기억하면 대부분의 출력 문제를 해결할 수 있습니다.

%d — 정수 출력

%d는 이미 사용해 왔습니다. 정수(int) 값을 10진수로 출력합니다.

Cformat_d.c
1#include <stdio.h>
2
3int main(void) {
4 int score = 95;
5 int negative = -10;
6
7 printf("점수: %d\n", score);
8 printf("음수: %d\n", negative);
9 printf("계산: %d\n", 100 + 200);
10 printf("여러 개: %d, %d, %d\n", 1, 2, 3);
11 return 0;
12}
실행 결과
점수: 95 음수: -10 계산: 300 여러 개: 1, 2, 3

%f — 실수 출력

%f는 실수(소수점이 있는 숫자)를 출력합니다. 기본적으로 소수점 아래 6자리까지 출력됩니다.

Cformat_f.c
1#include <stdio.h>
2
3int main(void) {
4 printf("%f\n", 3.14);
5 printf("%f\n", 1.0);
6 printf("%f\n", 0.1 + 0.2);
7 return 0;
8}
실행 결과
3.140000 1.000000 0.300000
실수 자료형 미리 보기 실수를 저장하는 자료형은 float(4바이트)과 double(8바이트)이 있습니다. 코드에서 3.14처럼 소수점을 쓰면 기본적으로 double로 취급됩니다. 자세한 내용은 L2-U09에서 학습합니다.

소수점 이하가 6자리나 출력되는 것이 불편할 수 있습니다. 자릿수를 지정하는 방법은 이 유닛 뒤쪽에서 학습합니다.

%c — 문자 출력

%c는 문자(character) 하나를 출력합니다. 문자는 작은따옴표(' ')로 감쌉니다.

Cformat_c.c
1#include <stdio.h>
2
3int main(void) {
4 char grade = 'A';
5 printf("학점: %c\n", grade);
6 printf("문자: %c, %c, %c\n", 'H', 'i', '!');
7 return 0;
8}
실행 결과
학점: A 문자: H, i, !
작은따옴표 vs 큰따옴표 작은따옴표 ' ' — 문자 1개: 'A', 'z', '3', '!'
큰따옴표 " " — 문자열(여러 문자): "Hello", "ABC"

'A'는 문자(char), "A"는 문자열(char[])입니다. 의미가 다릅니다.

char 자료형은 내부적으로 정수(ASCII 코드)를 저장합니다. 'A'는 65, 'a'는 97, '0'은 48입니다.

Cascii.c
1#include <stdio.h>
2
3int main(void) {
4 char ch = 'A';
5 printf("문자: %c\n", ch); /* A */
6 printf("ASCII: %d\n", ch); /* 65 */
7 printf("문자: %c\n", 66); /* B (66번) */
8 return 0;
9}
실행 결과
문자: A ASCII: 65 문자: B
같은 값, 다른 해석 %c로 출력하면 문자로, %d로 출력하면 숫자로 표시됩니다. 저장된 값은 같지만 서식 지정자에 따라 해석 방식이 달라집니다.

%s — 문자열 출력

%s는 문자열(string)을 출력합니다. 문자열은 큰따옴표(" ")로 감쌉니다.

Cformat_s.c
1#include <stdio.h>
2
3int main(void) {
4 printf("%s\n", "Hello, World!");
5 printf("이름: %s, 도시: %s\n", "Kim", "Seoul");
6 return 0;
7}
실행 결과
Hello, World! 이름: Kim, 도시: Seoul

문자열 변수는 char 배열로 선언합니다. 배열에 대해서는 이후 레이어에서 학습하므로, 지금은 printf에서 직접 문자열을 넣는 방법만 사용합니다.

서식 지정자 조합

하나의 printf에서 여러 서식 지정자를 섞어 쓸 수 있습니다.

Cmixed.c
1#include <stdio.h>
2
3int main(void) {
4 char grade = 'A';
5 int score = 95;
6
7 printf("학점: %c, 점수: %d점\n", grade, score);
8 printf("원주율: %f\n", 3.141592);
9 printf("%s %d번 학생: %c등급\n", "Kim", 7, 'B');
10 return 0;
11}
실행 결과
학점: A, 점수: 95점 원주율: 3.141592 Kim 7번 학생: B등급
서식 지정자와 값의 순서·개수·자료형을 맞추십시오 printf("%d %c %f", a, b, c);

첫 번째 %d → a (정수)
두 번째 %c → b (문자)
세 번째 %f → c (실수)

순서가 바뀌거나, 개수가 맞지 않거나, 자료형이 다르면 쓰레기 값이 출력됩니다.

자릿수 지정 — 정수

%d 앞에 숫자를 넣으면 출력 폭(width)을 지정할 수 있습니다.

Cwidth_d.c
1#include <stdio.h>
2
3int main(void) {
4 printf("[%d]\n", 42); /* 기본 */
5 printf("[%5d]\n", 42); /* 5칸, 오른쪽 정렬 */
6 printf("[%-5d]\n", 42); /* 5칸, 왼쪽 정렬 */
7 printf("[%05d]\n", 42); /* 5칸, 빈자리 0 */
8 return 0;
9}
실행 결과
[42] [ 42] [42 ] [00042]
서식의미결과
%d기본 (값 크기만큼)42
%5d최소 5칸, 오른쪽 정렬 42
%-5d최소 5칸, 왼쪽 정렬42
%05d최소 5칸, 빈 칸을 0으로 채움00042

자릿수 지정 — 실수

%f의 소수점 이하 자릿수를 .숫자로 지정합니다.

Cwidth_f.c
1#include <stdio.h>
2
3int main(void) {
4 printf("[%f]\n", 3.14); /* 기본: 소수점 6자리 */
5 printf("[%.2f]\n", 3.14); /* 소수점 2자리 */
6 printf("[%.0f]\n", 3.14); /* 소수점 없음 (반올림) */
7 printf("[%8.2f]\n", 3.14); /* 8칸, 소수점 2자리 */
8 return 0;
9}
실행 결과
[3.140000] [3.14] [3] [ 3.14]
실수 서식 지정 패턴 %.2f — 소수점 이하 2자리 (가장 많이 사용)
%.0f — 소수점 없이 정수처럼 출력
%8.2f — 전체 8칸, 소수점 이하 2자리

대회 문제에서 "소수점 아래 N자리까지 출력"이라는 조건이 나오면 %.Nf를 사용합니다.

%o와 %x — 8진수, 16진수

정수를 8진수(octal)나 16진수(hexadecimal)로 출력할 수 있습니다.

Coct_hex.c
1#include <stdio.h>
2
3int main(void) {
4 int num = 255;
5 printf("10진수: %d\n", num);
6 printf("8진수: %o\n", num);
7 printf("16진수: %x\n", num);
8 printf("16진수(대문자): %X\n", num);
9 return 0;
10}
실행 결과
10진수: 255 8진수: 377 16진수: ff 16진수(대문자): FF
언제 사용하나 8진수와 16진수는 일반 프로그래밍에서 자주 사용하지 않습니다. 메모리 주소, 색상 코드(#FF0000), 비트 연산 등에서 16진수를 사용합니다. 지금은 "이런 것도 된다" 정도만 기억하면 충분합니다.

에러 체험

서식 지정자를 잘못 사용했을 때 발생하는 문제를 경험합니다.

Cerr1.c
1#include <stdio.h>
2
3int main(void) {
4 int num = 42;
5 printf("%f\n", num); /* int인데 %f 사용 */
6 return 0;
7}
에러 1: 자료형 불일치 (int에 %f) 경고: warning: format '%f' expects 'double', but argument has type 'int'
증상: 0.000000 또는 쓰레기 값이 출력됩니다.
원인: int 값을 %f로 출력하면 컴파일러가 비트를 실수로 해석합니다.
수정: %d를 사용하거나 (double)num으로 형변환
Cerr2.c
1#include <stdio.h>
2
3int main(void) {
4 printf("%d %d %d\n", 1, 2); /* %d 3개, 값 2개 */
5 return 0;
6}
에러 2: 서식 지정자 개수 초과 경고: warning: format '%d' expects a matching 'int' argument
증상: 세 번째 %d에 쓰레기 값이 출력됩니다.
수정: 서식 지정자 개수와 값의 개수를 일치시키십시오.
Cerr3.c
1#include <stdio.h>
2
3int main(void) {

5 return 0;
6}
에러 3: 문자에 큰따옴표 사용 경고: warning: format '%c' expects 'int', but argument has type 'char *'
원인: 'A'는 문자(char), "A"는 문자열(char[])입니다. %c에는 'A'를 사용해야 합니다.
수정: printf("%c\n", 'A');
Cerr4.c
1#include <stdio.h>
2

원인: printf가 %\를 서식 지정자로 해석하려 합니다.
수정: printf("비율: 50%%\n"); — %를 출력하려면 %%를 사용

용어 정리

용어설명
서식 지정자(format specifier)printf에서 %로 시작하며, 값의 출력 형식을 지정합니다. %d(정수), %f(실수), %c(문자), %s(문자열) 등이 있습니다.
출력 폭(width)%5d처럼 숫자를 지정하면 최소 해당 칸만큼의 공간을 확보하여 출력합니다.
정밀도(precision)%.2f처럼 .숫자를 지정하면 소수점 이하 자릿수를 제한합니다.
ASCII 코드문자를 숫자로 대응시킨 표준 코드입니다. 'A'=65, 'a'=97, '0'=48입니다.
8진수(octal)0~7까지의 숫자를 사용하는 진법입니다. %o로 출력합니다.
16진수(hexadecimal)0~9, A~F까지의 숫자와 문자를 사용하는 진법입니다. %x 또는 %X로 출력합니다.

10축 매핑

이번 유닛에서 배운 내용을 10개의 학습 축으로 연결합니다.

축 1: C언어 [필수] 이 유닛의 C 문법: 서식 지정자(%d, %f, %c, %s, %o, %x, %%), 자릿수 지정(%5d, %.2f, %05d, %-5d)
깊이: LAYER 2 — printf의 출력 형식 제어
핵심 규칙: 서식 지정자와 값의 자료형·개수·순서가 일치해야 합니다
축 2: 컴퓨팅 사고력 [필수] 사고 패턴: 이중 부호화(Dual Coding)
같은 데이터(65)를 문자(%c→A)와 숫자(%d→65)로 표현할 수 있습니다. 데이터의 의미는 해석 방식에 따라 달라진다는 것을 이해하는 것이 컴퓨팅 사고의 핵심입니다.

컴퓨터에게 모든 데이터는 숫자(비트)입니다. 의미를 부여하는 것은 프로그래머입니다.
축 3: 디버깅 [필수] 이 유닛의 에러:
1. 자료형 불일치(int에 %f) → 쓰레기 값
2. 개수 불일치(%d 3개, 값 2개) → 쓰레기 값
3. 문자에 큰따옴표("A") → 타입 불일치
4. %를 %%로 쓰지 않음 → 잘못된 해석

디버깅 원칙: 출력이 이상하면 서식 지정자를 먼저 확인하십시오.

- %.Nf — 소수점 N자리 출력

- 서식 지정자를 쓸 때 바로 옆에 대응하는 값을 확인합니다
- %f보다 %.2f처럼 자릿수를 명시합니다
- 디버깅용 출력에는 변수 이름을 함께 출력합니다: printf("score=%d\n", score);
축 8: 타과목 연결 [선택] 수학과의 연결:
- 8진수, 16진수는 수학의 진법 변환과 같습니다
- 10 = 12(8진) = A(16진) = 1010(2진)
- 컴퓨터가 내부적으로 2진수를 사용하므로, 16진수(4비트 단위)가 자주 쓰입니다
