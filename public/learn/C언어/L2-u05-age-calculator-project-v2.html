Chapter 05

내 나이 계산기

변수, 자료형, 서식 지정자를 종합하여 나이 계산 프로그램을 만들고, 다양한 형태로 확장합니다.

이 유닛에서 배우는 것

  • 변수에 연도·월·일을 저장하고 나이를 계산할 수 있습니다.
  • 서식 지정자를 활용하여 정렬된 출력을 할 수 있습니다.
  • 여러 변수를 조합하여 하나의 완성된 프로그램을 만들 수 있습니다.
  • 프로그램을 단계별로 확장하는 개발 과정을 경험합니다.

선수 학습

L2-U01~U04 변수, 이름 규칙, int 연산, 서식 지정자
예측해 봅시다
아래 코드를 실행하면 어떤 결과가 출력될지 예측해 보십시오.
#include <stdio.h>

int main(void) {
int birth_year = 2010;
int current_year = 2026;
int age = current_year - birth_year;
printf("%d년생, 만 %d세\n", birth_year, age);
return 0;
}

프로젝트 소개

이번 프로젝트에서는 나이 계산기를 만듭니다. 간단한 버전에서 시작하여 3단계로 확장합니다.

1
버전 1: 기본 나이 계산
출생연도와 현재연도로 만 나이를 계산합니다.
2
버전 2: 상세 프로필
이름, 생년월일, 혈액형 등 다양한 정보를 포함합니다.
3
버전 3: 다중 프로필 비교
여러 사람의 정보를 정렬된 형태로 출력합니다.

버전 1: 기본 나이 계산

Cage_v1.c
1#include <stdio.h>
2
3int main(void) {
4 int birth_year = 2010;
5 int current_year = 2026;
6 int age = current_year - birth_year;
7
8 printf("=== 나이 계산기 ===\n");
9 printf("출생연도: %d년\n", birth_year);
10 printf("현재연도: %d년\n", current_year);
11 printf("만 나이: %d세\n", age);
12 printf("==================\n");
13 return 0;
14}
실행 결과
=== 나이 계산기 === 출생연도: 2010년 현재연도: 2026년 만 나이: 16세 ==================
핵심 포인트 1. 변수 이름에 snake_case를 사용했습니다 (birth_year, current_year)
2. 나이 = 현재연도 - 출생연도 (간단한 산술 연산)
3. printf와 %d로 변수 값을 출력합니다

이름, 생년월일, 혈액형 등 다양한 정보를 추가합니다. 여러 서식 지정자(%d, %c, %s)를 함께 사용합니다.

실행 결과

개발 과정 프로그램은 한 번에 완성하지 않습니다. 버전 1의 간단한 코드에서 시작하여 기능을 하나씩 추가합니다. 이것을 점진적 개발(incremental development)이라고 합니다.

1. 먼저 동작하는 코드를 만듭니다
2. 컴파일·실행하여 확인합니다
3. 기능을 하나 추가합니다
4. 다시 컴파일·실행합니다
5. 반복합니다

버전 3: 다중 프로필 비교

여러 사람의 정보를 정렬된 표 형태로 출력합니다. 자릿수 지정(%5d, %-8s)을 활용합니다.

Cage_v3.c
1#include <stdio.h>
2
3int main(void) {
4 /* 학생 1 */
5 int year_1 = 2009;
6 int score_1 = 92;
7 char grade_1 = 'A';
8
9 /* 학생 2 */
10 int year_2 = 2010;
11 int score_2 = 85;
12 char grade_2 = 'B';
13
14 /* 학생 3 */
15 int year_3 = 2011;
16 int score_3 = 78;
17 char grade_3 = 'C';
18
19 int current = 2026;
20
21 printf("+------+------+------+------+------+\n");
22 printf("| 번호 | 출생 | 나이 | 점수 | 학점 |\n");
23 printf("+------+------+------+------+------+\n");
24 printf("| %3d | %4d | %3d | %3d | %c |\n",
25 1, year_1, current - year_1, score_1, grade_1);
26 printf("| %3d | %4d | %3d | %3d | %c |\n",
27 2, year_2, current - year_2, score_2, grade_2);
28 printf("| %3d | %4d | %3d | %3d | %c |\n",
29 3, year_3, current - year_3, score_3, grade_3);
30 printf("+------+------+------+------+------+\n");
31
32 /* 평균 계산 */
33 int avg = (score_1 + score_2 + score_3) / 3;
34 printf("| 평균 점수: %d점 |\n", avg);
35 printf("+------+------+------+------+------+\n");
36 return 0;
37}
실행 결과
+------+------+------+------+------+ | 번호 | 출생 | 나이 | 점수 | 학점 | +------+------+------+------+------+ | 1 | 2009 | 17 | 92 | A | | 2 | 2010 | 16 | 85 | B | | 3 | 2011 | 15 | 78 | C | +------+------+------+------+------+ | 평균 점수: 85점 | +------+------+------+------+------+
자릿수 지정으로 표 만들기 %3d — 3칸에 오른쪽 정렬하여 정수 출력
%4d — 4칸에 오른쪽 정렬하여 정수 출력
%c — 문자 1개 출력

각 열의 폭을 고정하면 깔끔한 표를 만들 수 있습니다.

BMI = 몸무게(kg) * 10000 / (키(cm) * 키(cm))

키: 165cm
몸무게: 55kg
BMI: 20

에러 1: 변수 이름 오타 에러: error: 'brith' undeclared
원인: birth를 brith로 잘못 입력했습니다.
해결: 변수 이름의 철자를 확인합니다. 에디터의 자동 완성(Ctrl+Space)을 활용하면 오타를 줄일 수 있습니다.
Cerr2.c
1#include <stdio.h>
2
3int main(void) {
4 int height = 165;
5 int weight = 55;
6 int bmi = weight / (height * height) * 10000;
7 printf("BMI: %d\n", bmi); /* 0 출력! */
8 return 0;
9}
에러 2: 정수 나눗셈 순서 증상: BMI가 0으로 출력됩니다.
원인: 55 / (165 * 165) = 55 / 27225 = 0 (정수 나눗셈)
수정: 곱셈을 먼저: weight * 10000 / (height * height)
→ 550000 / 27225 = 20
