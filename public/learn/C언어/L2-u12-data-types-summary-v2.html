자료형 총정리

C언어의 기본 자료형을 크기·범위·서식 지정자 중심으로 총정리합니다.

이 유닛에서 배우는 것

  • C언어의 정수형(short, int, long, long long)과 실수형(float, double)을 비교할 수 있습니다.
  • 각 자료형의 크기(바이트)와 값의 범위를 설명할 수 있습니다.
  • 상황에 맞는 자료형을 선택할 수 있습니다.
  • unsigned 키워드의 의미와 효과를 이해합니다.

핵심 규칙 3가지
1. 정수는 int를 기본으로 사용합니다. int 범위를 넘는 큰 수에만 long long을 사용합니다.
2. 실수는 double을 기본으로 사용합니다. float는 정밀도가 낮아 권장하지 않습니다.
3. 문자는 char를 사용합니다. char는 사실 1바이트 정수형이기도 합니다.

에러 체험

undefined

에러 1: short 오버플로
short s = 40000;
printf("%d\n", s);

→ short의 최댓값은 32,767입니다. 40,000은 범위를 넘어 음수(-25536)가 출력됩니다.
에러 2: 서식 지정자 불일치
long long big = 9000000000LL;
printf("%d\n", big); // ✗ %d는 int용
printf("%lld\n", big); // ✓ long long은 %lld

→ %d로 출력하면 엉뚱한 값이 나옵니다. 자료형과 서식 지정자를 반드시 맞추어야 합니다.
에러 3: LL 접미사 누락
long long big = 9000000000; // LL 없음

→ 컴파일러가 9000000000을 int로 해석하려다 경고가 발생합니다. 큰 수에는 반드시 LL을 붙이십시오.

대회 기출

요약

핵심 정리
1. 정수형: char(1B) → short(2B) → int(4B) → long(4B) → long long(8B)
2. 실수형: float(4B, 7자리) → double(8B, 15자리)
3. 기본 선택: 정수=int, 실수=double, 문자=char
4. unsigned: 음수 포기 → 양수 범위 2배
5. long long 리터럴에는 LL, float 리터럴에는 f 접미사
