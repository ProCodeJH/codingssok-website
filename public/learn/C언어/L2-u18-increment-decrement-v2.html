증감 연산자 ++, -- (전위와 후위)

변수를 1씩 증가/감소시키는 ++, -- 연산자와 전위/후위의 차이를 학습합니다.

이 유닛에서 배우는 것

  • ++ 연산자와 -- 연산자의 동작을 이해합니다.
  • 전위 연산(++a)과 후위 연산(a++)의 차이를 설명할 수 있습니다.
  • 증감 연산자를 독립 문장으로 안전하게 사용할 수 있습니다.
  • 증감 연산자가 반복문에서 어떻게 쓰이는지 미리 살펴봅니다.

증감 연산자 기본
a++ 또는 ++a → a를 1 증가 (a = a + 1)
a-- 또는 --a → a를 1 감소 (a = a - 1)

독립 문장으로 쓸 때는 전위/후위 차이가 없습니다.
a++; 과 ++a; 모두 a를 1 증가시킵니다.
C
1#include <stdio.h>
2int main(void) {
3 int count = 0;
4 printf("초기값: %d\n", count);
5
6 count++; // 1 증가
7 printf("count++: %d\n", count);
8
9 count++; // 또 1 증가
10 printf("count++: %d\n", count);
11
12 count--; // 1 감소
13 printf("count--: %d\n", count);
14
15 return 0;
16}
실행 결과
초기값: 0 count++: 1 count++: 2 count--: 1

전위 vs 후위

전위(Prefix) ++a
1. 먼저 a를 1 증가시킨다.
2. 증가된 값을 사용한다.

int a = 5;
int b = ++a;
→ a가 먼저 6이 되고, b에 6이 대입됩니다.
→ a = 6, b = 6
후위(Postfix) a++
1. 현재 a 값을 먼저 사용한다.
2. 그 후 a를 1 증가시킨다.

int a = 5;
int b = a++;
→ b에 현재 값 5가 대입되고, 그 후 a가 6이 됩니다.
→ a = 6, b = 5
C
1#include <stdio.h>
2int main(void) {
3 int a, b;
4
5 // 전위: 먼저 증가, 후에 사용
6 a = 10;
7 b = ++a;
8 printf("전위: a=%d, b=%d\n", a, b); // a=11, b=11
9
10 // 후위: 먼저 사용, 후에 증가
11 a = 10;
12 b = a++;
13 printf("후위: a=%d, b=%d\n", a, b); // a=11, b=10
14
15 return 0;
16}
실행 결과
전위: a=11, b=11 후위: a=11, b=10

에러 체험

에러 1: 상수에 증감 연산
5++; // ✗ 컴파일 에러
(a+b)++; // ✗ 컴파일 에러

→ ++와 --는 변수에만 사용할 수 있습니다. 상수나 수식에는 사용할 수 없습니다.
에러 2: 미정의 동작
int a = 5;
int b = a++ + a++; // ✗ 미정의 동작!

→ 같은 변수에 대해 한 문장에서 두 번 이상 증감하면 결과가 보장되지 않습니다.
→ 컴파일러마다 다른 값이 나올 수 있습니다. 절대 이렇게 쓰지 마십시오.
에러 3: 전위/후위 혼동
int a = 5;
printf("%d\n", a++); // 5 출력 (사용 후 증가)
// a는 이제 6이지만 출력된 값은 5

→ 후위 연산은 출력된 값과 이후의 변수 값이 다릅니다. 헷갈리면 독립 문장으로 쓰십시오.

핵심 정리
1. a++ / ++a: a를 1 증가 (a = a + 1의 축약)
2. a-- / --a: a를 1 감소 (a = a - 1의 축약)
3. 전위(++a): 먼저 증가 → 값 사용
4. 후위(a++): 값 먼저 사용 → 나중에 증가
5. 안전한 사용: 독립 문장 (a++;), 다른 연산과 섞지 않기
