C언어에서 참과 거짓

참과 거짓의 규칙
거짓: 0 (오직 0만 거짓)
참: 0이 아닌 모든 값 (1, -1, 42, 100 등)

비교 연산자는 참이면 1, 거짓이면 0을 반환합니다.

에러 체험

에러 1: = 와 == 혼동
if (score = 100) { // ✗ 대입! 항상 참이 됨
printf("만점!\n");
}

→ score = 100은 score에 100을 대입하고, 그 값(100)이 조건식이 됩니다.
→ 100은 0이 아니므로 항상 참입니다.
→ if (score == 100)이 올바릅니다.
에러 2: 세미콜론 실수
if (score >= 60); { // ✗ if 뒤에 세미콜론!
printf("합격\n");
}

→ 세미콜론이 if의 본문(빈 문장)이 됩니다.
→ 중괄호 안의 코드는 조건과 상관없이 항상 실행됩니다.
에러 3: 중괄호 없이 여러 줄
if (score >= 60)
printf("합격\n");
printf("축하합니다\n"); // 항상 실행됨!

→ 중괄호가 없으면 if는 바로 다음 한 줄만 제어합니다.
→ "축하합니다"는 조건과 상관없이 항상 출력됩니다.
