에러 체험

에러 1: else에 조건식 붙이기
if (score >= 60) {
printf("합격\n");
} else (score < 60) { // ✗ else에 조건 붙이면 안 됨
printf("불합격\n");
}

→ else는 조건식 없이 씁니다. 조건이 필요하면 else if를 사용합니다.
에러 2: else 위치 오류
if (x > 0) {
printf("양수\n");
}
printf("중간 코드\n");
else { // ✗ if와 else 사이에 다른 코드!
printf("음수 또는 영\n");
}

→ else는 반드시 if의 닫는 중괄호 바로 뒤에 와야 합니다.
에러 3: if-else에서 둘 다 실행된다고 착각
if (score >= 60) {
printf("합격\n");
} else {
printf("불합격\n");
}

→ 이 코드에서 "합격"과 "불합격"이 동시에 출력되는 것은 불가능합니다.
→ if-else는 반드시 둘 중 하나만 실행됩니다.
