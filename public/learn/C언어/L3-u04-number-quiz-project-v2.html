  • if-else if-else 구조를 활용하여 판정 프로그램을 만들 수 있습니다.
  • 정답과 입력값의 차이를 계산하여 힌트를 제공할 수 있습니다.
  • 프로그램을 3단계로 점진적으로 발전시키는 개발 방법을 적용합니다.
  • 조건문과 산술 연산을 결합한 문제를 해결할 수 있습니다.

에러 체험

에러 1: 절댓값 계산 실수
int diff = guess - answer;
if (diff <= 5) { ... } // diff가 음수일 수 있음!

→ guess=37, answer=42이면 diff=-5입니다.
→ -5 <= 5는 참이지만, 의도한 것은 거리(5)입니다.
→ 음수이면 -diff로 바꿔야 합니다.
에러 2: 조건 중복 검사
if (guess == answer) { ... }
if (diff <= 5) { ... } // guess==answer일 때도 실행!

→ 독립 if이므로 정답일 때 "정답!"과 "아깝습니다!"가 동시에 출력됩니다.
→ else if를 사용하여 하나만 실행되게 해야 합니다.

요약

핵심 정리
1. 숫자 맞히기: 정답 비교 → 같으면 정답, 크면/작으면 힌트
2. 절댓값(차이): diff < 0이면 diff = -diff
3. 차이 기반 세분화: diff <= 5 아깝다, diff <= 20 가깝다
4. 3단계 개발: 기본 → 힌트 추가 → 여러 번 시도
5. 반복 코드는 Layer 4(반복문)에서 개선 예정
