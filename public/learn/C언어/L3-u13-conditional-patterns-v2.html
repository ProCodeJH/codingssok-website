  • 문제 유형에 따라 적합한 조건문을 선택할 수 있습니다.

패턴별 코드 모음

패턴 1: 범위 분류
if (x >= 90) ...
else if (x >= 80) ...
else if (x >= 70) ...
→ 학점, 등급, 요금 구간 등
패턴 2: 메뉴 선택
switch (menu) {
case 1: ... break;
case 2: ... break;
}
→ 메뉴, 요일, 월 등 정확한 값 대응
패턴 3: 입력 검증
if (x < 0 || x > 100) {
printf("범위 오류\n");
return 1;
}
→ 잘못된 입력을 먼저 걸러냄
패턴 4: 최대/최소
int max = (a > b) ? a : b;
→ 삼항 연산자로 간결하게
패턴 5: 플래그 검사
if (found && count > 0) {
printf("결과 있음\n");
}
→ 논리 연산자로 복합 조건

에러 체험

에러 1: switch에 범위 조건 사용 시도
switch (score) {
case score >= 90: // 컴파일 에러!
grade = 'A';
}

→ case에는 상수만 가능. 범위 비교는 else if 사용.
에러 2: else if 없이 if만 나열
if (s >= 90) grade = 'A';
if (s >= 80) grade = 'B'; // 90점도 여기 진입!
if (s >= 70) grade = 'C'; // 90점도 여기 진입!

→ if만 나열하면 모든 조건을 독립적으로 검사합니다.
→ else if를 사용하면 하나만 실행됩니다.
에러 3: 삼항으로 복잡한 로직
(a > b) ? (printf("big\n"), x = a) : (printf("small\n"), x = b);

→ 문법은 맞지만 읽기 어렵습니다.
→ if-else가 훨씬 명확합니다.
