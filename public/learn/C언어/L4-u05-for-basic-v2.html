에러 체험

에러 1: 세미콜론 위치 오류
for (int i = 0; i < 5; i++); ← 세미콜론!
{
printf("%d\n", i);
}

→ 세미콜론이 for의 빈 본문이 됩니다.
→ 블록은 for와 무관하게 한 번만 실행됩니다.
에러 2: 쉼표와 세미콜론 혼동
for (int i = 0, i < 5, i++) // 컴파일 에러!

→ for문의 세 요소는 세미콜론(;)으로 구분합니다.
→ for (int i = 0; i < 5; i++)
에러 3: 변화식 빠뜨림
for (int i = 0; i < 5; ) { // i++ 없음
printf("%d\n", i);
}

→ i가 변하지 않으므로 무한 루프.
→ 변화식을 반드시 작성하십시오.

대회 기출

요약

핵심 정리
1. for문: for (초기화; 조건; 변화) { 본문 }
2. 실행 순서: 초기화(1회) → 조건 → 본문 → 변화 → 조건 → ...
3. while 대응: 세 요소를 한 줄에 모은 것
4. 변수 범위: for 안에서 선언한 변수는 for 밖에서 사용 불가
5. 적합한 상황: 횟수가 정해진 반복
