for문으로 별 찍기 패턴

for문으로 별(*)을 찍어 직각삼각형, 역삼각형, 직사각형 등 기본 도형 패턴을 만드는 연습을 합니다.

이 유닛에서 배우는 것

  • for문으로 한 줄에 별을 N개 출력할 수 있습니다.
  • 중첩 for문 없이 단일 for문으로 기본 패턴을 만들 수 있습니다.
  • 줄 번호와 별 개수의 관계를 파악하여 패턴을 설계할 수 있습니다.
  • printf의 반복으로 도형을 그리는 사고를 훈련합니다.

패턴 3 — 역삼각형

C
1#include <stdio.h>
2int main(void) {
3 int n;
4 scanf("%d", &n);
5 for (int i = n; i >= 1; i--) { // n부터 1까지
6 for (int j = 0; j < i; j++) { // i개의 별
7 printf("*");
8 }
9 printf("\n");
10 }
11 return 0;
12}
실행 결과
5 ***** **** *** ** *
줄 번호와 별 개수의 관계
1번째 줄 → 별 n개
2번째 줄 → 별 n-1개
...
n번째 줄 → 별 1개

방법 1: i를 n부터 1까지 감소
방법 2: i를 1부터 n까지, 별 (n-i+1)개

패턴 4 — 직각삼각형 (오른쪽 정렬)

C
1#include <stdio.h>
2int main(void) {
3 int n;
4 scanf("%d", &n);
5 for (int i = 1; i <= n; i++) {
6 // 공백 (n-i)개
7 for (int j = 0; j < n - i; j++) {
8 printf(" ");
9 }
10 // 별 i개
11 for (int j = 0; j < i; j++) {
12 printf("*");
13 }
14 printf("\n");
15 }
16 return 0;
17}
실행 결과
5 * ** *** **** *****
줄 번호와 공백/별의 관계
i=1 → 공백 4개 + 별 1개
i=2 → 공백 3개 + 별 2개
i=3 → 공백 2개 + 별 3개
i=4 → 공백 1개 + 별 4개
i=5 → 공백 0개 + 별 5개

규칙: 공백 (n-i)개 + 별 i개 = 항상 n개

패턴 분석 방법

에러 체험

에러 1: 줄바꿈 위치 오류
for (int i = 1; i <= n; i++) {
for (int j = 0; j < i; j++) {
printf("*\n"); // 별마다 줄바꿈!
}
}

→ 별이 세로로 한 줄에 하나씩 출력됩니다.
→ printf("\n")은 안쪽 for 바깥에 작성하십시오.
에러 2: 공백을 빠뜨림
// 오른쪽 정렬인데 공백 없이 별만 출력
for (int i = 1; i <= n; i++) {
for (int j = 0; j < i; j++) printf("*");
printf("\n");
}
// 결과: 왼쪽 정렬이 됨

→ 별 앞에 공백을 출력하는 for문을 추가하십시오.
에러 3: 안쪽 for의 조건 오류
for (int i = 1; i <= n; i++) {
for (int j = 0; j < n; j++) { // i가 아니라 n!
printf("*");
}
printf("\n");
}
// 결과: 직사각형이 됨 (삼각형이 아님)

→ 안쪽 for의 조건을 j < i로 수정하십시오.

요약

핵심 정리
1. 중첩 for: 바깥 = 줄(i), 안쪽 = 열(j)
2. 직각삼각형: j < i (별 i개)
3. 역삼각형: i를 n부터 감소 또는 별 n-i+1개
4. 오른쪽 정렬: 공백 (n-i)개 + 별 i개
5. 패턴 분석: 줄 번호와 공백/별 개수의 관계 파악이 핵심
