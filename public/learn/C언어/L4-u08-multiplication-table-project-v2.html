버전 2 — 가로 배치 (핵심 기법)

C
1#include <stdio.h>
2int main(void) {
3 for (int j = 1; j <= 9; j++) { // 곱하는 수 (줄)
4 for (int i = 2; i <= 9; i++) { // 단 (열)
5 printf("%d x %d = %2d ", i, j, i * j);
6 }
7 printf("\n");
8 }
9 return 0;
10}
실행 결과
2 x 1 = 2 3 x 1 = 3 4 x 1 = 4 5 x 1 = 5 6 x 1 = 6 7 x 1 = 7 8 x 1 = 8 9 x 1 = 9 2 x 2 = 4 3 x 2 = 6 4 x 2 = 8 5 x 2 = 10 6 x 2 = 12 7 x 2 = 14 8 x 2 = 16 9 x 2 = 18 ...
세로 vs 가로의 핵심 차이
세로 출력: 바깥=단(i), 안쪽=곱(j) → 한 단을 다 출력하고 다음 단
가로 배치: 바깥=곱(j), 안쪽=단(i) → 같은 줄에 모든 단의 같은 행

i와 j의 역할을 뒤집는 것이 가로 배치의 핵심입니다.

버전 3 — 4단씩 나누어 출력

C
1#include <stdio.h>
2int main(void) {
3 // 2~5단
4 printf("========== 구구단 (2~5단) ==========\n");

8 }
9 printf("\n");
10 }
11
12 printf("\n");
13
14 // 6~9단
15 printf("========== 구구단 (6~9단) ==========\n");
16 for (int j = 1; j <= 9; j++) {
17 for (int i = 6; i <= 9; i++) {
18 printf("%d x %d = %2d ", i, j, i * j);
19 }
20 printf("\n");
21 }
22
23 // 통계
24 printf("\n========== 통계 ==========\n");
25 int total = 0;
26 for (int i = 2; i <= 9; i++) {
27 int row_sum = 0;

29 row_sum += i * j;
30 }

33 }

38}

에러 체험

에러 1: i와 j를 뒤바꿈
for (int i = 2; i <= 9; i++) {
for (int j = 1; j <= 9; j++) {
printf("%d x %d = %2d ", j, i, j * i); // j와 i 혼동
}
printf("\n");
}

→ 출력은 되지만 형식이 뒤바뀝니다.
→ 어떤 변수가 단이고 어떤 변수가 곱인지 명확히 하십시오.
에러 2: 가로 배치에서 줄바꿈 위치
for (int j = 1; j <= 9; j++) {
for (int i = 2; i <= 9; i++) {
printf("%d x %d = %2d\n", i, j, i * j); // \n이 안쪽에!
}
}

→ 각 항목마다 줄바꿈되어 세로로 출력됩니다.
→ \n은 안쪽 for 바깥에 작성하십시오.
에러 3: 서식 미정렬
printf("%d x %d = %d", i, j, i*j);
// 2 x 9 = 18과 9 x 9 = 81의 자릿수가 달라 정렬이 깨짐

→ printf("%d x %d = %2d", i, j, i*j);
→ %2d로 최소 2자리 출력하여 정렬하십시오.
