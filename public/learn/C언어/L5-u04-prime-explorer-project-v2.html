에러 체험

에러 1: 소인수분해에서 n 값 변경 후 복구 안 함
case 3: {
int n;
scanf("%d", &n);
for (int i = 2; i <= n; i++)
while (n % i == 0) n /= i; // n이 1이 됨
// 이후 n을 다시 사용하면 1
}

→ 원본 값이 필요하면 별도 변수에 저장하십시오.
에러 2: 범위 소수에서 시작값 처리 누락
for (int n = a; n <= b; n++) // a가 0이나 1이면 소수로 오판

→ 시작값이 2 미만이면 2부터 시작하도록 처리하십시오.
→ int start = (a < 2) ? 2 : a;
에러 3: GCD에서 0 입력
scanf("%d %d", &a, &b); // b가 0이면?
while (tb != 0) { ... } // 즉시 종료, ta = a

→ GCD(a, 0) = a는 수학적으로 맞지만 사용자에게 안내가 필요합니다.

설명하기

설명하기

소인수분해에서 while(n % i == 0) n /= i 패턴이 소인수를 정확히 추출하는 이유를 설명하십시오. i가 합성수일 때는 왜 이미 처리가 끝나 있는지도 서술하십시오.
