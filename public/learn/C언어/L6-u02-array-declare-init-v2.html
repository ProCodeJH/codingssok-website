sizeof로 배열 크기 계산

C
1#include <stdio.h>
2int main(void) {
3 int arr[] = {10, 20, 30, 40, 50};
4
5 int total_bytes = sizeof(arr); // 배열 전체 바이트
6 int elem_bytes = sizeof(arr[0]); // 요소 하나 바이트
7 int count = sizeof(arr) / sizeof(arr[0]); // 요소 개수
8
9 printf("배열 전체: %d바이트\n", total_bytes);
10 printf("요소 하나: %d바이트\n", elem_bytes);
11 printf("요소 개수: %d\n", count);
12
13 // 요소 개수로 반복
14 for (int i = 0; i < count; i++)
15 printf("%d ", arr[i]);
16 printf("\n");
17 return 0;
18}
실행 결과
배열 전체: 20바이트 요소 하나: 4바이트 요소 개수: 5 10 20 30 40 50
배열 크기 공식
요소 개수 = sizeof(배열) / sizeof(배열[0])

int arr[5] → sizeof(arr) = 20, sizeof(arr[0]) = 4
요소 개수 = 20 / 4 = 5

에러 체험

에러 1: 배열 크기에 변수 사용 (C89)
int n = 5;
int arr[n]; // C99에서는 허용, C89에서는 에러

→ 일부 컴파일러에서는 VLA(가변 길이 배열)를 지원하지 않습니다.
→ 안전하게는 상수 또는 #define을 사용하십시오.
에러 2: 선언 후 중괄호 초기화 시도
int arr[5];
arr = {1, 2, 3, 4, 5}; // 에러! 선언 이후 불가

→ 중괄호 초기화는 선언과 동시에만 가능합니다.
→ 이후에는 arr[0] = 1; arr[1] = 2; 처럼 개별 대입
에러 3: 배열 크기 생략 시 초기화 누락
int arr[]; // 에러! 크기도 없고 초기화도 없음

→ 크기를 생략하려면 반드시 초기화해야 합니다.
→ int arr[] = {1, 2, 3}; // 크기 3으로 자동 결정

요약

핵심 정리
1. 선언: int arr[N]; 또는 int arr[] = {...};
2. 부분 초기화: int arr[5] = {1, 2}; → 나머지는 0
3. 전체 0 초기화: int arr[N] = {0};
4. sizeof: 요소 수 = sizeof(arr) / sizeof(arr[0])
5. 주의: 초기화 안 하면 쓰레기 값, 선언 후 {} 대입 불가
