배열에서 값 찾기 (선형 탐색)

선형 탐색 알고리즘으로 배열에서 특정 값을 찾고, 탐색 성공/실패 처리와 다양한 변형을 학습합니다.

이 유닛에서 배우는 것

  • 선형 탐색의 원리를 이해하고 구현할 수 있습니다.
  • 탐색 성공과 실패를 구분하여 처리할 수 있습니다.
  • 첫 번째 위치, 마지막 위치, 등장 횟수 등 변형을 구현할 수 있습니다.
  • 선형 탐색의 시간 복잡도 O(n)을 이해합니다.

핵심 개념

선형 탐색(Linear Search)
배열의 처음부터 끝까지 순서대로 비교하여 값을 찾는 알고리즘.

시간 복잡도: O(n)
최선: 첫 번째에서 발견 → 1번 비교
최악: 마지막에서 발견 또는 없음 → n번 비교
평균: n/2번 비교

기본 선형 탐색

C
1#include <stdio.h>
2int main(void) {
3 int arr[] = {34, 78, 12, 95, 43, 67, 89, 21};
4 int n = sizeof(arr) / sizeof(arr[0]);
5 int target;
6
7 printf("찾을 값: ");
8 scanf("%d", &target);
9
10 int found = -1;
11 for (int i = 0; i < n; i++) {
12 if (arr[i] == target) {
13 found = i;
14 break;
15 }
16 }
17
18 if (found != -1)
19 printf("%d은(는) 인덱스 %d에 있습니다.\n", target, found);
20 else
21 printf("%d은(는) 배열에 없습니다.\n", target);
22 return 0;
23}
실행 결과
찾을 값: 67 67은(는) 인덱스 5에 있습니다.

등장 횟수 세기

C
1#include <stdio.h>
2int main(void) {
3 int arr[] = {3, 7, 3, 1, 7, 3, 9, 7, 4};
4 int n = sizeof(arr) / sizeof(arr[0]);
5 int target;
6
7 printf("찾을 값: ");
8 scanf("%d", &target);
9
10 int count = 0;
11 for (int i = 0; i < n; i++) {
12 if (arr[i] == target)
13 count++;
14 }
15
16 printf("%d은(는) %d번 등장합니다.\n", target, count);
17 return 0;
18}
실행 결과
찾을 값: 3 3은(는) 3번 등장합니다.

C
1#include <stdio.h>

C
1#include <stdio.h>
2int main(void) {

5
6 int max = arr[0];

9

조건부 탐색

C
1#include <stdio.h>
2int main(void) {
3 int scores[] = {85, 42, 93, 67, 78, 31, 95, 88};
4 int n = sizeof(scores) / sizeof(scores[0]);
5
6 // 60점 미만인 학생 찾기
7 printf("60점 미만:\n");
8 for (int i = 0; i < n; i++) {
9 if (scores[i] < 60)
10 printf(" %d번: %d점\n", i + 1, scores[i]);
11 }
12
13 // 90점 이상 첫 번째 학생
14 int first_a = -1;
15 for (int i = 0; i < n; i++) {
16 if (scores[i] >= 90) {
17 first_a = i;
18 break;
19 }
20 }
21 if (first_a != -1)
22 printf("첫 90점 이상: %d번 (%d점)\n",
23 first_a + 1, scores[first_a]);
24 return 0;
25}
실행 결과
60점 미만: 2번: 42점 6번: 31점 첫 90점 이상: 3번 (93점)

에러 체험

에러 1: break 없이 첫 번째만 찾으려 할 때
int pos;
for (int i = 0; i < n; i++)
if (arr[i] == target) pos = i; // 마지막 위치가 저장됨!

→ 첫 번째 위치를 찾으려면 break를 사용하십시오.
에러 2: found 변수 초기화 누락
int found; // 초기화 안 함!
for (int i = 0; i < n; i++)
if (arr[i] == target) { found = i; break; }
if (found != -1) ... // found가 쓰레기 값일 수 있음!

→ int found = -1;로 초기화하십시오.
에러 3: 반복문 안에서 '없음' 출력
for (int i = 0; i < n; i++) {
if (arr[i] == target) printf("찾음\n");
else printf("없음\n"); // 매번 출력!
}

→ 반복문이 끝난 후 found 변수로 판단하십시오.

핵심 정리
1. 선형 탐색: 처음부터 끝까지 순서대로 비교 — O(n)
2. 첫 번째 위치: 찾으면 break, found = i
3. 등장 횟수: break 없이 끝까지 순회, count++
4. 모든 위치: break 없이 순회하며 각 위치 출력
5. 탐색 실패: found = -1로 초기화, 반복 후 판단
