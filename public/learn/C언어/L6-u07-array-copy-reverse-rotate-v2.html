배열의 내용을 복사하고, 요소 순서를 뒤집고, 왼쪽·오른쪽으로 회전하는 조작 기법을 학습합니다.

이 유닛에서 배우는 것

  • 배열을 다른 배열로 복사하는 방법을 구현할 수 있습니다.
  • 배열 요소의 순서를 뒤집을 수 있습니다.
  • 배열을 왼쪽/오른쪽으로 회전할 수 있습니다.

1#include <stdio.h>
2int main(void) {
3 int src[] = {10, 20, 30, 40, 50};
4 int n = sizeof(src) / sizeof(src[0]);

8 for (int i = 0; i < n; i++)
9 dst[i] = src[i];
10
11 // 복사 확인
12 printf("원본: ");
13 for (int i = 0; i < n; i++) printf("%d ", src[i]);
14 printf("\n복사: ");
15 for (int i = 0; i < n; i++) printf("%d ", dst[i]);
16
17 // dst를 변경해도 src는 영향 없음
18 dst[0] = 99;
19 printf("\n\ndst[0] = 99 후:\n");
20 printf("원본: %d, 복사: %d\n", src[0], dst[0]);

22}

배열 반전 (in-place)

C
1#include <stdio.h>
2int main(void) {
3 int arr[] = {1, 2, 3, 4, 5};
4 int n = sizeof(arr) / sizeof(arr[0]);
5
6 printf("반전 전: ");
7 for (int i = 0; i < n; i++) printf("%d ", arr[i]);
8
9 // 양쪽 끝에서 중앙으로 swap
10 for (int i = 0; i < n / 2; i++) {
11 int temp = arr[i];
12 arr[i] = arr[n - 1 - i];
13 arr[n - 1 - i] = temp;
14 }
15
16 printf("\n반전 후: ");
17 for (int i = 0; i < n; i++) printf("%d ", arr[i]);
18 printf("\n");
19 return 0;
20}
실행 결과
반전 전: 1 2 3 4 5 반전 후: 5 4 3 2 1

에러 체험

에러 1: 배열 대입으로 복사 시도
int b[5] = a; // 에러!
int b[] = a; // 에러!

→ C에서 배열은 = 로 복사할 수 없습니다.
→ 반복문으로 요소를 하나씩 복사하십시오.
에러 2: 회전에서 첫 요소 저장 누락
for (int i = 0; i < n-1; i++)
arr[i] = arr[i+1]; // arr[0] 원본이 사라짐!
arr[n-1] = arr[0]; // 이미 변경된 값!

→ 회전 전에 int first = arr[0];으로 보존하십시오.
에러 3: 반전에서 전체 순회
for (int i = 0; i < n; i++) { // n/2가 아닌 n!
int t = arr[i]; arr[i] = arr[n-1-i]; arr[n-1-i] = t;
}
// 앞반은 뒤집고, 뒷반에서 다시 원래대로!

→ i < n/2까지만 반복해야 합니다.

설명하기

in-place 반전에서 반복 범위가 n/2인 이유를 설명하십시오. n이 홀수일 때 가운데 요소가 어떻게 처리되는지도 서술하십시오.

대회 기출

요약

핵심 정리
1. 복사: for문으로 dst[i] = src[i] (= 대입 불가)
2. 반전: 양쪽 끝에서 swap, i < n/2까지
3. 왼쪽 회전: 첫 요소 저장 → 앞으로 밀기 → 끝에 삽입
4. 오른쪽 회전: 마지막 저장 → 뒤로 밀기 → 앞에 삽입
5. 삽입/삭제: 뒤로 밀기/앞으로 당기기 + n 조정
