대회 기출

요약

핵심 정리
1. 2차원 배열: int a[행][열] — 표/행렬 저장
2. 초기화: 행별 중괄호 {{}, {}, {}} 또는 일렬 나열
3. 순회: 바깥 for = 행, 안쪽 for = 열
4. 메모리: 행 우선 저장(row-major), 연속 배치
5. 행렬 연산: 덧셈(같은 위치 더하기), 전치(행↔열 교환)

용어설명
2차원 배열행(row)과 열(column) 두 개의 인덱스로 접근하는 배열. int a[행][열] 형태.
행 우선 저장2차원 배열이 메모리에 행 단위로 연속 저장되는 방식(row-major order).
전치(transpose)행렬의 행과 열을 뒤바꾸는 연산. a[i][j] → t[j][i].
주대각선정방행렬에서 a[0][0], a[1][1], ..., a[n-1][n-1] 위치의 원소들.
