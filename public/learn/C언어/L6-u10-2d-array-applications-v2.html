2차원 배열 활용

2차원 배열로 좌석 배치, 격자 탐색, 상하좌우 이동, 경계 검사 등 실전 패턴을 익힙니다.

이 유닛에서 배우는 것

  • 2차원 배열로 좌석표/격자판을 모델링할 수 있습니다.
  • 상하좌우 방향 배열(dx, dy)을 활용한 격자 탐색을 구현할 수 있습니다.
  • 경계 검사(boundary check)를 올바르게 작성할 수 있습니다.
  • 격자 기반 문제(색칠, 영역 카운팅)를 해결할 수 있습니다.

대회 기출

요약

핵심 정리
1. 좌석표: 0/1로 빈 좌석과 예약 구분, 카운팅
2. 방향 배열: dx[]={-1,1,0,0}, dy[]={0,0,-1,1}로 4방향 탐색
3. 8방향: dx[]={-1,-1,-1,0,0,1,1,1}, dy[]={-1,0,1,-1,1,-1,0,1}
4. 경계 검사: nx>=0 && nx=0 && ny
5. 격자 색칠: 영역을 1로 표시 후 전체 카운트
