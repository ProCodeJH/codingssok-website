대회 기출

요약

핵심 정리
1. 좌석 상태: int seat[행][열], 0=빈 좌석, 1=예약
2. 예약/취소: 경계 검사 + 상태 검사 후 값 변경
3. 현황: 2중 for로 예약 수 카운트, 예약률 계산
4. 연속 빈 좌석: 카운터 패턴 — 빈 좌석이면 count++, 아니면 count=0
5. 자동 배정: 연속 구간을 찾아 첫 번째 위치에 일괄 예약
