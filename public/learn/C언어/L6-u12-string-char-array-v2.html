문자열 = char 배열

C언어에서 문자열은 char 배열이며, 널 문자('\0')로 끝납니다. 문자열의 구조와 기본 조작을 학습합니다.

이 유닛에서 배우는 것

  • C언어에서 문자열이 char 배열 + 널 문자임을 이해할 수 있습니다.
  • 문자열을 선언하고 초기화하는 다양한 방법을 사용할 수 있습니다.
  • 문자열을 한 글자씩 순회하며 처리할 수 있습니다.
  • 널 문자의 역할과 문자열 길이의 관계를 설명할 수 있습니다.

실습: 문자열 기본 처리

C
1#include <stdio.h>
2int main(void) {
3 char str[100];
4 printf("문자열 입력: ");
5 scanf("%s", str);
6
7 // 1. 길이 계산
8 int len = 0;
9 while (str[len] != '\0') len++;
10
11 // 2. 대문자/소문자/숫자/기타 개수
12 int upper = 0, lower = 0, digit = 0, other = 0;
13 for (int i = 0; str[i] != '\0'; i++) {
14 if (str[i] >= 'A' && str[i] <= 'Z') upper++;
15 else if (str[i] >= 'a' && str[i] <= 'z') lower++;
16 else if (str[i] >= '0' && str[i] <= '9') digit++;
17 else other++;
18 }
19
20 printf("길이: %d\n", len);
21 printf("대문자: %d, 소문자: %d\n", upper, lower);
22 printf("숫자: %d, 기타: %d\n", digit, other);
23 return 0;
24}

핵심 정리
1. 문자열 = char 배열 + 널 문자('\0')
2. 선언: char str[] = "Hello"; → 크기 6 자동 결정
3. 순회: for (i=0; str[i]!='\0'; i++) 패턴
4. 입력: scanf("%s", str) — & 없음, 공백에서 끊김
5. 문자 조작: ASCII 연산 (대문자 +32 = 소문자)
