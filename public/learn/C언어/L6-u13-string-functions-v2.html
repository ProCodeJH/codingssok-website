strcmp 반환값 암기법
strcmp(a, b)의 반환값:
0 → a와 b가 같다
음수 → a가 사전에서 먼저 온다 (a < b)
양수 → a가 사전에서 나중에 온다 (a > b)
주의: == 로 문자열을 비교하면 주소를 비교하게 됩니다. 반드시 strcmp를 사용하십시오.

24 i++;
25 }

27}

대회 기출

요약

핵심 정리
1. strlen(s): 문자열 길이 반환 (\0 제외)
2. strcpy(dst, src): src를 dst에 복사 (= 대입 불가)
3. strcmp(s1, s2): 사전순 비교 (0이면 같음)
4. strcat(dst, src): dst 뒤에 src 이어붙이기
5. 공통 주의: dst 배열 크기가 충분해야 함
