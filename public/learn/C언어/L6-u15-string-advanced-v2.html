문자열 심화

문자열 분리(strtok), 문자열↔숫자 변환(atoi, sprintf), 문자열 검색(strstr) 등 심화 함수를 학습합니다.

이 유닛에서 배우는 것

  • strtok으로 문자열을 구분자 기준으로 분리할 수 있습니다.
  • atoi, atof로 문자열을 숫자로 변환할 수 있습니다.
  • sprintf로 서식화된 문자열을 만들 수 있습니다.
  • strstr로 문자열 내에서 부분 문자열을 검색할 수 있습니다.

strtok 사용 시 주의사항
1. 원본 문자열이 변경됩니다 (구분자가 \0으로 바뀜).
2. 문자열 리터럴에는 사용할 수 없습니다 (char *p = "abc" 불가).
3. 첫 호출은 문자열을 전달하고, 이후 호출은 NULL을 전달합니다.
4. 연속된 구분자는 하나로 취급됩니다.

대회 기출

요약

핵심 정리
1. atoi(s): 문자열→정수 변환, atof(s): 문자열→실수
2. sprintf(buf, fmt, ...): 서식화된 문자열을 buf에 저장
3. strtok(s, delim): 구분자 기준 문자열 분리 (원본 변경됨)
4. strstr(s, sub): 부분 문자열 검색, 찾으면 포인터, 없으면 NULL
5. CSV 파싱: strtok + atoi 조합으로 데이터 추출
