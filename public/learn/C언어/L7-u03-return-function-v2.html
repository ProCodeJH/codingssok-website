return 있는 함수

값을 계산하여 반환하는 함수를 다양하게 활용하고, 반환값을 변수에 저장하거나 식에 직접 사용합니다.

이 유닛에서 배우는 것

  • int, double 등 반환형을 가진 함수를 정의할 수 있습니다.
  • return 문으로 계산 결과를 호출자에게 전달할 수 있습니다.
  • 반환값을 변수에 저장하거나 식에 직접 사용할 수 있습니다.
  • 조건에 따라 다른 값을 반환하는 함수를 작성할 수 있습니다.

핵심 정리
1. 반환 함수: 반환형 함수명(매개변수) { return 값; }
2. 반환값 활용: 변수 저장, 직접 출력, 다른 함수 인자
3. 판정 함수: 조건 결과(0/1) 반환 — isPrime, isEven
4. 여러 return: 조건마다 다른 위치에서 반환 가능
