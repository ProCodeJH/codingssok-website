1. 기본 도구: myAbs, myMax, myMin, myPower
2. 수론 도구: gcd(유클리드), lcm(a/gcd*b), isPrime(sqrt 최적화)

4. 함수 합성: gcd→lcm, factorial→comb, gcd(gcd(a,b),c)
5. 사용 통계: 카운팅 배열로 각 기능의 호출 횟수 집계
