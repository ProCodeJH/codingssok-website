매개변수와 인자 (Parameter vs Argument)

매개변수(parameter)와 인자(argument)의 차이를 명확히 하고, 함수에 값이 전달되는 과정을 학습합니다.

이 유닛에서 배우는 것

  • 매개변수(parameter)와 인자(argument)의 차이를 명확히 구분할 수 있습니다.
  • 함수 호출 시 인자가 매개변수에 복사되는 과정을 추적할 수 있습니다.
  • 여러 매개변수의 순서와 자료형 일치의 중요성을 이해할 수 있습니다.
  • 기본값이 없는 C언어에서 매개변수를 유연하게 다루는 방법을 알 수 있습니다.

요약

핵심 정리
1. 매개변수(parameter): 함수 정의의 변수 — int add(int a, int b)
2. 인자(argument): 함수 호출 시 전달하는 값 — add(3, 5)
3. 값 복사: 인자의 값이 매개변수에 복사됨 (원본 불변)
4. 순서/타입 일치: 매개변수와 인자의 순서, 자료형 반드시 일치
5. Call by Value: C의 기본 전달 방식 — 복사본 전달
