  • 함수 프로토타입(함수 선언)의 문법과 필요성을 이해할 수 있습니다.

  • #include가 헤더 파일을 포함하는 원리를 이해할 수 있습니다.

대회 기출

요약

핵심 정리
1. 프로토타입: 반환형 함수명(매개변수); — main 위에 선언
2. 정의: 본체가 있는 완전한 함수 — main 아래에 작성 가능
3. #include: 헤더 파일 내용을 해당 위치에 복사/삽입
4. <꺾쇠>: 시스템 헤더, "큰따옴표": 사용자 헤더
5. 표준 헤더: stdio.h, stdlib.h, string.h, math.h, ctype.h
