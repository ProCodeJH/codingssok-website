요약

핵심 정리
1. 반환형 선택: 계산→int/double, 판정→int(0/1), 출력→void
2. 매개변수: 필요한 것만, 배열은 크기 별도 전달, 3개 이하 권장
3. 함수 분리: 한 함수 = 한 가지 일 (단일 책임 원칙)
4. 데이터 공유: 매개변수+반환값(우선) > 배열 > static > 전역변수(최후)
5. 이름 규칙: 동사 시작, 동작 명확, getMax/isPrime/printMenu
