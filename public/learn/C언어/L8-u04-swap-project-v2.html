두 변수 값 교환(swap) (프로젝트)

Call by Value의 한계를 포인터로 극복하여, 함수 안에서 두 변수의 값을 교환하는 swap을 완성합니다.

이 유닛에서 배우는 것

  • Call by Value에서 swap이 실패하는 이유를 메모리 관점으로 설명할 수 있습니다.
  • 포인터 매개변수로 함수 안에서 외부 변수를 변경할 수 있습니다.
  • swap 함수를 int, double, char 등 여러 자료형으로 구현할 수 있습니다.
  • 포인터를 활용한 정렬과 다중 값 반환을 구현할 수 있습니다.

대회 기출

요약

핵심 정리
1. Call by Value: 복사본만 변경 → 원본 불변
2. 포인터 전달: &x를 넘기고 *a로 원본 변경
3. swap: int t = *a; *a = *b; *b = t;
4. 다중 반환: 포인터 매개변수로 여러 값 동시 반환
5. 정렬에 활용: bubbleSort에서 swap(&arr[j], &arr[j+1])
6. 핵심: 함수에서 외부 변수를 변경하려면 주소를 전달
