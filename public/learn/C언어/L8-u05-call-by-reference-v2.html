실행 결과
Value 후: 10 Ref 후: 20
언제 Call by Reference를 사용하는가?
1. 외부 변수를 변경해야 할 때 (swap, 초기화 등)
2. 여러 값을 동시에 반환할 때 (return은 하나만 가능)
3. 큰 데이터를 복사하지 않고 전달할 때 (효율성)
4. 입력 함수에서 결과를 저장할 때 (scanf 패턴)

대회 기출

요약

핵심 정리
1. Call by Reference: 주소를 전달하여 원본을 변경
2. 패턴: void func(int *p) + 호출 시 func(&x)
3. 다중 반환: 포인터 여러 개로 동시에 결과 전달
4. 입력 함수: 포인터로 값 저장 + return으로 성공/실패
5. 상태 변경: 게임 HP/EXP 등을 함수에서 직접 수정
6. 핵심: C에서 외부 변수를 바꾸는 유일한 방법은 주소 전달
