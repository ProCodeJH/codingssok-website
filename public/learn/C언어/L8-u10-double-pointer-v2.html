실행 결과
변경 전: *ptr = 100 Fail 후: *ptr = 100 OK 후: *ptr = 200
패턴 일관성
값을 바꾸려면 → 값의 주소를 전달 (int → int*)
포인터를 바꾸려면 → 포인터의 주소를 전달 (int* → int**)

항상 '한 단계 위의 포인터'로 전달하면 원본을 변경할 수 있습니다.

대회 기출

요약

핵심 정리
1. 이중 포인터: 포인터의 주소를 저장하는 변수
2. 선언: int **pp = &p;
3. *pp → 포인터 p의 값, **pp → 최종 값
4. 용도: 함수에서 포인터 자체를 변경할 때
5. 패턴: int → int* → int** (한 단계씩 추가)
6. char **argv == char *argv[] (문자열 배열)

다음 유닛으로

다음: 포인터 배열 vs 배열 포인터
이중 포인터를 배웠습니다. 다음 유닛에서는 혼동하기 쉬운 두 개념, '포인터의 배열(int *arr[])'과 '배열을 가리키는 포인터(int (*arr)[N])'의 차이를 명확히 정리합니다.
