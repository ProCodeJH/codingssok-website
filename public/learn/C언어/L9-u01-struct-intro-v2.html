구조체란?

서로 다른 자료형의 데이터를 하나로 묶는 구조체(struct)의 개념, 선언, 초기화 방법을 배웁니다.

이 유닛에서 배우는 것

  • 구조체가 왜 필요한지 이해합니다.
  • struct 키워드로 구조체를 선언할 수 있습니다.
  • 구조체 변수를 선언하고 초기화할 수 있습니다.
  • 구조체 멤버에 값을 저장하고 읽을 수 있습니다.

구조체 선언 문법
struct 구조체이름 {

주의: 닫는 중괄호 뒤에 세미콜론(;)이 반드시 필요합니다.
구조체 이름은 대문자로 시작하는 것이 관례입니다.
C
1#include <stdio.h>
2
3/* 구조체 선언 — 새로운 자료형을 정의 */
4struct Student {
5 char name[20];
6 int age;
7 double score;
8}; /* 세미콜론 필수! */
9
10int main(void) {
11 /* 구조체 변수 선언 + 초기화 */
12 struct Student s1 = {"Kim", 17, 95.5};
13
14 /* 멤버 접근: 변수.멤버 */
15 printf("이름: %s\n", s1.name);
16 printf("나이: %d\n", s1.age);
17 printf("점수: %.1f\n", s1.score);
18 return 0;
19}

19 /* 방법 2: 선언 후 대입 */

28 struct Student s;
29 strcpy(s.name, "Park"); /* s.name = "Park" 불가! */
30 s.age = 16;
31 s.score = 88.0;
32 printf("%s, %d세, %.0f점\n", s.name, s.age, s.score);
33 return 0;
34}
구조체 멤버 대입 규칙
숫자 멤버: s.age = 17; (직접 대입 가능)
문자열 멤버: strcpy(s.name, "Kim"); (strcpy 사용 필수)
s.name = "Kim"; 은 컴파일 에러!
(배열에 문자열 직접 대입 불가 — L6에서 학습)

구조체끼리 대입: s2 = s1; (모든 멤버가 복사됨)

에러 체험

C
1// 에러 1: 세미콜론 누락
2struct Point { int x; int y; } // ← ; 빠짐
3// 수정: struct Point { int x; int y; };
4
5// 에러 2: 문자열 멤버에 = 사용
6struct Student s;
7s.name = "Kim"; // 배열에 직접 대입 불가
8// 수정: strcpy(s.name, "Kim");
9
10// 에러 3: struct 키워드 누락
11Student s; // C에서는 struct 키워드 필요
12// 수정: struct Student s; (typedef 없이는 struct 필수)

대회 기출

요약

핵심 정리
1. 구조체: 서로 다른 자료형을 하나로 묶는 사용자 정의 자료형
2. 선언: struct 이름 { 멤버들; };
3. 변수: struct 이름 변수 = {초기값};
4. 접근: 변수.멤버
5. 문자열 멤버: strcpy(s.name, "값") 사용
6. 구조체 대입: s2 = s1 (모든 멤버 복사)
