대회 기출

요약

핵심 정리
1. 종합 활용: struct + enum + 비트 필드 + 중첩 + 포인터
2. 캐릭터: typedef struct { Stats stat; StatusFlags flags; } Character;
3. 직업/스킬: typedef enum { WARRIOR, MAGE, ... } CharClass;
4. 상태이상: unsigned int poisoned:1; (비트 필드)
5. 파티: Character party[] + 포인터 함수
6. 설계 원칙: 데이터(struct) + 동작(함수) + 상태(enum/비트필드) 분리
