의사코드란 무엇일까?

안녕! 오늘은 의사코드를 배울 거야. '의사'는 '가짜'라는 뜻이야. 의사코드는 가짜 코드 — 진짜 프로그래밍 언어는 아니지만 프로그램처럼 쓴 글이야!

코드를 바로 쓰면 어떻게 될까요? 생각을 정리하지 않고 바로 코딩하면 중간에 헷갈리고, 실수도 많아져요. 그래서 프로그래머들은 코드를 쓰기 전에 먼저 의사코드로 생각을 정리해요.

[핵심] 의사코드 (Pseudocode)
의사코드 = 프로그래밍 언어가 아닌 자연어(한국어, 영어)로 쓴 프로그램 설계도

✅ 사람이 읽기 위한 것
✅ 문법 규칙이 자유로움
✅ 논리와 순서만 정확하면 OK!
❌ 컴퓨터가 직접 실행할 수는 없음
Pseudo
'Pseudo'는 그리스어로 '가짜'라는 뜻이에요. 의사코드 = 가짜 코드 = 진짜 코드가 되기 전의 설계도!

출력: PRINT, OUTPUT, DISPLAY, '~를 출력한다'
대입: ← (화살표), SET, ':='
조건: IF ~ THEN / ELSE / ELSE IF
반복: WHILE ~ DO / FOR i ← 1 TO N / REPEAT ~ UNTIL
종료: END, RETURN, STOP

[심화] 복잡한 알고리즘의 의사코드

pseudo_binary_search.c
1// 의사코드: 이진 탐색
2// 1. INPUT sorted_array A[0..n-1], target
3// 2. low ← 0, high ← n - 1
4// 3. WHILE low ≤ high DO
5// mid ← (low + high) / 2
6// IF A[mid] = target THEN
7// RETURN mid
8// ELSE IF A[mid] < target THEN
9// low ← mid + 1
10// ELSE
11// high ← mid - 1
12// 4. RETURN -1 (못 찾음)
13
14#include <stdio.h>
15
16int binarySearch(int arr[], int n, int target) {
17 int low = 0, high = n - 1;
18 while (low <= high) {
19 int mid = (low + high) / 2;
20 if (arr[mid] == target)
21 return mid;
22 else if (arr[mid] < target)
23 low = mid + 1;
24 else
25 high = mid - 1;
26 }
27 return -1;
28}
29
30int main(void) {
31 int arr[] = {2, 5, 8, 12, 16, 23, 38, 56, 72, 91};
32 int idx = binarySearch(arr, 10, 23);
33 printf("23은 인덱스 %d에 있음\n", idx); // 5
34 return 0;
35}

[심화] 의사코드로 버블 정렬 설계

[심화] 버블 정렬 의사코드
PROCEDURE BubbleSort(A, n)
FOR i ← 0 TO n-2 DO
FOR j ← 0 TO n-2-i DO
IF A[j] > A[j+1] THEN
SWAP A[j], A[j+1]
RETURN A

분석:
• 바깥 루프: n-1번
• 안쪽 루프: 최대 n-1번
• 총 비교: O(n²)
pseudo_bubble.c

4 for (int j = 0; j < n - 1 - i; j++) {
5 if (arr[j] > arr[j + 1]) {
6 // SWAP
7 int tmp = arr[j];
8 arr[j] = arr[j + 1];
9 arr[j + 1] = tmp;
10 }
11 }
12 }
13}

[심화 내용] 의사코드로 복잡도 분석

[심화 내용] 의사코드에서 Big-O 추론
의사코드의 루프 구조를 보면 시간 복잡도를 바로 알 수 있어요!

단일 루프:
FOR i ← 1 TO n → O(n)

중첩 루프:
FOR i ← 1 TO n
FOR j ← 1 TO n → O(n²)

반감 루프:
WHILE n > 0
n ← n / 2 → O(log n)

중첩 + 반감:
FOR i ← 1 TO n
j ← n
WHILE j > 0: j ← j/2 → O(n log n)

[심화 내용] 의사코드로 정당성 검증

[심화 내용] 루프 불변식 + 의사코드

PROCEDURE LinearSearch(A, n, target)
// 불변식: A[0..i-1]에는 target이 없다
FOR i ← 0 TO n-1 DO
IF A[i] = target THEN RETURN i
RETURN -1 // 전체에 없음

종료 후: A[0..n-1] 전체에 target 없음 → -1 반환 정당

[심화 내용] 의사코드와 정형 검증

[심화 내용] 루프 불변식 (Loop Invariant)
루프 불변식 = 반복문 실행 전후에 항상 참인 조건

알고리즘의 정확성을 증명하는 핵심 도구!

예: 선형 탐색의 루프 불변식
SEARCH(A, n, x):
for i = 0 to n-1
if A[i] == x return i
return -1

불변식: "A[0..i-1]에 x가 없다"

초기: i=0, 검사 구간 없음 → 참
유지: A[i]!=x이면 A[0..i]에 x 없음 → 참
종료: i=n, A[0..n-1]에 x 없음 → -1 반환 올바름!

DIJKSTRA(G, source):
dist[source] = 0
for each vertex v != source: dist[v] = INF
Q = 모든 정점의 우선순위 큐
while Q is not empty:
u = Q에서 dist가 최소인 정점 추출
for each neighbor v of u:
if dist[u] + weight(u,v) < dist[v]:
dist[v] = dist[u] + weight(u,v)
Q에서 v의 우선순위 갱신

BUBBLE-SORT(A, n):
for i = 0 to n-2:
for j = 0 to n-2-i:
if A[j] > A[j+1]:
swap A[j] and A[j+1]

FIBONACCI(n):
if n <= 1: return n
a = 0, b = 1
for i = 2 to n:
c = a + b
a = b
b = c
return b
