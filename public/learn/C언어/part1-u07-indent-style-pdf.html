들여쓰기가 뭘까?

책을 쓸 때 문단의 첫 줄을 한 칸 띄우잖아요? 코드에서도 비슷한 걸 해요. 이걸 들여쓰기(indent)라고 해요!

들여쓰기는 중괄호 { } 안쪽 코드를 오른쪽으로 밀어서 쓰는 거예요. 이렇게 하면 어디가 함수 안쪽인지, 어디가 바깥인지 한눈에 보여요.

✕ 잘못된 코드
#include int main(void) { printf("A\n"); printf("B\n"); return 0; }
들여쓰기 없음 — 어디가 안쪽인지 모르겠어요
✓ 올바른 코드
#include int main(void) { printf("A\n"); printf("B\n"); return 0; }
4칸 들여쓰기 — 구조가 한눈에!

스페이스 4칸 vs 탭(Tab)

들여쓰기 방법은 두 가지예요. 스페이스 4칸을 직접 누르거나, Tab 키를 한 번 누르거나. 둘 다 괜찮지만, 한 가지로 통일하는 게 중요해요!

[핵심] 우리 교재의 규칙: 스페이스 4칸
이 교재에서는 스페이스 4칸을 사용해요. VS Code에서 Tab을 누르면 자동으로 스페이스 4칸이 입력돼요. (설정에서 바꿀 수 있어요)
2 vs 4
구글은 2칸, 리눅스 커널은 탭(8칸), 대부분의 C 교재는 4칸을 써요. 정답은 없지만, 중요한 건 한 프로그램 안에서 통일하는 거예요!
스페이스와 탭을 섞어 쓰면 코드가 뒤죽박죽 돼요. 하나만 골라서 쓰세요!

[심화] 들여쓰기 전쟁: 탭 vs 스페이스

[심화] 탭 vs 스페이스 — 영원한 논쟁
탭 파:
- 한 번 누르면 끝 (효율적!)
- 각자 에디터에서 폭 조절 가능
- Linux 커널은 탭 8칸 사용
- 파일 크기가 작음 (1바이트)

스페이스 파:
- 어디서 열어도 동일하게 보임
- 정밀한 정렬 가능
- 대부분의 코딩 스타일 가이드 권장
- Stack Overflow 설문: 스페이스 개발자가 연봉 높음(?!)

C 프로젝트별:
Linux 커널: 탭 8칸
Google: 스페이스 2칸
LLVM: 스페이스 2칸
Mozilla: 스페이스 2칸
GNU: 스페이스 2칸

결론: 팀 규칙을 따르자!

[심화 내용] 들여쓰기와 인지 부하

[심화 내용] 인지 복잡도 (Cognitive Complexity)
인지 복잡도 = 코드 읽기 어려움

들여쓰기 깊이가 깊을수록 인지 부하 증가!

측정법 (SonarQube):
- if/else/for/while 시작: +1
- 중첩될 때마다: +1씩 추가
- break/continue/goto: +1

예:
for (...) { // +1 (깊이 0)
if (...) { // +2 (기본1+중첩1)
if (...) { // +3 (기본1+중첩2)
... // 여기서 인지 부하 6!
}
}
}

권장:
- 함수당 인지 복잡도 15 이하
- 들여쓰기 3단계 이하
- 초과 시 → 함수 분리 or 가드 절

[심화] 공백 규칙 상세

[심화] 공백(whitespace) 규칙
연산자 주위 공백:
✅ x = a + b * c;
❌ x=a+b*c;

콤마 뒤 공백:
✅ func(a, b, c);
❌ func(a,b,c);

괄호 안 공백:
✅ if (x > 0) { (K&R/Google)
❌ if( x > 0 ){ (공백 위치 이상)

포인터 위치:
int *p; ← C 스타일 (변수에 붙임)
int* p; ← C++ 스타일 (타입에 붙임)
→ int *a, *b; 에서 C 스타일이 안전
→ int* a, b; ← b는 포인터 아님!

빈 줄:
함수 사이: 1줄 빈 줄
논리적 블록 사이: 1줄 빈 줄
함수 내부: 0~1줄
연속 빈 줄 2줄 이상: 금지!

15}

18 return PI * c.radius * c.radius;
19}
20

22 return 2 * PI * c.radius;
23}
24
25int circle_contains_point(Circle c, double px, double py) {
26 double dx = px - c.center_x;
27 double dy = py - c.center_y;
28 double dist_sq = dx * dx + dy * dy;
29 return dist_sq <= c.radius * c.radius;
30}
31
32void circle_print(Circle c) {
33 printf("원: 중심(%.1f, %.1f) 반지름=%.1f\n",
34 c.center_x, c.center_y, c.radius);
35 printf(" 넓이: %.2f\n", circle_area(c));
36 printf(" 둘레: %.2f\n", circle_circumference(c));
37}

42
