별로 그림 그리기 (1) — 사각형과 삼각형

printf와 반복적인 출력으로 별(*) 도형을 그려요.

📚 이번에 배울 것

  • 별(*)로 사각형을 출력할 수 있어요.
  • 별(*)로 직각삼각형을 출력할 수 있어요.
  • 출력 패턴을 관찰하고 규칙을 찾을 수 있어요.

✅ 시작 전 체크

  • printf로 여러 줄을 출력할 수 있다
  • \n과 \t를 쓸 줄 안다
Part 1 진행률: 20%

코드로 그림을 그린다고?

글자만 출력하는 건 이제 식상하죠? 오늘은 별(*)로 도형을 그려볼 거예요!

문자로 그림을 그리는 것을 아스키 아트(ASCII Art)라고 해요. 프로그래머들은 오래전부터 이 방법으로 재미있는 그림을 만들었어요. 오늘은 가장 기본적인 도형부터 시작할게요.

정사각형 그리기

먼저 5x5 정사각형을 그려볼게요. 규칙은 간단해요 — 매 줄마다 별 5개를 출력하면 돼요.

1#include <stdio.h>
2
3int main() {
4 printf("*****\n");
5 printf("*****\n");
6 printf("*****\n");
7 printf("*****\n");
8 printf("*****\n");
9 return 0;
10}
실행 결과
***** ***** ***** ***** *****

같은 printf를 5번 반복했어요. 나중에 반복문(for)을 배우면 이걸 2줄로 줄일 수 있어요!

지금은 printf를 여러 번 쓰는 게 불편하지만, 패턴을 이해하는 게 중요해요. 반복문은 나중에 배워요!

직각삼각형 그리기 (왼쪽 정렬)

이번에는 삼각형이에요. 1줄에 별 1개, 2줄에 별 2개... 이렇게 늘어나는 패턴이에요.

1#include <stdio.h>
2
3int main() {
4 printf("*\n");
5 printf("**\n");
6 printf("***\n");
7 printf("****\n");
8 printf("*****\n");
9 return 0;
10}
실행 결과
* ** *** **** *****
패턴 찾기
1줄: 별 1개
2줄: 별 2개
3줄: 별 3개
...
n줄: 별 n개

이런 규칙을 패턴이라고 해요. 패턴을 찾는 능력이 프로그래밍의 핵심이에요!

직각삼각형 (오른쪽 정렬)

삼각형을 오른쪽으로 정렬하려면 별 앞에 공백을 넣어야 해요.

1#include <stdio.h>
2
3int main() {
4 printf(" *\n");
5 printf(" **\n");
6 printf(" ***\n");
7 printf(" ****\n");
8 printf("*****\n");
9 return 0;
10}
실행 결과
* ** *** **** *****

1줄에는 공백 4개 + 별 1개, 2줄에는 공백 3개 + 별 2개... 이런 패턴이에요. 공백 + 별 = 항상 5라는 규칙이 있어요!

공백도 중요한 문자예요! 눈에 안 보인다고 무시하면 안 돼요. 공백의 개수가 도형의 모양을 결정해요.

빈 사각형 그리기

속이 빈 사각형도 만들 수 있어요. 첫 줄과 마지막 줄은 별로 가득, 중간 줄은 양쪽 끝만 별이에요.

코딩 기초 습관
도형을 그릴 때는 먼저 종이에 그려보세요. 각 줄에 별이 몇 개, 공백이 몇 개인지 세어보면 코드 짜기가 훨씬 쉬워요!

[심화 내용] 별 찍기 패턴 분석법
모든 별 찍기의 핵심:

1. 줄 수(행) = 바깥 for문
2. 각 줄의 문자 = 안쪽 for문
3. 공백/별 관계 = 수학 공식

피라미드 분석:
줄 i (1~n):
공백: n-i개
별: 2i-1개

역피라미드:
줄 i (n~1):
공백: n-i개
별: 2i-1개

다이아몬드 = 피라미드 + 역피라미드

모래시계 = 역피라미드 + 피라미드

규칙을 찾으면 어떤 패턴이든 코딩 가능!
[활동] ASCII 아트 도전
1. 숫자 n을 입력받아 n단 피라미드 출력
2. 속이 빈 다이아몬드 출력
3. 알파벳 삼각형 (A, AB, ABC...)
4. 숫자 피라미드 (1, 12, 123...)
5. 나만의 로고를 ASCII 아트로 만들기

패턴 규칙: "줄 번호와 칸 번호의 관계"를 찾자!
for 중첩
바깥 for = 줄(행), 안쪽 for = 칸(열). 2중 for = 2D 패턴. 3중 for = 3D 탐색.
별 찍기의 비밀: 줄 번호 i와 칸 번호 j의 수학적 관계를 찾으면 돼! 공백=n-i, 별=2i-1 이런 식으로!

[심화 내용] 이미지를 ASCII 아트로

[심화 내용] 이미지→ASCII 변환 알고리즘
원리:
1. 이미지를 그레이스케일로 변환
2. 작은 블록으로 나누기 (예: 8x8)
3. 각 블록의 평균 밝기 계산 (0~255)
4. 밝기를 ASCII 문자로 매핑

밝기 순서 (어두움→밝음):
@#W$9876543210?!abc;:+=-,._ (space)

간단한 매핑:
0~31: @ (가장 어두움)
32~63: #
64~95: *
96~127: +
128~159: :
160~191: .
192~255: (space, 가장 밝음)

비율 보정:
문자는 세로가 더 길음 (2:1)
→ y 방향 2배 건너뛰기!
[C 코드] 간단한 이미지→ASCII
1#include <stdio.h>
2#include <math.h>
3
4// 밝기를 ASCII 문자로 변환
5char brightness_to_char(int brightness) {
6 const char *chars = " .:-=+*#%@";
7 int idx = brightness * 9 / 255;
8 if (idx < 0) idx = 0;
9 if (idx > 9) idx = 9;
10 return chars[9 - idx]; // 어두운게 밀도 높은 문자
11}
