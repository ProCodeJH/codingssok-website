다이아몬드 그리기

다이아몬드는 위쪽 삼각형 + 아래쪽 역삼각형을 합친 모양이에요. 패턴을 먼저 분석해 볼게요.

1#include <stdio.h>
2
3int main() {
4 // 위쪽 삼각형
5 printf(" *\n");
6 printf(" ***\n");
7 printf("*****\n");
8 // 아래쪽 역삼각형
9 printf(" ***\n");
10 printf(" *\n");
11 return 0;
12}
실행 결과
* *** ***** *** *
패턴 분석
1줄: 공백2 + 별1
2줄: 공백1 + 별3
3줄: 공백0 + 별5
4줄: 공백1 + 별3
5줄: 공백2 + 별1

공백은 줄어들었다 늘어나고, 별은 늘어났다 줄어들어요. 가운데를 기준으로 대칭이에요!
복잡한 도형은 두 부분으로 나눠서 생각하세요. 다이아몬드 = 삼각형 + 역삼각형! 이게 프로그래머의 사고방식이에요.

[심화 내용] 재귀 패턴 생성

[심화 내용] 재귀로 프랙탈 만들기
재귀 패턴 원리:
큰 패턴 = 작은 패턴의 반복

코흐 곡선 (Koch Curve):
1. 선분을 3등분
2. 가운데 1/3을 삼각형으로
3. 각 선분에 대해 반복

시에르핀스키 카펫:
1. 정사각형을 9등분
2. 가운데를 제거
3. 나머지 8개에 대해 반복

칸토어 집합:
1. 선분을 3등분
2. 가운데 1/3 제거
3. 나머지에 대해 반복

재귀 깊이 = 디테일 수준!

6void draw_tree(int x, int y, int height, int direction) {
7 if (height <= 0 || y < 0 || y >= 40 || x < 0 || x >= 78) return;

[심화] 패턴과 수학의 관계
삼각형: 행 i에서 별 i개
피라미드: 행 i에서 공백 n-i, 별 2i-1
다이아몬드: 피라미드 + 역피라미드
하트: 원 2개 + 삼각형 조합
나비: 삼각형 2개 좌우 대칭
시에르핀스키: (x & y) == 0

핵심 수학:
1. 등차수열: 1, 3, 5, 7... (홀수)
2. 대칭: |x - center|
3. 비트 연산: x & y (프랙탈)
4. 삼각함수: sin, cos (곡선)
5. 거리 공식: sqrt(x*x + y*y) (원)
재귀 프랙탈
재귀 함수로 자기 유사 패턴 생성. 각 호출이 더 작은 버전을 그리며, 기저 조건에서 멈춤.
패턴 분석의 핵심: "행 번호 i와 각 요소의 관계"를 수학 공식으로 표현하는 거야! 공식을 찾으면 코드는 자동!

[심화 내용] ANSI 이스케이프 코드 전체
포맷: ESC[<코드>m
ESC = \033 = \x1B = 27

텍스트 스타일:
0: 리셋
1: 굵게, 2: 흐리게
3: 이탤릭, 4: 밑줄
5: 깜빡임, 7: 반전
9: 취소선

전경색 (30~37):
30:검정 31:빨강 32:초록 33:노랑
34:파랑 35:자홍 36:청록 37:흰색

배경색 (40~47):
40~47: 위와 같은 순서

256색:
38;5;n (전경), 48;5;n (배경)

RGB (24비트):
38;2;r;g;b (전경)
48;2;r;g;b (배경)
ANSI 256색
\033[38;5;Nm으로 256색 출력. 0-7 기본, 8-15 밝은, 16-231 RGB, 232-255 그레이스케일.

5#define MH 11

9void init_maze() {
10 for (int y = 0; y < MH; y++)
11 for (int x = 0; x < MW; x++)
12 maze[y][x] = (y%2==0 || x%2==0) ? '#' : ' ';
13}

15void carve(int y, int x) {
16 int dirs[4][2] = {{0,2},{0,-2},{2,0},{-2,0}};
17 // 섞기
18 for (int i = 3; i > 0; i--) {
19 int j = rand() % (i+1);
20 int t0=dirs[i][0],t1=dirs[i][1];
21 dirs[i][0]=dirs[j][0]; dirs[i][1]=dirs[j][1];
22 dirs[j][0]=t0; dirs[j][1]=t1;
23 }
24 for (int d = 0; d < 4; d++) {
25 int ny = y + dirs[d][0];
26 int nx = x + dirs[d][1];
27 if (ny>0 && ny<MH-1 && nx>0 && nx<MW-1 && maze[ny][nx]==' ') {
28 maze[y+dirs[d][0]/2][x+dirs[d][1]/2] = ' ';
29 maze[ny][nx] = '.';
30 carve(ny, nx);
31 }
32 }
33}

42 maze[0][1] = ' ';
43 maze[MH-1][MW-2] = ' ';
44
45 printf("=== Random Maze ===\n");
46 for (int y = 0; y < MH; y++) {
47 for (int x = 0; x < MW; x++)

[심화 내용] 미로 생성 알고리즘
DFS (깊이 우선 탐색):
1. 시작점에서 출발
2. 랜덤 방향으로 벽 뚫기
3. 막다른 길 → 백트래킹
4. 모든 셀 방문할 때까지 반복
→ 가장 간단! 재귀로 구현

Kruskal 알고리즘:
1. 모든 벽을 리스트에 넣기
2. 랜덤 벽 선택
3. 양쪽이 다른 집합이면 벽 제거
→ Union-Find 자료구조 필요

Prim 알고리즘:
1. 시작점에서 인접 벽 추가
2. 랜덤 벽 선택 후 연결
→ 미로 밀도 균일

알고리즘 = 패턴! 같은 문제도 다양한 해법!
DFS
Depth-First Search. 깊이 우선 탐색. 스택/재귀 기반. 미로 생성, 경로 탐색의 기본 알고리즘.
미로 생성기를 30줄로 만들 수 있어! DFS + 랜덤 방향 = 매번 다른 미로. 알고리즘의 힘!
[활동] ASCII 아트 2 과제
1. 달력 프로그램 완성 (윤년 처리 포함)

백트래킹. 해를 찾다가 막히면 되돌아가서 다른 경로 시도. 미로 생성/탐색의 핵심 기법.
