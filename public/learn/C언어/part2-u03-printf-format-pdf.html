Lesson 4-3

printf로 변수 출력하기

%d 서식 지정자로 변수 값을 출력하는 다양한 방법을 배워요.

📚 이번에 배울 것

  • %d 서식 지정자의 원리를 이해해요.
  • printf 하나로 여러 변수를 동시에 출력할 수 있어요.
  • 변수끼리 계산한 결과도 바로 출력할 수 있어요.

✅ 시작 전 체크

  • 변수를 선언하고 값을 넣을 수 있다
  • printf의 기본 사용법을 안다
Part 1 진행률: 10%

%d가 뭘까?

Part 1에서는 printf에 글자만 넣었는데, 이제 변수 값도 넣어볼 거예요!

printf 안에서 %d는 '여기에 정수를 넣을게요'라는 표시예요. 빈칸을 만들어 놓고, 뒤에서 변수로 채우는 거예요.

%d = 빈칸 → 변수가 채워요
printf("나이: %d살", age);
빈칸 표시 ←←←← 빈칸을 채울 값

기본 사용법

[코딩 습관] printf 디버깅
코드가 이상하게 동작하면, printf("%d\n", 변수);를 넣어서 변수 값을 중간중간 확인해 보세요. 이걸 printf 디버깅이라고 해요. 간단하지만 정말 유용해요!
format1.c
1#include <stdio.h>
2
3int main(void) {
4 int apples = 5;
5 int oranges = 3;
6
7 printf("사과: %d개\n", apples);
8 printf("오렌지: %d개\n", oranges);
9 return 0;
10}
실행 결과
사과: 5개 오렌지: 3개
7줄"사과: %d개\n"에서 %d 자리에 apples의 값(5)이 들어가요. 나머지 글자는 그대로 출력돼요.
8줄같은 원리로 %d에 oranges의 값(3)이 들어가요.

%d를 여러 개 쓰기

printf 하나에 %d를 여러 개 넣을 수 있어요. 쉼표(,)로 구분해서 값을 왼쪽부터 순서대로 채워요.

format2.c
1#include <stdio.h>
2
3int main(void) {
4 int a = 10;
5 int b = 20;
6 int c = 30;
7
8 printf("%d + %d + %d = %d\n", a, b, c, a + b + c);
9 return 0;
10}
실행 결과
10 + 20 + 30 = 60
%d 4개, 값도 4개! %d 개수와 값 개수가 반드시 같아야 해요. 안 맞으면 이상한 숫자가 나와요.

계산 결과 바로 출력하기

변수에 저장하지 않고, printf 안에서 바로 계산할 수도 있어요.

실행 결과

자주 하는 실수

%d를 쓸 때 흔히 하는 실수예요!
✕ 잘못된 코드
printf("%d + %d\n", a);
%d가 2개인데 값은 1개! 엉뚱한 숫자 출력
✓ 올바른 코드
printf("%d + %d\n", a, b);
%d 개수 = 값 개수 → 맞춰야 해요
✕ 잘못된 코드
printf(score);
서식 문자열 없이 변수만! 에러 또는 엉뚱한 결과
✓ 올바른 코드
printf("%d\n", score);
큰따옴표 안에 %d를 쓰고, 뒤에 변수

[심화] printf 포맷 완전 가이드
기본 형식: %[플래그][폭][.정밀도][길이]타입

타입:

%u: 부호 없는 정수
%o: 8진수
%x, %X: 16진수

%g, %G: 짧은 쪽 자동 선택
%c: 문자
%s: 문자열
%p: 포인터 주소
%n: 출력된 문자 수 저장 (위험!)
%%: % 문자 자체

플래그:
-: 왼쪽 정렬
+: 양수에도 + 표시
0: 빈칸을 0으로 채움
(공백): 양수 앞에 공백
#: 대체 형식 (0x, 0o 접두어)

[심화 내용] 가변 인자와 printf 구현

[심화 내용] printf의 내부 동작
printf는 가변 인자 함수!
int printf(const char *fmt, ...);

동작 과정:
1. fmt 문자열을 순회
2. 일반 문자 → 그대로 출력
3. % 발견 → 포맷 파싱
4. va_arg()로 인자 가져오기
5. 타입에 맞게 변환 + 출력

위험!
printf("%d %d", 42); // 인자 부족! UB
printf("%d", 3.14); // 타입 불일치! UB
→ 컴파일러가 체크: -Wformat

보안 위험:
printf(user_input); // 포맷 문자열 공격!
→ 반드시 printf("%s", user_input);
[C 코드] 간단한 my_printf 구현
1#include <stdio.h>
2#include <stdarg.h>
3
4void my_printf(const char *fmt, ...) {
5 va_list args;
6 va_start(args, fmt);
7
8 for (; *fmt; fmt++) {
9 if (*fmt != '%') {
10 putchar(*fmt);
11 continue;
12 }
13 fmt++; // % 다음 문자
14 switch (*fmt) {
15 case 'd': {
16 int val = va_arg(args, int);
17 // 간단한 정수 출력
18 if (val < 0) { putchar('-'); val = -val; }
19 char buf[20];
20 int i = 0;
21 do { buf[i++] = '0' + val % 10; val /= 10; } while (val);
22 while (i > 0) putchar(buf[--i]);
23 break;
24 }
25 case 's': {
26 const char *s = va_arg(args, const char*);
27 while (*s) putchar(*s++);
28 break;
29 }
30 case 'c':
31 putchar(va_arg(args, int));
32 break;
33 case '%':
34 putchar('%');
35 break;
36 default:
37 putchar('%');
38 putchar(*fmt);
39 }
40 }
41 va_end(args);
42}
43
44int main() {
45 my_printf("Hello %s! You are %d years old.\n", "Kim", 20);
46 my_printf("Char: %c, Percent: %%\n", 'A');
47 return 0;
48}
실행 결과
Hello Kim! You are 20 years old. Char: A, Percent: %
%n
printf에서 출력된 문자 수를 변수에 저장. 보안 취약점의 원인! 사용 금지. -Wformat-security로 경고.
va_list
의 가변 인자 타입. va_start, va_arg, va_end로 순회. printf 등 가변 인자 함수의 핵심.
printf를 직접 구현해보면 C의 핵심(포인터, 가변인자, switch)이 전부 들어있어! 최고의 연습 문제!
%n은 쓰지 마! 해커가 메모리에 값을 쓸 수 있어. printf(user_input)도 금지! 항상 printf("%s", input)!
[심화] 실용적인 printf 활용
1. 영수증 출력:
printf("%-20s %8.2f원\n", item, price);

2. 진행 상황:
printf("\r[%3d%%] 처리 중...", percent);
fflush(stdout);

3. 디버깅:
printf("[DEBUG] %s:%d %s = %d\n", __FILE__, __LINE__, #x, x);
→ 매크로로 자동화!

4. 로그 타임스탬프:
printf("[%04d-%02d-%02d %02d:%02d:%02d] %s\n",
y, m, d, h, min, s, msg);

5. 16진수 덤프:
printf("%08X: ", addr);
for(...) printf("%02X ", byte);

[심화 내용] printf 보안 취약점 상세
포맷 문자열 공격 (Format String Attack):

char user[100];
fgets(user, 100, stdin);
printf(user); // 위험!!!

사용자가 "%x %x %x %x" 입력하면?
→ 스택 메모리 내용 노출!

사용자가 "%n" 입력하면?
→ 메모리에 값을 쓸 수 있음!
→ 리턴 주소 덮어쓰기 → 코드 실행!

방어:
1. printf("%s", user); // 항상 포맷 지정!
2. -Wformat-security 컴파일 옵션
3. -Wformat-nonliteral (포맷이 변수면 경고)
4. FORTIFY_SOURCE (glibc 보호)
%*d
가변 폭 포맷. 폭을 인자로 전달. printf("%*d", 10, 42) = printf("%10d", 42). 동적 표 출력에 필수!
printf는 C의 스위스 아미 나이프! 포맷 지시자를 마스터하면 어떤 출력이든 가능해!
[C 코드] 디버그 매크로 시스템
1#include <stdio.h>
2
3// 디버그 모드 토글

23}

[심화 내용] C11/C23 printf 확장

%jd: intmax_t 출력

int → %d (항상 OK)

PRId64
매크로. int64_t를 printf로 출력하는 이식성 있는 방법. 플랫폼마다 long/long long 차이 해결.

[활동] printf 마스터 체크리스트
%d, %f, %s, %c, %x, %o를 안다
%-10d, %010d, %+d 플래그를 안다
%.2f 정밀도 지정을 안다
%*d 가변 폭을 활용할 수 있다
%.5s 문자열 자르기를 안다
디버그 매크로(DUMP_INT)를 구현할 수 있다
포맷 문자열 공격의 위험을 안다
PRId64 등 이식성 매크로를 안다

6개 이상이면 printf 마스터!
FORTIFY_SOURCE
glibc 보안 기능. printf, memcpy 등에 버퍼 크기 체크 추가. -D_FORTIFY_SOURCE=2로 활성화.

[심화] snprintf와 문자열 조합
snprintf = 안전한 sprintf
char buf[100];
int n = snprintf(buf, sizeof(buf), "%s은 %d세", name, age);
if (n >= sizeof(buf)) → 잘림!

문자열 연결:
char msg[200];
int pos = 0;
pos += snprintf(msg+pos, sizeof(msg)-pos, "이름: %s\n", name);
pos += snprintf(msg+pos, sizeof(msg)-pos, "나이: %d\n", age);
→ 여러 줄을 하나의 버퍼에 안전하게 조합!

주의:
- sizeof(buf)-pos가 0이면 아무것도 안 씀
- 반환값은 "잘리지 않았다면"의 길이
- 항상 \0 종료 보장
fprintf
파일에 포맷 출력. fprintf(fp, "x=%d", 42). stderr에 에러 출력: fprintf(stderr, "Error!").
sprintf
문자열에 포맷 출력. 버퍼 오버플로우 위험! 반드시 snprintf(buf, size, ...) 사용.
printf 패밀리: printf(화면), fprintf(파일), sprintf(문자열), snprintf(안전한 문자열). 목적에 맞게 선택!
__FILE__
현재 파일명 매크로. __LINE__은 현재 줄 번호. __func__은 현재 함수명. 디버깅 로그에 필수!
##__VA_ARGS__
가변 매크로 인자. ##은 인자 없을 때 쉼표 제거. 디버그 매크로의 핵심 기법.
printf 포맷을 외우는 팁: d=decimal, f=float, s=string, c=char, x=heX, o=Octal. 첫 글자!
%zu
size_t 출력 포맷 (C99). sizeof의 반환 타입. %d로 출력하면 경고! 이식성을 위해 %zu 사용.
puts
문자열 출력 + 자동 줄바꿈. puts("Hello") = printf("Hello\n"). 포맷 불필요 시 더 간단!
putchar
한 문자 출력. putchar('A') = printf("%c", 'A'). 대량 문자 출력 시 printf보다 빠름.
출력 함수 선택: 포맷 필요→printf, 단순 문자열→puts, 한 문자→putchar. 상황에 맞는 도구를 골라!
wprintf
와이드 문자열 출력. . wprintf(L"한글 %d\n", 42). setlocale(LC_ALL, "") 필수!
%g
실수를 짧은 형식으로. %f와 %e 중 짧은 쪽 자동 선택. 불필요한 0 제거. 깔끔한 숫자 출력!
#010x
%#010x = "0x" 접두어 + 0채움 + 10칸. printf("%#010x",255) → 0x000000ff. 메모리 주소 출력에 유용.
printf 면접 문제: printf("%d %d", a, b)에서 a와 b의 평가 순서는? 답: 정의되지 않음(UB)! 순서에 의존하지 마!
dprintf
POSIX. 파일 디스크립터에 직접 출력. dprintf(fd, "x=%d", 42). 저수준 파일 I/O에서 활용.
vprintf
va_list를 받는 printf. 래퍼 함수(log_msg 등)에서 가변 인자를 전달할 때 사용. vfprintf, vsnprintf도 있음.
%b (C23)
C23에서 추가된 2진수 출력. printf("%b", 255) → 11111111. 이전에는 직접 구현 필요했던 기능!
perror
errno 기반 에러 메시지 출력. perror("fopen") → "fopen: No such file or directory". 시스템 에러에 유용.
assert
. 조건 거짓이면 프로그램 중단. assert(x>0); 디버깅 전용. NDEBUG 정의 시 비활성화.
%p
포인터 주소 출력. printf("%p", (void*)ptr). 디버깅에서 메모리 주소 확인 시 필수.
locale
지역 설정. setlocale(LC_ALL,"")로 시스템 로케일 적용. 천 단위 구분자, 소수점 표기 등에 영향.
서식 지정자
printf 안에서 값이 들어갈 자리를 표시하는 기호. %d(정수), %f(실수), %c(문자) 등이 있어요.
%d
decimal(10진수). 정수 값을 출력할 때 사용해요.

값을 계산하는 표현. a + b, price * qty 같은 것을 수식이라고 해요.
