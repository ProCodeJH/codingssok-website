6줄바깥 for(i) — 줄(행)을 담당해요. 5줄을 만들어요.
7줄안쪽 for(j) — 칸(열)을 담당해요. 한 줄에 별 5개를 찍어요.
10줄한 줄의 별이 다 찍히면 \n으로 줄을 바꿔요.

18void number_spiral(int n) {

나선 행렬
Spiral Matrix. 방향 전환: 오른쪽→아래→왼쪽→위. dx/dy 배열로 구현. 코딩 면접 단골!
파스칼 삼각형
C(n,k) = C(n-1,k-1) + C(n-1,k). 이중 for로 생성. 이항 계수, 조합론의 기초!
나선 행렬 = for문 + 방향 배열의 조합! dx/dy로 4방향 이동, 벽/방문 체크로 방향 전환!
별 찍기의 비밀: 공백 = n-i, 별 = 2i-1. 이 공식만 알면 어떤 패턴이든 만들 수 있어!

[심화 내용] 패턴의 시간/공간 복잡도
n×n 별 찍기:
시간: O(n²) — 이중 for
공간: O(1) — 변수만

파스칼 삼각형:
시간: O(n²) — 이중 for
공간: O(n²) — 2D 배열
→ O(n)으로 최적화 가능! (1행만 유지)

나선 행렬:
시간: O(n²) — 모든 칸 방문
공간: O(n²) — 2D 배열

n이 커질 때:
n=100: 10,000 연산 (즉시)
n=1000: 1,000,000 연산 (0.01초)
n=10000: 100,000,000 연산 (1초)
Floyd 삼각형
Robert Floyd의 삼각형. 자연수를 삼각형으로 배치. 1, 2 3, 4 5 6... for문 연습의 기본!
파스칼 삼각형의 비밀: C(n,k) = C(n-1,k-1)+C(n-1,k). 이항 계수, 확률, 조합의 시각화!

라틴 방진. n×n 격자에 n개 기호를 행/열 중복 없이. 스도쿠의 일반화! 실험 설계에 활용.

Young Tableau. 행과 열이 모두 정렬된 행렬. O(n) 검색 가능. 알고리즘 면접 문제!

열 방정식. 2D 격자에서 이웃 평균으로 온도 확산. for 반복으로 시뮬레이션. 물리 시뮬의 기초!

dithering
디더링. 적은 색으로 많은 색 표현. Floyd-Steinberg 알고리즘. for문으로 오차 확산. 흑백 인쇄의 핵심!
