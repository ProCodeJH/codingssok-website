[핵심] 중첩 for의 원리 — 줄과 칸
바깥 for = 줄(행, row)
안쪽 for = 칸(열, column)

바깥 for가 1번 돌 때, 안쪽 for가 전부 돌아요.

비유: 극장 좌석표를 생각해 보세요!
바깥 for = 1열, 2열, 3열... (줄 이동)
안쪽 for = 1번, 2번, 3번... (좌석 이동)
안쪽이 끝나면 줄바꿈(printf "\n")!

중첩 for에서 핵심: 바깥 루프 1회전 = 안쪽 루프 전체 실행 = 화면 한 줄. 이 관계를 이해하면 어떤 패턴이든 만들 수 있어요!

별 찍기 3 — 오른쪽 정렬 삼각형

별 앞에 공백을 넣으면 오른쪽으로 정렬!

1
공백 개수 계산
n-1-i개의 공백 먼저 출력
2
별 개수 계산
i+1개의 별 출력
3
줄바꿈
printf("\n")으로 다음 행으로
4
반복!
i가 0부터 n-1까지

오른쪽 정렬의 비밀: 공백 + 별 = 항상 n개! n=5일 때: 4+1, 3+2, 2+3, 1+4, 0+5.

공백+별=n
오른쪽 정렬의 핵심 공식! 공백 = n-1-i, 별 = i+1. 합은 항상 n!
star3.c

2
3int main(void) {
4 int n;

6
7 for (int i = 1; i <= n; i++) {
8 // 공백 (n-i)개
9 for (int j = 0; j < n - i; j++)
10 printf(" ");
11 // 별 i개
12 for (int j = 0; j < i; j++)
13 printf("*");
14 printf("\n");
15 }
16 return 0;
17}

줄3: 공백2 + 별3

2. i와의 관계를 찾아요

2칸 폭으로 정수 출력. 자릿수 정렬에 필수! 표/표격 출력 시 항상 사용.

[컴퓨팅 사고력] 2차원 사고

[사고력] 4요소 심화 연결
1. 분해(Decomposition):
다이아몬드 = 윗삼각형 + 아랫삼각형으로 분해!
각 줄 = 공백 for + 별 for + 줄바꿈으로 분해!

2. 패턴 인식(Pattern Recognition):
줄1: 공백4 별1 → 줄2: 공백3 별2 → 줄3: 공백2 별3
공백 = n-i, 별 = i (또는 2i+1) ← 패턴!
이 관계식을 찾으면 코드가 바로 나와요!

3. 추상화(Abstraction):
별 찍기의 핵심: '줄 번호 i와 출력 개수의 관계'만 찾으면 됨
별 모양, 문자 종류, 크기는 관계식에 영향 없음!

4. 알고리즘 설계(Algorithm Design):
중첩 for의 시간 복잡도: O(n²)
n이 커지면 실행 시간이 n의 제곱으로 증가!
이중 반복 = 2차원 문제의 기본 해법.

패턴별 핵심 공식
직사각형: 별=n (고정)
왼쪽 삼각형: 별=i+1
역삼각형: 별=n-i

다이아몬드: 윗삼각+역삼각
모래시계: 역삼각+윗삼각
숫자 피라미드: 별 대신 j 출력
