합계 — 누적 패턴

1+2+...+100을 구할 때 sum = 0에서 시작해서 하나씩 더해 가는 것을 누적 패턴이라고 해요!

sum += x

반복 누적
결과
sum=0으로 초기화 → for로 반복 → sum+=x로 누적. 이 3단계가 핵심!

누적 패턴은 프로그래밍의 가장 기본적인 패턴입니다. 초기화 → 반복 누적 → 결과 사용

sum_basic.c
1#include <stdio.h>
2
3int main(void) {
4 int n;
5 printf("N: "); scanf("%d", &n);
6
7 int sum = 0; // 누적 변수: 반드시 0으로 초기화!
8 for (int i = 1; i <= n; i++) {
9 sum += i; // sum = sum + i
10 }
11
12 printf("1~%d 합계: %d\n", n, sum);
13 return 0;
14}
실행 결과
N: 100 1~100 합계: 5050
[핵심] 누적 패턴의 3단계
1. 초기화: sum = 0 (합계) 또는 product = 1 (곱)
2. 반복: 매번 sum += 값
3. 결과: 반복 끝나면 sum에 결과!

주의: 초기화를 안 하면 쓰레기값에 더해져서 틀려요!
비유: 저금통! 0원에서 시작, 매일 동전 넣기, 마지막에 깨서 확인!

[컴퓨팅 사고력] 누적과 리덕션

[사고력] 4요소 심화 연결
1. 분해: '평균 구하기'를 → 합계+개수+나눗셈 3단계로 분해
2. 패턴 인식: sum+=x, product*=x, count++ — 전부 누적 패턴! 초기값만 다름(0/1/0)
3. 추상화: 리덕션(Reduction): 여러 값을 하나로 '축약'. sum, max, min, count 전부 리덕션!
4. 알고리즘 설계: 온라인 알고리즘: 데이터를 저장 안 하고 흘려보내며 계산. 메모리 O(1)!

10축 심화 연결

합계 공식: 가우스 공식

1+2+3+...+n = n(n+1)/2. 반복문 없이 O(1)으로 계산!

✕ 잘못된 코드
int sum=0; for (int i=1;i<=n;i++) sum+=i; // O(n) 반복
for 반복 = O(n)
✓ 올바른 코드
int sum = n*(n+1)/2; // O(1) 상수 시간!
공식 = O(1)! n=1억이어도 즉시!
가우스가 10살 때 발견! 1~100 합=100*101/2=5050!
가우스
1+2+...+n = n(n+1)/2. 1~100 = 5050. 7살 때 발견한 수학 천재!
10축 심화 연결
[컴퓨팅사고력 — 리덕션] 합계/평균/최대/최소는 전부 '여러 값→하나' 리덕션. 초기값+연산만 다름!

[심화] 분산과 표준편차

평균만으로는 부족합니다. 데이터가 평균에서 얼마나 퇨어져 있는지도 중요!

variance.c
1#include <stdio.h>
2#include <math.h>
3
4int main(void) {
5 int data[] = {85, 90, 78, 92, 88};
6 int n = 5;
7
8 // 1. 평균
9 double sum = 0;
10 for (int i=0; i<n; i++) sum += data[i];
11 double avg = sum / n;
12
13 // 2. 분산 = 편차제곱의 평균
14 double var_sum = 0;
15 for (int i=0; i<n; i++)
16 var_sum += (data[i]-avg)*(data[i]-avg);
17 double variance = var_sum / n;
18 double stddev = sqrt(variance);
19
20 printf("평균: %.2f\n", avg);
21 printf("분산: %.2f\n", variance);
22 printf("표준편차: %.2f\n", stddev);
23 return 0;
24}
실행 결과
평균: 86.60 분산: 23.84 표준편차: 4.88

표준편차
분산의 제곱근. 원래 단위로 복원. AI의 정규분포에 필수!

[심화] MapReduce 패턴

빅데이터에서 합계/평균을 구하려면? 데이터를 나누어(Map) 각각 합계하고, 결과를 합치는(Reduce) 패턴!

Map + Reduce
Map: 데이터를 N개 부분으로 나누기
Reduce: 각 부분의 합계를 하나로 합치기

우리의 for(sum+=x)는 가장 기본적인 Reduce!
Google의 MapReduce! 수십억 개 데이터를 수천 대 컴퓨터에 나눠서 합계!
