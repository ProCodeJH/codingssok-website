[핵심] 최대/최소 패턴
최대값: max = INT_MIN (또는 첫 번째 값)으로 초기화
if (x > max) max = x;

최소값: min = INT_MAX로 초기화
if (x < min) min = x;

비유: 키 재기 대회!
'현재 1등(max)'을 기억해두고, 새 참가자가 더 크면 1등 교체!

[심화] 토너먼트 방식 max/min

비교 횟수를 줄이는 방법: 쌍으로 비교 → 최대 후보까리, 최소 후보까리 분리!

tournament_max.c
1#include <stdio.h>
2
3int main(void) {
4 int arr[] = {23, 67, 12, 89, 45, 34, 78, 56};
5 int n = 8;
6
7 // 쌍으로 비교해서 max/min 후보 분리
8 int big[4], small[4];
9 for (int i = 0; i < n/2; i++) {
10 if (arr[2*i] > arr[2*i+1]) {
11 big[i] = arr[2*i];
12 small[i] = arr[2*i+1];
13 } else {
14 big[i] = arr[2*i+1];
15 small[i] = arr[2*i];
16 }
17 }
18
19 // big에서 max, small에서 min
20 int max=big[0], min=small[0];
21 for(int i=1;i<n/2;i++) {
22 if(big[i]>max) max=big[i];
23 if(small[i]<min) min=small[i];
24 }
25 printf("max=%d, min=%d\n", max, min);

27}
