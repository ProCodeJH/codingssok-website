[심화] FizzBuzz와 비트 연산

모듈로 없이 비트 플래그로 FizzBuzz를 구현할 수도 있습니다.

비트 플래그 방식
int flag = 0;
if (i%3==0) { printf("Fizz"); flag |= 1; }
if (i%5==0) { printf("Buzz"); flag |= 2; }
if (!flag) printf("%d", i);

flag=0: 숫자, flag=1: Fizz, flag=2: Buzz, flag=3: FizzBuzz
비트 OR
flag |= 1 = flag의 비트 0을 1로. flag |= 2 = 비트 1을 1로. 저수준 플래그!
