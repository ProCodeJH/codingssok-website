fprintf = printf의 파일 버전! 화면 대신 파일에 쓰기!
fopen/fclose
파일 열기/닫기. "w"=쓰기, "r"=읽기, "a"=추가. fclose 필수!
fprintf
printf의 파일 버전. fprintf(fp, "형식", 값)으로 파일에 출력!
