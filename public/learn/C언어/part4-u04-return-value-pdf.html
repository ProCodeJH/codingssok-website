반환값이 있는 함수

return문으로 함수가 계산한 결과를 돌려주는 방법을 배워요.

📚 이번에 배울 것

  • return문으로 값을 돌려주는 함수를 만들 수 있어요.
  • 반환값을 변수에 저장하거나 printf에 직접 쓸 수 있어요.
  • int, double, char 등 다양한 반환형을 사용할 수 있어요.
  • void 함수와 반환형 함수의 차이를 이해해요.

✅ 시작 전 체크

  • 매개변수가 있는 함수를 만들 수 있다
  • call by value를 이해한다
Part 1 진행률: 4%

계산 결과를 돌려받고 싶다면?

void 함수는 일만 하고 끝나. 그런데 덧셈 함수를 만들었는데 결과를 main()에서 쓰고 싶으면? 그때 return을 써!
return_basic.c
1#include <stdio.h>
2
3int add(int a, int b) {
4 return a + b;
5}
6
7int main(void) {
8 int result = add(3, 5);
9 printf("3 + 5 = %d\n", result);
10 return 0;
11}
실행 결과
3 + 5 = 8
3줄int add(int a, int b) — 앞의 int가 반환형이에요. '이 함수는 정수를 돌려줄 거야'라는 뜻.
4줄return a + b; — a + b의 결과를 호출한 곳으로 돌려보내요. return을 만나면 함수가 즉시 끝나요.
8줄int result = add(3, 5); — add 함수가 돌려준 8이 result에 저장돼요.

return문의 동작 흐름

함수가 return한 값은 호출한 곳으로 돌아온다
return은 두 가지 일을 해. 1) 값을 돌려주고, 2) 함수를 끝낸다. return 뒤의 코드는 실행되지 않아!

void 함수 vs 반환형 함수

출력만 하면 void, 결과가 필요하면 반환형 함수

자주 하는 실수

return을 빼먹으면 함수가 쓰레기 값을 돌려줘. 컴파일러가 경고를 주니까 꼭 확인해!
✕ 잘못된 코드
int add(int a, int b) { int sum = a + b; // return 깜빡! }
return이 없으면 쓰레기 값이 반환돼요
✓ 올바른 코드
int add(int a, int b) { int sum = a + b; return sum; }
반드시 return으로 값을 돌려줘야 해요
✕ 잘못된 코드
void add(int a, int b) { return a + b; }
void 함수에서 값을 return하면 에러!
✓ 올바른 코드
int add(int a, int b) { return a + b; }
값을 돌려주려면 반환형을 맞춰줘야 해요

[심화 내용] 반환값과 ABI
반환값 전달 방식 (x86-64):
int/pointer → RAX 레지스터

큰 구조체 → 숨겨진 포인터 매개변수!

NRVO (Named Return Value Optimization):
Point p; return p; → 복사 생략!

1#include <stdio.h>

4// 패턴 1: 구조체 반환
5typedef struct { int quot; int rem; } DivResult;
6
7DivResult safe_div(int a, int b) {
8 DivResult r;
9 r.quot = a / b;
10 r.rem = a % b;
11 return r;
12}
13
14// 패턴 2: 에러 코드 반환 + 포인터로 결과
15typedef enum { OK=0, ERR_OVERFLOW=1, ERR_DIVZERO=2 } ErrorCode;
16
17ErrorCode checked_add(int a, int b, int *result) {
18 long long sum = (long long)a + b;
19 if (sum > 2147483647 || sum < -2147483648)
20 return ERR_OVERFLOW;
21 *result = (int)sum;
22 return OK;
23}
24
25// 패턴 3: 센티널 값 (-1 = 에러)
26int find_char(const char *s, char c) {
27 for (int i = 0; s[i]; i++)
28 if (s[i] == c) return i;
29 return -1; // 센티널: 없음
30}
31
32// 패턴 4: 2차 방정식 근
33typedef struct { double r1, r2; int count; } Roots;
34
35Roots quadratic(double a, double b, double c) {
36 Roots r = {0, 0, 0};
37 double disc = b*b - 4*a*c;
38 if (disc > 0) {
39 r.r1 = (-b + sqrt(disc)) / (2*a);
40 r.r2 = (-b - sqrt(disc)) / (2*a);
41 r.count = 2;
42 } else if (disc == 0) {
43 r.r1 = -b / (2*a);
44 r.count = 1;
45 }
46 return r;
47}
48
49int main() {
50 DivResult d = safe_div(17, 5);
51 printf("17/5 = %d 나머지 %d\n", d.quot, d.rem);
52
53 int result;
54 ErrorCode ec = checked_add(2000000000, 1000000000, &result);
55 printf("2B+1B: %s\n", ec==ERR_OVERFLOW?"OVERFLOW":"OK");
56
57 printf("find 'l' in hello: %d\n", find_char("hello", 'l'));
58 printf("find 'z' in hello: %d\n", find_char("hello", 'z'));

[C 코드] 반환값 — 비트 조작 유틸리티
1#include <stdio.h>
2
3typedef struct { int value; int bits[32]; int nbits; } BitInfo;
4
5BitInfo analyze_bits(int n) {
6 BitInfo info;
7 info.value = n;
8 info.nbits = 0;
9 unsigned int u = (unsigned int)n;
10 while (u > 0) {
11 info.bits[info.nbits++] = u & 1;
12 u >>= 1;
13 }
14 if (info.nbits == 0) { info.bits[0] = 0; info.nbits = 1; }
15 return info;
16}
17
18void print_bits(BitInfo bi) {
19 printf("%d = 0b", bi.value);
20 for (int i = bi.nbits-1; i >= 0; i--) printf("%d", bi.bits[i]);
21 printf(" (%d bits)\n", bi.nbits);
22}
23
24int popcount(int n) {
25 int count = 0;
26 unsigned int u = (unsigned int)n;
27 while (u) { count += u & 1; u >>= 1; }
28 return count;
29}
30
31int is_bit_set(int n, int pos) { return (n >> pos) & 1; }
32int set_bit(int n, int pos) { return n | (1 << pos); }
33int clear_bit(int n, int pos) { return n & ~(1 << pos); }
34int toggle_bit(int n, int pos) { return n ^ (1 << pos); }
35
36int main() {
37 BitInfo b = analyze_bits(42);
38 print_bits(b);
39
40 printf("popcount(42) = %d\n", popcount(42));
41 printf("bit 3 set? %d\n", is_bit_set(42, 3));
42 printf("set bit 0: %d\n", set_bit(42, 0));
43 printf("clear bit 1: %d\n", clear_bit(42, 1));
44 printf("toggle bit 5: %d\n", toggle_bit(42, 5));

48}

1#include <stdio.h>
2
3typedef struct {
4 int is_prime; int is_perfect; int is_armstrong;
5 int digit_count; int digit_sum; int reversed;
6 int factors[20]; int nfactors;
7} NumAnalysis;
8
9NumAnalysis analyze(int n) {
10 NumAnalysis r = {1,0,0,0,0,0,{0},0};
11 // prime
12 if(n<2) r.is_prime=0;
13 for(int i=2;i*i<=n&&r.is_prime;i++) if(n%i==0) r.is_prime=0;
14 // factors
15 for(int i=1;i<=n;i++) if(n%i==0) r.factors[r.nfactors++]=i;
16 // perfect (sum of proper divisors = n)
17 int sum=0; for(int i=0;i<r.nfactors-1;i++) sum+=r.factors[i];
18 r.is_perfect = (sum==n && n>1);
19 // digits
20 int tmp=n; while(tmp>0){r.digit_count++;r.digit_sum+=tmp%10;
21 r.reversed=r.reversed*10+tmp%10;tmp/=10;}
22 // armstrong
23 tmp=n; int asum=0;
24 while(tmp>0){int d=tmp%10;int pw=1;for(int i=0;i<r.digit_count;i++)pw*=d;
25 asum+=pw;tmp/=10;} r.is_armstrong=(asum==n);
26 return r;
27}
28

30 int nums[]={28,153,17,6,371};
31 for(int i=0;i<5;i++) {
32 NumAnalysis a=analyze(nums[i]);
33 printf("%d: prime=%d perfect=%d armstrong=%d digits=%d sum=%d rev=%d\n",
34 nums[i],a.is_prime,a.is_perfect,a.is_armstrong,
35 a.digit_count,a.digit_sum,a.reversed);
36 }
37 return 0;
