코드 분석: 왜 함수로 나눌까?

함수 분리의 장점
1. 가독성: main을 보면 프로그램의 전체 흐름이 한눈에 보여요
2. 재사용성: calcAverage 같은 함수는 다른 프로그램에서도 쓸 수 있어요
3. 디버깅: 문제가 생기면 해당 함수만 확인하면 돼요
4. 확장성: 기능을 추가할 때 새 함수만 만들면 돼요
함수로 나누면 코드가 깔끔하고 관리하기 쉬워져요
이 프로그램에서 주의할 점: count를 포인터로 넘긴 이유는 inputData 안에서 값을 채워야 하기 때문이야. 값 전달이면 main의 count가 안 바뀌거든!

qsort 콜백
C 표준 정렬. 비교 함수를 콜백으로 전달. 어떤 타입이든 정렬 가능. 평균 O(n log n)!
석차 알고리즘
정렬 후 순위 부여. 동점자 처리: 같은 점수면 같은 등수. 다음 등수는 건너뜀(1,1,3) or 연속(1,1,2).

qsort + 콜백 = C의 정렬 마법! 비교 함수만 바꾸면 이름순, 점수순, 등수순 자유자재!

15void calc_stats(Student *s) {
16 s->total = 0;
17 for (int i = 0; i < 5; i++) s->total += s->scores[i];
18 s->avg = s->total / 5.0;
19 if (s->avg >= 90) s->grade = 'A';
20 else if (s->avg >= 80) s->grade = 'B';
21 else if (s->avg >= 70) s->grade = 'C';
22 else if (s->avg >= 60) s->grade = 'D';
23 else s->grade = 'F';
24}

38void sort_by_avg(Student *arr, int n) {
39 for (int i = 0; i < n-1; i++)
40 for (int j = 0; j < n-1-i; j++)
41 if (arr[j].avg < arr[j+1].avg) {
42 Student t = arr[j]; arr[j] = arr[j+1]; arr[j+1] = t;
43 }
44}

46void class_stats(Student *arr, int n) {
47 double sum = 0, max_avg = 0, min_avg = 100;
48 int grade_count[5] = {0}; // A,B,C,D,F
49
50 for (int i = 0; i < n; i++) {
51 sum += arr[i].avg;
52 if (arr[i].avg > max_avg) max_avg = arr[i].avg;
53 if (arr[i].avg < min_avg) min_avg = arr[i].avg;
54 int gi = arr[i].grade=='A'?0:arr[i].grade=='B'?1:
55 arr[i].grade=='C'?2:arr[i].grade=='D'?3:4;
56 grade_count[gi]++;
57 }

3
4typedef struct {
5 char name[30];
6 int kor, eng, math;
7 double avg;
8 char grade;
9} Student;
10
11void calc(Student *s) {
12 s->avg = (s->kor + s->eng + s->math) / 3.0;
13 s->grade = s->avg>=90?'A':s->avg>=80?'B':s->avg>=70?'C':s->avg>=60?'D':'F';
14}

16int save_csv(const char *filename, Student *arr, int n) {
17 FILE *fp = fopen(filename, "w");
18 if (!fp) return -1;
19 fprintf(fp, "이름,국어,영어,수학,평균,등급\n");
20 for (int i = 0; i < n; i++)
21 fprintf(fp, "%s,%d,%d,%d,%.1f,%c\n",
22 arr[i].name, arr[i].kor, arr[i].eng, arr[i].math,
23 arr[i].avg, arr[i].grade);
24 fclose(fp);
25 return 0;
26}

28int load_csv(const char *filename, Student *arr, int max_n) {
29 FILE *fp = fopen(filename, "r");
30 if (!fp) return -1;
31 char line[256];
32 fgets(line, 256, fp); // 헤더 건너뜀
33 int n = 0;
34 while (n < max_n && fgets(line, 256, fp)) {
35 sscanf(line, "%[^,],%d,%d,%d",
36 arr[n].name, &arr[n].kor, &arr[n].eng, &arr[n].math);
37 calc(&arr[n]);
38 n++;
39 }
40 fclose(fp);
41 return n;
42}

14int filter_above(Student *arr, int n, double min_avg, Student *result) {
15 int count = 0;
16 for (int i = 0; i < n; i++)
17 if (arr[i].avg >= min_avg) result[count++] = arr[i];
18 return count;
19}
