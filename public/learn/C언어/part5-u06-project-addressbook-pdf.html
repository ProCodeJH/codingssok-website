프로젝트 개요: 주소록 만들기

프로그램 설계

메뉴를 출력하고, 선택에 따라 기능을 실행한 뒤 다시 메뉴로 돌아간다
주소록 프로그램 요구사항
1. 추가 -- 이름, 전화번호를 입력받아 배열에 저장
2. 검색 -- 이름으로 검색하여 전화번호 출력
3. 삭제 -- 이름으로 찾아 해당 항목 삭제
4. 전체 출력 -- 저장된 모든 연락처 출력
0. 종료 -- 프로그램 종료

17void trie_insert(TrieNode *root, const char *name) {
18 TrieNode *cur = root;
19 for (int i = 0; name[i]; i++) {
20 int idx = name[i] - 'A';
21 if (idx < 0 || idx >= ALPHA) idx = name[i] - 'a';
22 if (idx < 0 || idx >= ALPHA) continue;
23 if (!cur->children[idx]) cur->children[idx] = trie_new();
24 cur = cur->children[idx];
25 }
26 cur->is_end = 1;
27 strncpy(cur->full_name, name, 29);
28}

34 if (node->children[i]) {

36 prefix[depth+1] = 0;
37 trie_autocomplete(node->children[i], prefix, depth+1);

Trie (Prefix Tree):
각 노드가 문자 하나를 나타냄
루트→리프 경로 = 하나의 문자열
검색/삽입: O(문자열 길이) — 문자열 수와 무관!

메모리 최적화:
배열: children[26] → 26포인터 (208B/노드)
해시맵: 필요한 자식만 → 메모리 절약
Patricia Trie: 단일 자식 경로 압축
Radix Tree: 공통 접두사 합침

응용:
자동완성 (Google 검색창!)
맞춤법 검사, IP 라우팅 (longest prefix match)
