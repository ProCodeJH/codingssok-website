프로젝트 설계

14void calc_zscores(Student *arr, int n) {
15 double sum = 0, sum2 = 0;
16 for (int i = 0; i < n; i++) { sum += arr[i].score; sum2 += arr[i].score * arr[i].score; }
17 double mean = sum / n;
18 double stddev = sqrt(sum2/n - mean*mean);
19 for (int i = 0; i < n; i++)
20 arr[i].zscore = (arr[i].score - mean) / stddev;
21}

6double correlation(double *x, double *y, int n) {
7 double sx=0,sy=0,sxy=0,sx2=0,sy2=0;
8 for (int i = 0; i < n; i++) {
9 sx += x[i]; sy += y[i];
10 sxy += x[i]*y[i];
11 sx2 += x[i]*x[i]; sy2 += y[i]*y[i];
12 }
13 double num = n*sxy - sx*sy;
14 double den = sqrt((n*sx2-sx*sx) * (n*sy2-sy*sy));
15 return den == 0 ? 0 : num/den;
16}

18void regression(double *x, double *y, int n, double *slope, double *intercept) {
19 double sx=0,sy=0,sxy=0,sx2=0;
20 for (int i = 0; i < n; i++) {
21 sx += x[i]; sy += y[i];
22 sxy += x[i]*y[i]; sx2 += x[i]*x[i];
23 }
24 *slope = (n*sxy - sx*sy) / (n*sx2 - sx*sx);
25 *intercept = (sy - (*slope)*sx) / n;
26}

[심화 내용] 통계와 머신러닝 기초
피어슨 상관계수 (r):
r = Σ(x-x̄)(y-ȳ) / √(Σ(x-x̄)²·Σ(y-ȳ)²)
r = +1: 완벽한 양의 상관
r = -1: 완벽한 음의 상관
r = 0: 상관 없음

선형 회귀:
y = mx + b (최소 제곱법)
m = (nΣxy - ΣxΣy) / (nΣx² - (Σx)²)
→ 예측 모델의 기본!

머신러닝으로 확장:
다중 회귀: y = w₁x₁ + w₂x₂ + ... + b
경사 하강법: 오차를 줄이는 방향으로 w 업데이트
→ 딥러닝도 결국 이 원리의 확장!

상관관계+회귀 = 통계의 꽃! 국어-영어 상관 0.99! 수학으로 과학 점수 예측!
