qsort() 라이브러리 함수

C 표준 라이브러리의 qsort() 함수 사용법을 배워요. 비교 함수 작성법, 구조체 정렬까지 익혀요.

  • 비교 함수를 직접 작성할 수 있어요.
  • qsort()로 정수 배열을 오름차순/내림차순 정렬할 수 있어요.
  • 구조체 배열을 qsort()로 정렬할 수 있어요.

Part 1 진행률: 4%

직접 만들지 말고 라이브러리를 써!

버블, 선택, 삽입 정렬을 직접 구현하는 건 공부에 좋지만, 실전에서는 이미 만들어진 걸 쓰는 게 효율적이야. C 표준 라이브러리에 qsort()라는 강력한 정렬 함수가 있어!

qsort()는 <stdlib.h>에 포함된 C 표준 라이브러리 정렬 함수예요. 내부적으로 퀵 정렬(Quick Sort) 기반 알고리즘을 사용하며, 평균 시간 복잡도가 O(n log n)으로 매우 빨라요.

qsort() 사용법

qsort(배열, 원소개수, 원소크기, 비교함수)

비교 함수의 반환값

음수/0/양수의 의미
비교 함수는 qsort가 '두 원소의 순서를 어떻게 정할지' 물어보는 거야. 우리가 규칙을 알려주면 qsort가 알아서 정렬해 줘!

4줄비교 함수는 반드시 const void * 두 개를 받아요. qsort가 정한 규칙이에요.

[디버깅] qsort 사용 시 자주 하는 실수
1. sizeof 누락 -- sizeof(Student)가 아닌 sizeof(int)를 넣으면 엉뚱한 결과
2. 캐스팅 오류 -- void*를 올바른 타입으로 캐스팅하지 않으면 쓰레기값
3. 비교 함수 시그니처 -- 반드시 int (const void*, const void*) 형태
4. 오버플로 -- 큰 값끼리 뺄셈하면 오버플로 가능. 안전하게 if문 사용 권장
[KOI] qsort 활용 전략
KOI에서는 정렬이 필요한 문제가 매우 많아요. 직접 버블 정렬을 구현하면 시간 초과(TLE)가 날 수 있지만, qsort()를 쓰면 O(n log n)이라 안전해요. 비교 함수만 잘 만들면 어떤 데이터든 정렬 가능!

쉽게 이해하기: qsort 비교 함수
qsort가 요구하는 것:
int cmp(const void *a, const void *b);

반환값 규칙:
음수(-) → a가 b보다 앞
0 → 같음
양수(+) → a가 b보다 뒤

void* 캐스팅:
qsort는 타입을 모름 (void*로 받음)
비교 함수 안에서 원래 타입으로 캐스팅!
(int*)a → 정수 포인터로 변환 → *로 값 접근

함수 포인터 읽는 법:
int (*cmp)(const void*, const void*)
→ "void* 2개를 받아 int를 반환하는 함수의 포인터"

[C 코드] qsort — Quick Sort 직접 구현
1#include <stdio.h>
2
3void swap(int *a, int *b) { int t=*a; *a=*b; *b=t; }
4
5// Lomuto 파티션
6int partition(int *a, int lo, int hi) {
7 int pivot = a[hi]; // 마지막 원소를 피벗으로
8 int i = lo - 1;
9 for (int j = lo; j < hi; j++) {
10 if (a[j] <= pivot) {
11 i++;
12 swap(&a[i], &a[j]);
13 }
14 }
15 swap(&a[i+1], &a[hi]);
16 return i + 1;
17}
18
19// Hoare 파티션 (더 효율적)
20int partition_hoare(int *a, int lo, int hi) {
21 int pivot = a[(lo+hi)/2];
22 int i = lo-1, j = hi+1;
23 while (1) {
24 do i++; while (a[i] < pivot);
25 do j--; while (a[j] > pivot);
26 if (i >= j) return j;
27 swap(&a[i], &a[j]);
28 }
29}
30
31void quick_sort(int *a, int lo, int hi) {
32 if (lo < hi) {
33 int p = partition(a, lo, hi);
34 quick_sort(a, lo, p-1);
35 quick_sort(a, p+1, hi);
36 }
37}
38

42 printf("Before: "); for(int i=0;i<n;i++) printf("%d ",a[i]); printf("\n");
43 quick_sort(a, 0, n-1);

Lomuto vs Hoare
파티션 방식. Lomuto: 단순, 교환 많음. Hoare: 복잡하지만 교환 3배 적음! 실전은 Hoare.
Quick Sort의 핵심 = partition! 피벗 기준으로 작은 건 왼쪽, 큰 건 오른쪽. 재귀!
[심화 내용] 퀵소트 최적화
피벗 선택:
첫 원소: 정렬/역순에서 O(n²)!
랜덤: 기대 O(n log n)
Median-of-3: 첫/중/끝 중앙값. 실전 최적

3-way Partition (Dutch Flag):
중복 많으면 pivot 분리
→ 중복 구간 스킵! O(n) 가능

Tail Call Elimination:
작은 쪽을 재귀, 큰 쪽을 반복
→ 스택 깊이 O(log n) 보장

Introsort (C++ std::sort):
Quick + Heap + Insertion
깊이 > 2·log₂n → Heap으로 전환
Dutch National Flag
네덜란드 국기 문제. 3-way partition. 빨/흰/파 분류. 중복 많은 배열에서 퀵소트 최적화!
Median-of-3: 첫/중/끝에서 중앙값을 피벗으로! 최악 O(n²) 확률 대폭 감소!
[C 코드] qsort — 3-Way Quick Sort
1#include <stdio.h>
2
3void swap(int *a, int *b) { int t=*a;*a=*b;*b=t; }
4
5// Dutch National Flag: <pivot / ==pivot / >pivot
6void quicksort_3way(int *a, int lo, int hi) {
7 if (lo >= hi) return;
8 int lt = lo, gt = hi, i = lo;
9 int pivot = a[lo];
10 while (i <= gt) {
11 if (a[i] < pivot) swap(&a[lt++], &a[i++]);
12 else if (a[i] > pivot) swap(&a[i], &a[gt--]);
13 else i++;
14 }
15 // a[lo..lt-1] < pivot = a[lt..gt] < a[gt+1..hi]
16 quicksort_3way(a, lo, lt-1);
17 quicksort_3way(a, gt+1, hi);
18}
19
20int main() {
21 int a[] = {3,1,4,1,5,9,2,6,5,3,5,8,9,7,9,3,2,3,8,4};
22 int n = 20;
23 printf("Before: "); for(int i=0;i<n;i++) printf("%d ",a[i]); printf("\n");
24 quicksort_3way(a, 0, n-1);
25 printf("After: "); for(int i=0;i<n;i++) printf("%d ",a[i]); printf("\n");
26 return 0;
27}

3-way partition
3분할. 같은 값을 한 영역에 모음. 중복 많으면 효과 대! Dijkstra의 Dutch Flag Problem.
3-way: 중복 9가 3개면 한번에 스킵! 일반 퀵소트는 하나씩 처리. 중복 많을수록 빠름!
[C 코드] qsort — Quick Sort 직접 구현
1#include <stdio.h>
2
3// Lomuto 파티션
4int partition(int *a, int lo, int hi) {
5 int pivot = a[hi];
6 int i = lo - 1;
7 for (int j = lo; j < hi; j++) {
8 if (a[j] <= pivot) {
9 i++;
10 int t=a[i]; a[i]=a[j]; a[j]=t;
11 }
12 }
13 int t=a[i+1]; a[i+1]=a[hi]; a[hi]=t;
14 return i + 1;
15}
16
17// Hoare 파티션 (더 빠름)
18int hoare_partition(int *a, int lo, int hi) {
19 int pivot = a[(lo+hi)/2];
20 int i = lo-1, j = hi+1;
21 while (1) {
22 do i++; while (a[i] < pivot);
23 do j--; while (a[j] > pivot);
24 if (i >= j) return j;
25 int t=a[i]; a[i]=a[j]; a[j]=t;
26 }
27}
28
29void quicksort(int *a, int lo, int hi) {
30 if (lo < hi) {
31 int p = partition(a, lo, hi);
32 quicksort(a, lo, p-1);
33 quicksort(a, p+1, hi);
34 }
35}
36
37// 3-way Quick Sort (중복 최적화)
38void quicksort_3way(int *a, int lo, int hi) {
39 if (lo >= hi) return;
40 int lt=lo, gt=hi, pivot=a[lo], i=lo+1;
41 while (i <= gt) {
42 if (a[i] < pivot) { int t=a[lt];a[lt]=a[i];a[i]=t; lt++;i++; }
43 else if (a[i] > pivot) { int t=a[gt];a[gt]=a[i];a[i]=t; gt--; }
44 else i++;
45 }
46 quicksort_3way(a, lo, lt-1);
47 quicksort_3way(a, gt+1, hi);
48}

[C 코드] Quick Sort — 직접 구현
1#include <stdio.h>
2
3void swap(int *a, int *b) { int t=*a; *a=*b; *b=t; }
4
5// Lomuto 파티션
6int partition_lomuto(int *a, int lo, int hi) {
7 int pivot = a[hi];
8 int i = lo - 1;
9 for (int j = lo; j < hi; j++) {
10 if (a[j] <= pivot) { i++; swap(&a[i], &a[j]); }
11 }
12 swap(&a[i+1], &a[hi]);
13 return i + 1;
14}
15
16// Hoare 파티션 (더 빠름)
17int partition_hoare(int *a, int lo, int hi) {
18 int pivot = a[(lo+hi)/2];
19 int i = lo-1, j = hi+1;
20 while (1) {
21 do i++; while (a[i] < pivot);
22 do j--; while (a[j] > pivot);
23 if (i >= j) return j;
24 swap(&a[i], &a[j]);
25 }
26}
27
28void quicksort(int *a, int lo, int hi) {
29 if (lo < hi) {
30 int p = partition_lomuto(a, lo, hi);
31 quicksort(a, lo, p-1);
32 quicksort(a, p+1, hi);
33 }
34}
35
36// 3-way partition (중복 최적화, Dutch National Flag)
37void quicksort_3way(int *a, int lo, int hi) {
38 if (lo >= hi) return;
39 int lt = lo, gt = hi, i = lo+1;
40 int pivot = a[lo];
41 while (i <= gt) {
42 if (a[i] < pivot) swap(&a[lt++], &a[i++]);
43 else if (a[i] > pivot) swap(&a[i], &a[gt--]);
44 else i++;
45 }
46 quicksort_3way(a, lo, lt-1);
47 quicksort_3way(a, gt+1, hi);
48}
49
50int main() {
51 int a[] = {10,80,30,90,40,50,70};
52 int n = 7;
53 printf("QuickSort:\n Before: ");
54 for(int i=0;i<n;i++) printf("%d ",a[i]); printf("\n");
55 quicksort(a, 0, n-1);
56 printf(" After: ");
57 for(int i=0;i<n;i++) printf("%d ",a[i]); printf("\n");
58
59 int b[] = {3,1,4,1,5,9,2,6,5,3,5};
60 int m = 11;
61 printf("\n3-Way (many duplicates):\n Before: ");
62 for(int i=0;i<m;i++) printf("%d ",b[i]); printf("\n");
63 quicksort_3way(b, 0, m-1);
64 printf(" After: ");
65 for(int i=0;i<m;i++) printf("%d ",b[i]); printf("\n");
66 return 0;
67}

쉽게 이해하기: Quick Sort의 원리
비유: 키로 줄 세우기!
1. 한 명을 기준(pivot)으로 세움
2. 기준보다 작은 애들 → 왼쪽
3. 기준보다 큰 애들 → 오른쪽
4. 왼쪽/오른쪽 각각 반복!

파티션이 핵심:
[3 7 2 5 1 8 4] pivot=4
→ [3 2 1] [4] [7 5 8]
→ 각각 다시 파티션...

최악을 피하는 방법:
• 랜덤 피벗
• Median-of-3 (첫/중/끝 중앙값)
• 3-way: 중복이 많으면 한꺼번에 처리
⚠️ Quick Sort 최악의 경우
이미 정렬된 배열 + 첫 원소가 피벗:
[1 2 3 4 5] → pivot=1 → [](1)[2 3 4 5]
→ 한쪽만 계속 → O(n²)!

해결: median-of-3 또는 랜덤 피벗
Lomuto partition
Lomuto 파티션. 피벗=마지막 원소. i가 작은 값 영역 추적. 구현 쉬움. Hoare보다 교환 많음.

C 표준 라이브러리의 범용 정렬 함수. stdlib.h에 선언되어 있다.
비교 함수(comparator)
두 원소를 받아 음수/0/양수를 반환하여 순서를 결정하는 함수.
void 포인터(void *)
어떤 타입이든 가리킬 수 있는 범용 포인터. 사용 전 캐스팅 필요.
함수 포인터
함수의 주소를 저장하는 포인터. qsort의 4번째 인자가 이것.
O(n log n)
qsort의 평균 시간 복잡도. O(n^2)보다 훨씬 빠르다.
