C에서 참과 거짓은 숫자다

C언어에는 원래 '참/거짓' 전용 자료형이 없었어. 대신 0이면 거짓, 0이 아니면 참이라는 규칙을 써 왔지. 그런데 이게 좀 헷갈리잖아?

C언어에서는 정수 0이 거짓(false)이고, 0이 아닌 모든 값이 참(true)이에요. 1, -1, 42, 100 모두 참이에요. 이건 C의 아주 오래된 규칙이에요.

7 if (0) printf("이건 출력 안 됨\n");
8
9 /* 비교 연산의 결과도 정수 */
10 printf("3 > 2 = %d\n", 3 > 2); /* 1 */
11 printf("3 < 2 = %d\n", 3 < 2); /* 0 */
12 return 0;

연습문제

OX 퀴즈

C에서 정수 -1은 조건식에서 참(true)으로 취급된다.

정답: ⭕ O
C에서 0만 거짓이고, -1은 0이 아니므로 참이에요.
객관식

stdbool.h를 포함했을 때, true의 값은?

  1. -1
  2. 0
  3. 1
  4. 참
정답: 3
stdbool.h에서 true는 정수 1로 정의되어 있어요.
단답형

bool 반환 함수의 이름을 지을 때 관례적으로 어떤 접두사로 시작하나요?

정답: is
isEven, isPrime처럼 is로 시작하면 '~인가?'라는 의미가 되어 가독성이 좋아져요.
객관식

다음 중 stdbool.h 없이도 동작하는 코드는?

  1. bool flag = true;
  2. int flag = 1; if (flag) { }
  3. _Bool flag = false;
  4. boolean flag = true;
정답: 2
int와 if는 C 기본 문법이에요. 0이 아닌 값은 참이므로 if(1)은 항상 실행돼요.
OX 퀴즈

C99 이전에는 stdbool.h 헤더가 없어서 bool 타입을 직접 정의해야 했다.

정답: ❌ X
bool 타입은 C99에서 stdbool.h를 통해 공식 도입되었습니다. 그 이전에는 #define이나 enum으로 직접 정의했습니다.

1#include <stdio.h>

4// === 무차별 대입 SAT Solver ===
5// (A OR B) AND (NOT A OR C) AND (NOT B OR NOT C)
6typedef bool(*Clause)(bool a, bool b, bool c);
7
8bool c1(bool a,bool b,bool c) { return a || b; }
9bool c2(bool a,bool b,bool c) { return !a || c; }
10bool c3(bool a,bool b,bool c) { return !b || !c; }
11
12void brute_force_sat() {
13 Clause clauses[] = {c1, c2, c3};
14 int nclauses = 3;
15 int solutions = 0;
16
17 printf(" A B C | c1 c2 c3 | SAT\n");
18 printf(" ------+---------+----\n");
19
20 for (int mask = 0; mask < 8; mask++) {
21 bool a = (mask>>2)&1, b = (mask>>1)&1, c = mask&1;
22 bool sat = true;
23 printf(" %d %d %d |", a, b, c);
24 for (int i = 0; i < nclauses; i++) {
25 bool r = clauses[i](a, b, c);
26 printf(" %d ", r);
27 if (!r) sat = false;
28 }
29 printf("| %s\n", sat?"✅":"❌");
30 if (sat) solutions++;
31 }
32 printf(" Solutions: %d/8\n", solutions);
33}

implication
함축. A→B = ¬A∨B. "A이면 B". 대우: ¬B→¬A. 수학적 증명과 프로그래밍 조건의 기초.

A→B = !A||B. "비가 오면 우산"="비 안 오거나 우산". 논리학을 코드로 증명!

C의 참/거짓. 0=false, 나머지=true. NULL도 false! if(ptr) = if(ptr!=NULL). 관용적 널 체크.
