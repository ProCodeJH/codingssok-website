변수가 뭘까?

안녕! 지난 시간에 print()로 글자를 출력했지? 오늘은 데이터를 저장하는 방법을 배울 거야. 바로 변수야!

변수는 값을 담는 상자예요. 상자에 이름을 붙여서, 나중에 그 이름으로 값을 꺼내 쓸 수 있어요.

예를 들어, '내 이름은 자현이다'를 저장하고 싶으면 name = "자현"이라고 쓰면 돼요. 이제 name이라는 상자 안에 "자현"이 들어있어요.

변수 = 이름이 붙은 상자
name←"자현"
name이라는 상자에 "자현"을 넣었어요

age←12
age라는 상자에 12를 넣었어요

3줄age = 12 — age 변수에 정수 12를 저장해요. 정수는 따옴표 없이 써요.
4줄height = 152.5 — height 변수에 실수(소수점) 152.5를 저장해요.

[핵심] = 기호의 의미
수학에서 =는 '같다'라는 뜻이에요.
프로그래밍에서 =는 '오른쪽 값을 왼쪽 변수에 넣어라'라는 뜻이에요!

age = 12 → '12를 age에 넣어라'
12 = age → 에러! 숫자에 값을 넣을 수 없어요.
파이썬의 =는 수학의 '같다'가 아니라 '저장해!'라는 뜻이에요. 화살표(←)로 생각하면 이해가 쉬워요.
[팁] 변수 이름과 문자열의 차이
print(name) → 변수 name의 값을 출력 (자현)
print("name") → 그냥 글자 name을 출력

변수 이름 짓기 규칙

변수 이름을 아무렇게나 지을 수는 없어요. 몇 가지 규칙이 있어요.

[핵심] 변수 이름 규칙
1. 영문자, 숫자, 밑줄(_)만 사용 가능
OK: name, age2, my_score
NO: my-score, my score, 이름

2. 숫자로 시작할 수 없어요
OK: score1
NO: 1score

3. 파이썬 예약어(키워드)는 쓸 수 없어요
NO: if, for, while, print, True, class

4. 대소문자를 구별해요
Name과 name은 다른 변수예요!
✕ 잘못된 코드
1st_place = "금메달" my-name = "자현" for = 10
숫자 시작, 하이픈, 예약어 — 전부 에러!
✓ 올바른 코드
first_place = "금메달" my_name = "자현" count = 10
밑줄(_)로 단어를 연결하고, 의미 있는 이름으로!
[코딩 습관] 좋은 변수 이름 짓기
나쁜 이름: a, x, data1 — 뭘 담는 건지 모르겠어요

좋은 이름: student_name, total_score, max_count

변수 이름만 봐도 무슨 값이 들어있는지 알 수 있어야 해요.

파이썬 관례: 여러 단어는 snake_case로 (밑줄로 연결)

계산하기 — 산술 연산자

파이썬으로 계산을 해봐요. 더하기, 빼기, 곱하기, 나누기 — 파이썬은 훌륭한 계산기예요!

arithmetic.py
1a = 17
2b = 5
3
4print(f"{a} + {b} = {a + b}") # 더하기: 22
5print(f"{a} - {b} = {a - b}") # 빼기: 12
6print(f"{a} * {b} = {a * b}") # 곱하기: 85
7print(f"{a} / {b} = {a / b}") # 나누기: 3.4 (항상 float!)
8print(f"{a} // {b} = {a // b}") # 몫: 3
9print(f"{a} % {b} = {a % b}") # 나머지: 2
10print(f"{a} ** {b} = {a ** b}") # 거듭제곱: 17^5
실행 결과
17 + 5 = 22 17 - 5 = 12 17 * 5 = 85 17 / 5 = 3.4 17 // 5 = 3 17 % 5 = 2 17 ** 5 = 1419857
[핵심] 파이썬 산술 연산자 7개
+ — 더하기
- — 빼기
* — 곱하기
/ — 나누기 (항상 float 반환!) 17/5 = 3.4
// — 몫 (정수 나누기) 17//5 = 3
% — 나머지 17%5 = 2
** — 거듭제곱 2**10 = 1024
[주의] /와 //의 차이
C언어에서 17 / 5 = 3 (정수끼리 나누면 정수)
파이썬에서 17 / 5 = 3.4 (항상 float!)

파이썬에서 정수 몫을 구하려면 //를 써야 해요.
17 // 5 = 3

나머지 연산(%)은 꼭 알아둬야 해요. 짝수/홀수 판별, 배수 확인에 많이 써요.

modulo.py
1# 나머지 연산 활용
2print(10 % 2) # 0 → 짝수! (2로 나눈 나머지가 0)
3print(7 % 2) # 1 → 홀수! (2로 나눈 나머지가 1)
4print(15 % 3) # 0 → 3의 배수!
5print(15 % 4) # 3 → 3의 배수 아님
실행 결과
0 1 0 3
나머지가 0이면 나누어 떨어진다는 뜻! n % 2 == 0이면 짝수, n % 3 == 0이면 3의 배수예요. 조건문 배우면 이걸로 판별 프로그램을 만들 수 있어요!

[심화] 변수는 상자가 아니라 이름표

사실 '변수 = 상자'라는 비유는 초보자용 설명이에요. 정확하게 말하면, 파이썬의 변수는 값에 붙이는 이름표(라벨)예요.

[심화] 변수 = 이름표 (Name Binding)
C언어에서 변수는 진짜 '상자'예요 — 메모리에 공간을 확보하고 값을 넣어요.

파이썬에서 변수는 '이름표'예요 — 값(객체)이 먼저 존재하고, 변수는 그 값을 가리키는 이름이에요.

a = 42 → 42라는 객체를 만들고, a라는 이름표를 붙임
b = a → 같은 42에 b라는 이름표를 하나 더 붙임
a = 99 → a 이름표를 떼서 99에 붙임 (b는 여전히 42)
id_demo.py
1a = 42
2b = a
3print(id(a)) # a가 가리키는 객체의 주소
4print(id(b)) # b도 같은 주소! (같은 객체를 가리킴)
5
6a = 99
7print(id(a)) # a는 이제 다른 주소
8print(id(b)) # b는 여전히 42의 주소
id() 함수는 객체의 메모리 주소를 알려줘요. C언어의 포인터(&변수)와 비슷한 개념이에요. 지금 당장은 '아, 파이썬 변수는 이름표구나' 정도만 알면 돼요.
id()
파이썬에서 모든 값은 '객체(object)'예요. id()는 그 객체의 고유 번호(메모리 주소)를 알려줘요. C에서 &변수와 비슷해요.

[핵심] None의 용도
None은 '아직 값이 정해지지 않았다'는 뜻이에요.

- 함수가 아무것도 반환하지 않을 때 → None
- 변수를 선언만 하고 싶을 때 → x = None
- C언어의 NULL과 비슷해요

del은 변수를 삭제하는 명령이에요. 삭제된 변수를 사용하면 NameError가 나요. 하지만 실제로 del을 쓰는 경우는 많지 않아요.

용어 정리

변수(variable)
값을 저장하는 이름. '이름 = 값' 형태로 만들어요.
대입(assignment)
= 기호로 값을 변수에 저장하는 것. 수학의 '같다'와 다른 의미예요.
int
정수형. 소수점 없는 숫자 (0, 1, -5, 1000)
float
실수형. 소수점 있는 숫자 (3.14, -0.5, 100.0)
str
문자열형. 따옴표로 감싼 글자 데이터 ("Hello", '파이썬')
input()
사용자에게 값을 입력받는 함수. 항상 문자열을 반환해요.
type()
값의 데이터 타입(종류)을 알려주는 함수.
f-string
f"...{변수}..." 형태로 문자열 안에 변수를 넣는 방법. Python 3.6+
snake_case
변수 이름에서 단어를 밑줄로 연결하는 방식. 파이썬의 표준 관례.
PEP 8
파이썬 공식 코딩 스타일 가이드. 들여쓰기, 이름 규칙 등을 정해요.
