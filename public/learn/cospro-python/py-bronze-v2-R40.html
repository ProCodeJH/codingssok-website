for 반복문이란

for 반복문은 정해진 횟수만큼 코드를 반복 실행합니다. '같은 일을 여러 번' 해야 할 때 사용합니다. 파이썬의 for문은 range() 함수와 함께 쓰는 경우가 가장 많습니다.

Pythonfor_basic.py
1for i in range(3):
2 print("안녕하세요")
3 print("반복 번호:", i)
실행 결과
안녕하세요 반복 번호: 0 안녕하세요 반복 번호: 1 안녕하세요 반복 번호: 2
줄코드역할
1for i in range(3):i가 0, 1, 2 순서로 바뀌며 아래 블록을 3번 실행합니다.
2print("안녕하세요")들여쓰기된 코드는 반복 블록에 포함됩니다.
3print("반복 번호:", i)현재 i 값을 함께 출력합니다.
주의 range(3)은 0, 1, 2를 만듭니다. 1, 2, 3이 아닙니다. range(n)은 항상 0부터 시작하여 n-1까지입니다. 이것은 시험에서 가장 자주 출제되는 함정입니다.

중첩 for문 핵심 바깥 for문이 1번 돌 때, 안쪽 for문은 처음부터 끝까지 전부 돕니다.

무한루프와 break

while True:는 조건이 항상 참이므로 무한히 반복합니다. break문을 사용하여 반복을 중단합니다. '사용자가 특정 값을 입력할 때까지 반복'하는 프로그램에서 자주 사용합니다.

Pythonbreak_example.py
1while True:
2 cmd = input("명령 입력 (q=종료): ")
3 if cmd == "q":
4 print("프로그램을 종료합니다.")
5 break
6 print(f"입력: {cmd}")
줄코드역할
1while True:무한 반복을 시작합니다.
3if cmd == "q":종료 조건을 검사합니다.
5break반복문을 즉시 빠져나갑니다.
break의 동작 break는 자신이 속한 가장 가까운 반복문 하나만 종료합니다.
중첩 for문에서 안쪽 break는 안쪽 for만 종료하고, 바깥 for는 계속 돕니다.
이것은 시험에서 자주 출제되는 포인트입니다.

리스트 슬라이싱

슬라이싱은 리스트의 일부분을 잘라내는 기능입니다. range()와 마찬가지로 시작은 포함, 끝은 미포함입니다.

Pythonlist_slice.py
1nums = [10, 20, 30, 40, 50]
2
3print(nums[1:4]) # [20, 30, 40]
4print(nums[:3]) # [10, 20, 30]
5print(nums[2:]) # [30, 40, 50]
6print(nums[::2]) # [10, 30, 50]
7print(nums[::-1]) # [50, 40, 30, 20, 10]
슬라이싱의미결과
nums[1:4]인덱스 1~3[20, 30, 40]
nums[:3]처음~인덱스 2[10, 20, 30]
nums[2:]인덱스 2~끝[30, 40, 50]
nums[::2]처음~끝, 2칸 간격[10, 30, 50]
nums[::-1]역순 전체[50, 40, 30, 20, 10]
슬라이싱 공식 list[시작:끝:간격]
시작 생략 → 처음부터
끝 생략 → 마지막까지
간격 생략 → 1
간격이 -1이면 역순

리스트 수정과 메서드

리스트는 생성 후에도 값을 바꾸거나, 추가하거나, 삭제할 수 있습니다. 이를 '가변(mutable)'이라 합니다.

Pythonlist_methods.py
1nums = [3, 1, 4]
2
3nums.append(5) # 끝에 추가: [3, 1, 4, 5]
4nums.insert(1, 9) # 인덱스 1에 삽입: [3, 9, 1, 4, 5]
5nums.remove(1) # 값 1 제거: [3, 9, 4, 5]
6popped = nums.pop() # 마지막 꺼내기: popped=5, nums=[3, 9, 4]
7nums.sort() # 오름차순 정렬: [3, 4, 9]
8nums.reverse() # 뒤집기: [9, 4, 3]
9print(nums)
실행 결과
[9, 4, 3]
메서드동작반환값
append(x)끝에 x 추가None
insert(i, x)인덱스 i에 x 삽입None
remove(x)처음 나오는 x 제거None
pop()마지막 요소 꺼내기꺼낸 값
pop(i)인덱스 i 요소 꺼내기꺼낸 값
sort()오름차순 정렬None
reverse()순서 뒤집기None
index(x)x의 인덱스 반환인덱스
count(x)x의 개수 반환개수
반환값 함정 append(), sort(), reverse()는 리스트를 직접 수정하고 None을 반환합니다.
nums = nums.sort()라고 쓰면 nums가 None이 됩니다!

1. for 반복문 for i in range(n): → 0부터 n-1까지 n번 반복
range(a, b): → a부터 b-1까지
range(a, b, step): → a부터 b-1까지 step 간격

누적 패턴: total = 0 → for문 안에서 total += i
중첩 for: 바깥 1회 × 안쪽 전체 = 총 실행 횟수
2. while 반복문 while 조건: → 조건이 True인 동안 반복
while True: + break → 무한루프 + 탈출
continue → 이번 회차만 건너뛰기

반드시 종료 조건에 다가가는 코드가 있어야 합니다.
3. 리스트 생성: nums = [1, 2, 3]
인덱싱: nums[0], nums[-1]
슬라이싱: nums[1:4], nums[::-1]
메서드: append, insert, remove, pop, sort, reverse
내장함수: len, sum, max, min, sorted

주의: sort()는 None 반환, sorted()는 새 리스트 반환
4. 문자열 인덱싱/슬라이싱: 리스트와 동일 규칙
불변: s[0] = 'x' 불가 → s = 'x' + s[1:]로 새 문자열
메서드: upper, lower, strip, split, join, find, replace, count
f-string: f"{변수:.2f}" 형식 포매팅
in 연산자: "abc" in s → True/False
5. 함수 def 함수명(매개변수):
return 반환값

return이 없으면 None 반환
return 만나면 함수 즉시 종료
지역 변수 ≠ 전역 변수 (이름이 같아도 별개)
기본값 매개변수: def f(a, b=10):
6. 딕셔너리 생성: d = {"키": 값}
접근: d["키"], d.get("키", 기본값)
메서드: keys, values, items, pop, update
for문: for k, v in d.items():
카운팅: counts[ch] = counts.get(ch, 0) + 1
