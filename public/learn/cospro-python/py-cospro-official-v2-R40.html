정답 및 해설

실전 팁 실제 COS PRO 시험에서는 오류 수정 문제가 2~3문항, 빈칸 채우기가 4~5문항, 코드 완성이 2~3문항 출제됩니다. 오류 수정은 부등호, off-by-one, 조건 논리 오류가 자주 나옵니다. 빈칸 채우기는 배열 인덱싱과 조건문을 정확히 이해해야 합니다.
난이도 분석 공식 샘플 기준: 문제 1~3(오류 수정)은 코드를 읽고 로직을 따라가는 능력이 핵심입니다. 문제 4~8(빈칸 채우기)은 배열 인덱싱, 2D 격자, 재귀를 다룹니다. 문제 9~10(코드 완성)은 알고리즘을 직접 구현해야 하며, 특히 문제 10(TSP 백트래킹)은 2급 최고 난이도입니다.
