변수란 무엇인가

변수(variable)는 값을 저장하는 이름표입니다. 편의점에서 음료수 가격을 기억해 두는 것과 같습니다. 파이썬에서는 = 기호로 값을 변수에 넣습니다.

Pythonvariable.py
1price = 1500
2quantity = 3
3total = price * quantity
4print(total)
실행 결과
4500
줄코드역할
1price = 1500변수 price에 정수 1500을 저장합니다.
2quantity = 3변수 quantity에 정수 3을 저장합니다.
3total = price * quantity두 변수를 곱한 결과를 total에 저장합니다.
4print(total)total의 값을 화면에 출력합니다.
COS PRO 포인트 COS PRO 2급에서 변수 문제는 매 회 출제됩니다. 변수 이름 규칙, 값 대입, 재대입을 정확히 이해해야 합니다.

변수 이름 규칙

변수 이름을 지을 때는 규칙이 있습니다. 규칙을 어기면 프로그램이 실행되지 않습니다.

규칙올바른 예잘못된 예
영문자, 숫자, _ 사용 가능score_1, my_namescore-1, my name
숫자로 시작할 수 없음item2, _count2item, 3rd
예약어 사용 불가my_list, is_validlist, for, if
대소문자 구분Name ≠ name—

네 가지 기본 자료형

파이썬의 값에는 종류가 있습니다. 이것을 자료형(type)이라 합니다. COS PRO 2급에서 반드시 알아야 할 네 가지입니다.

자료형의미예시
int정수 (소수점 없는 수)10, -3, 0
float실수 (소수점 있는 수)3.14, -0.5, 2.0
str문자열 (글자들)"hello", '파이썬'
bool참/거짓True, False
Pythontypes.py
1a = 10
2b = 3.14
3c = "hello"
4d = True
5
6print(type(a)) # <class 'int'>
7print(type(b)) # <class 'float'>
8print(type(c)) # <class 'str'>
9print(type(d)) # <class 'bool'>
TIP type() 함수에 변수를 넣으면 그 변수의 자료형을 알려줍니다. COS PRO 시험에서 type() 결과를 묻는 빈칸 문제가 자주 출제됩니다.

형변환 — 자료형 바꾸기

서로 다른 자료형 사이에서 변환이 필요할 때가 있습니다. 특히 input()은 항상 문자열을 반환하므로, 숫자 계산을 하려면 int()나 float()로 변환해야 합니다.
