파이썬에서 화면에 무언가를 보여주려면 print() 함수를 사용합니다. 괄호 안에 넣은 값이 화면에 나타납니다.

Pythonprint_basic.py

2print(100)
3print(3.14)
4print(True)

줄코드역할
1print("Hello, World!")큰따옴표 안의 문자열을 그대로 출력합니다.

3print(3.14)실수(소수점이 있는 수)를 출력합니다.
4print(True)불(bool) 값 True를 출력합니다.
핵심 따옴표가 있으면 문자열, 없으면 숫자(또는 변수)입니다.

화면 출력은 같아 보이지만 내부적으로 완전히 다릅니다.

% 연산자 활용

나머지 연산자 %는 짝수/홀수 판별, 배수 확인 등에 핵심적으로 사용됩니다.

Pythonmodulo.py
1# 짝수/홀수 판별

이 패턴은 시험과 대회에서 가장 자주 쓰이는 기본 도구입니다.

연산자 우선순위

수학과 마찬가지로 곱셈·나눗셈이 덧셈·뺄셈보다 먼저 계산됩니다. 괄호로 순서를 바꿀 수 있습니다.

우선순위연산자예시

2*, /, //, %10 + 3 * 2 → 16
3 (가장 낮음)+, -10 - 3 + 2 → 9
괄호가 최우선()(10 + 3) * 2 → 26
Pythonpriority.py

if score >= 60:
grade = "D"
elif score >= 70:
grade = "C"
elif score >= 80:
grade = "B"
elif score >= 90:
grade = "A"
else:
grade = "F"
