AI란 무엇일까?

인공지능(AI, Artificial Intelligence)은 '컴퓨터가 사람처럼 똑똑하게 행동하도록 만드는 기술'이에요. 하지만 영화 속 AI와 현실의 AI는 많이 달라요.

영화에서 보는 AI는 사람과 대화하고, 감정을 느끼고, 스스로 판단해요. 예를 들어 아이언맨의 자비스나 터미네이터의 스카이넷처럼요. 하지만 현실의 AI는 특정한 일 하나를 아주 잘하도록 만들어진 프로그램이에요.

탐색 알고리즘 -- AI의 기초

AI의 가장 기본적인 능력은 탐색이에요. '여러 가능성 중에서 답을 찾는 것'이죠. 미로 찾기를 예로 들어 볼게요.

미로에서 출구를 찾으려면 어떻게 해야 할까요? 크게 두 가지 방법이 있어요.

[핵심] BFS -- 너비 우선 탐색 BFS(Breadth-First Search)는 '가까운 곳부터 차례로' 탐색하는 방식이에요.

비유: 연못에 돌을 던지면 물결이 동심원으로 퍼져나가죠?
BFS도 출발점에서 한 칸씩 퍼져나가며 탐색해요.

특징:
- 최단 경로를 보장해요 (가장 짧은 길을 찾아요)
- 메모리를 많이 사용해요 (방문할 곳을 다 기억해야 해요)
- 큐(Queue) 자료구조를 사용해요 (선입선출, 줄 서기)
[핵심] DFS -- 깊이 우선 탐색 DFS(Depth-First Search)는 '한 방향으로 끝까지 가보고, 막히면 되돌아오는' 방식이에요.

비유: 미로에서 오른손을 벽에 대고 계속 걸어가는 것과 비슷해요.
막다른 길이 나오면 되돌아와서 다른 길을 가요.

특징:
- 최단 경로를 보장하지 않아요
- 메모리를 적게 사용해요
- 스택(Stack) 자료구조를 사용해요 (후입선출, 접시 쌓기)

간단한 미로로 BFS와 DFS가 어떻게 다르게 탐색하는지 볼까요? 아래와 같은 미로가 있다고 해요.

머신러닝 기초 개념

머신러닝을 이해하려면 몇 가지 핵심 용어를 알아야 해요.

[핵심] 머신러닝 핵심 용어 학습 데이터(Training Data): 컴퓨터에게 보여주는 예시들이에요.

특성(Feature): 데이터의 특징이에요. 예: 과일의 무게, 색깔, 크기

레이블(Label): 정답이에요. 예: '이 과일은 사과야'

모델(Model): 컴퓨터가 데이터에서 배운 규칙이에요.

예측(Prediction): 새로운 데이터가 들어오면, 모델이 답을 추측하는 거예요.

시험 공부에 비유하면 이해하기 쉬워요.

[비유] 머신러닝 = 시험 공부 학습 데이터 = 문제집의 문제들
특성(Feature) = 문제에 나오는 조건들
레이블(Label) = 문제집 뒤의 정답
학습(Training) = 문제를 풀어보면서 패턴을 익히는 것

지도 학습 vs 비지도 학습 vs 강화 학습

머신러닝에는 세 가지 주요 학습 방식이 있어요. 각각 어떤 상황에서 쓰이는지 알아봐요.

[핵심] 세 가지 학습 방식 1. 지도 학습 (Supervised Learning)
- 선생님이 정답을 알려주면서 가르쳐요
- 데이터에 레이블(정답)이 붙어 있어요
- 예: '이 사진은 고양이야' → 고양이 인식 학습

2. 비지도 학습 (Unsupervised Learning)
- 정답 없이, 데이터 속 패턴을 스스로 찾아요
- 비슷한 것끼리 그룹 만들기(군집화)
- 예: 고객 구매 패턴으로 그룹 나누기

3. 강화 학습 (Reinforcement Learning)
- 시행착오로 배워요. 잘하면 보상, 못하면 벌점
- 예: 게임 AI가 이기면 점수 +, 지면 점수 -

일상생활에 비유하면 이렇게 달라요.

[비유] 세 가지 학습 방식 지도 학습 = 선생님이 채점해 주는 문제집
'이건 O, 이건 X' 하면서 답을 알려줘요

비지도 학습 = 정답 없이 스스로 분류하기
단추 상자에서 '비슷한 것끼리 모아봐' 하는 것과 같아요
색깔별, 크기별, 모양별... 어떻게 모을지는 스스로 결정해요

강화 학습 = 게임하면서 배우기
자전거 타기를 생각해 봐요. 넘어지면 '이러면 안 되는구나',
균형을 잡으면 '이렇게 하면 되는구나' 하고 스스로 배우죠?

분류(Classification) 맛보기 -- KNN

가장 이해하기 쉬운 머신러닝 알고리즘 중 하나가 KNN(K-Nearest Neighbors, K-최근접 이웃)이에요.

[핵심] KNN 알고리즘 KNN은 '새로운 데이터가 들어오면, 가장 가까운 K개의 이웃을 보고 결정'하는 방식이에요.

비유: 전학생이 왔을 때, 주변에 앉은 친구들이 대부분 축구부면
'아, 이 학생도 축구를 좋아할 수 있겠다'고 추측하는 것과 비슷해요.

K = 이웃의 수 (보통 3, 5, 7 같은 홀수)
거리 = 두 데이터가 얼마나 비슷한지 (유클리드 거리)

거리를 계산하는 방법을 알아볼까요? 2차원 좌표에서 두 점 사이의 거리는 유클리드 거리 공식을 써요.

[핵심] 유클리드 거리 두 점 (x1, y1)과 (x2, y2) 사이의 거리:

d = sqrt((x2-x1)^2 + (y2-y1)^2)

이것은 피타고라스 정리와 같아요!
가로 차이의 제곱 + 세로 차이의 제곱을 더한 후 제곱근을 구하면 돼요.

신경망(Neural Network) 개념

딥러닝의 기본 단위는 인공 뉴런이에요. 사람의 뇌에 있는 뉴런(신경세포)을 흉내 낸 거예요.

[핵심] 인공 뉴런의 작동 원리 인공 뉴런은 이렇게 작동해요:

1. 입력값을 받아요 (x1, x2, x3, ...)
2. 각 입력에 가중치를 곱해요 (x1*w1, x2*w2, x3*w3, ...)
3. 곱한 값을 모두 더해요 (합 = x1*w1 + x2*w2 + ...)
4. 편향(bias)을 더해요 (합 + b)
5. 활성화 함수를 통과시켜요 → 최종 출력

수식: output = f(x1*w1 + x2*w2 + ... + b)

가중치(w)가 크면 → 그 입력이 중요하다는 뜻이에요
학습 = 가중치와 편향을 조절하는 과정이에요

이런 뉴런을 여러 개, 여러 층으로 쌓으면 딥러닝이 돼요. 층이 많을수록 더 복잡한 패턴을 배울 수 있어요.

[비유] 신경망의 층 입력층: 데이터가 들어오는 곳 (눈, 귀 같은 감각기관)

C 코드 -- KNN 분류기

두 번째 프로그램으로 KNN 분류기를 C로 만들어 봐요. 2D 좌표의 점을 빨강(0) 또는 파랑(1)으로 분류해요.

CKNN 분류기 (2D 점 분류)
1#include <stdio.h>
2#include <math.h>
3
4#define MAX_DATA 20
5#define K 3
6
7typedef struct {
8 double x, y;
9 int label; // 0=빨강, 1=파랑
10} Point;
11
12// 유클리드 거리 계산
13double distance(Point a, Point b) {
14 double dx = a.x - b.x;
15 double dy = a.y - b.y;
16 return sqrt(dx * dx + dy * dy);
17}
18
19// KNN 분류
20int knn_classify(Point data[], int n, Point query) {
21 // 거리와 인덱스 저장
22 double dists[MAX_DATA];
23 int indices[MAX_DATA];
24
25 for (int i = 0; i < n; i++) {
26 dists[i] = distance(data[i], query);
27 indices[i] = i;
28 }
29
30 // 거리 기준 정렬 (선택 정렬)
31 for (int i = 0; i < n - 1; i++) {
32 for (int j = i + 1; j < n; j++) {
33 if (dists[j] < dists[i]) {
34 double td = dists[i];
35 dists[i] = dists[j];
36 dists[j] = td;
37 int ti = indices[i];
38 indices[i] = indices[j];
39 indices[j] = ti;
40 }
41 }
42 }
43
44 // 가장 가까운 K개의 투표
45 int votes[2] = {0};
46 printf("가장 가까운 %d개:\n", K);
47 for (int i = 0; i < K; i++) {
48 int idx = indices[i];
49 printf(" (%.1f, %.1f) = %s (거리: %.2f)\n",
50 data[idx].x, data[idx].y,
51 data[idx].label == 0 ? "빨강" : "파랑",
52 dists[i]);
53 votes[data[idx].label]++;
54 }
55
56 return (votes[1] > votes[0]) ? 1 : 0;
57}
58
59int main(void) {
60 Point data[] = {
61 {1, 1, 0}, {2, 1, 0}, {1, 2, 0},
62 {2, 2, 0}, {3, 1, 0},
63 {6, 6, 1}, {7, 7, 1}, {6, 7, 1},
64 {7, 6, 1}, {8, 7, 1}
65 };
66 int n = 10;
67
68 // 새로운 점 분류
69 Point query = {4, 4, -1};
70 printf("새 점: (%.1f, %.1f)\n\n", query.x, query.y);
71
72 int result = knn_classify(data, n, query);
73 printf("\n예측: %s\n",
74 result == 0 ? "빨강" : "파랑");
75
76 return 0;
77}

C 코드 -- 단일 퍼셉트론 (AND 게이트 학습)

세 번째 프로그램으로 퍼셉트론이 AND 게이트를 스스로 배우는 과정을 구현해요.

C퍼셉트론 AND 게이트 학습
1#include <stdio.h>
2
3// 활성화 함수 (계단 함수)
4int activate(double sum) {
5 return (sum >= 0) ? 1 : 0;
6}
7
8int main(void) {
9 // AND 게이트 학습 데이터
10 double inputs[][2] = {{0,0}, {0,1}, {1,0}, {1,1}};
11 int targets[] = {0, 0, 0, 1};
12 int n = 4;
13
14 // 가중치와 편향 초기화
15 double w1 = 0.0, w2 = 0.0, b = 0.0;
16 double lr = 0.1; // 학습률
17
18 printf("=== 퍼셉트론 AND 학습 ===\n\n");
19
20 for (int epoch = 0; epoch < 20; epoch++) {
21 int errors = 0;
22
23 for (int i = 0; i < n; i++) {
24 double x1 = inputs[i][0];
25 double x2 = inputs[i][1];
26
27 // 순전파
28 double sum = x1 * w1 + x2 * w2 + b;
29 int pred = activate(sum);
30
31 // 오차 계산
32 int error = targets[i] - pred;
33
34 if (error != 0) {
35 errors++;
36 // 가중치 업데이트
37 w1 += lr * error * x1;
38 w2 += lr * error * x2;
39 b += lr * error;
40 }
41 }
42
43 printf("에폭 %2d: w1=%.2f, w2=%.2f, b=%.2f",
44 epoch + 1, w1, w2, b);
45 printf(" (오류: %d)\n", errors);
46
47 if (errors == 0) {
48 printf("\n학습 완료!\n");
49 break;
50 }
51 }
52
53 // 결과 확인
54 printf("\n=== 학습 결과 ===\n");
55 for (int i = 0; i < n; i++) {
56 double x1 = inputs[i][0];
57 double x2 = inputs[i][1];
58 double sum = x1 * w1 + x2 * w2 + b;
59 int pred = activate(sum);
60 printf("%d AND %d = %d (정답: %d) %s\n",
61 (int)x1, (int)x2, pred, targets[i],
62 pred == targets[i] ? "O" : "X");
63 }
64
65 return 0;
66}

C 코드 -- 간단한 미로 탐색 (BFS)

네 번째 프로그램으로 BFS를 이용한 미로 탐색을 구현해요. 최단 경로를 찾아줘요.

CBFS 미로 탐색
1#include <stdio.h>
2
3#define ROWS 5
4#define COLS 5
5#define MAX_Q 100
6
7// 미로: 0=통로, 1=벽, 2=시작, 3=도착
8int maze[ROWS][COLS] = {
9 {2, 0, 1, 0, 0},
10 {1, 0, 1, 0, 1},
11 {0, 0, 0, 0, 0},
12 {0, 1, 1, 1, 0},
13 {0, 0, 0, 0, 3}
14};
15
16// 방향: 상, 하, 좌, 우
17int dr[] = {-1, 1, 0, 0};
18int dc[] = {0, 0, -1, 1};
19
20int visited[ROWS][COLS];
21int dist[ROWS][COLS];
22
23typedef struct {
24 int r, c;
25} Pos;
26
27Pos queue[MAX_Q];
28int front = 0, rear = 0;
29
30void enqueue(Pos p) { queue[rear++] = p; }
31Pos dequeue(void) { return queue[front++]; }
32int is_empty(void) { return front >= rear; }
33
34int main(void) {
35 // 시작점 찾기
36 int sr = 0, sc = 0, er = 0, ec = 0;
37 for (int r = 0; r < ROWS; r++) {
38 for (int c = 0; c < COLS; c++) {
39 if (maze[r][c] == 2) { sr = r; sc = c; }
40 if (maze[r][c] == 3) { er = r; ec = c; }
41 visited[r][c] = 0;
42 dist[r][c] = -1;
43 }
44 }
45
46 // BFS 시작
47 Pos start = {sr, sc};
48 enqueue(start);
49 visited[sr][sc] = 1;
50 dist[sr][sc] = 0;
51
52 printf("BFS 미로 탐색\n\n");
53
54 while (!is_empty()) {
55 Pos cur = dequeue();
56
57 if (cur.r == er && cur.c == ec) {
58 printf("도착! 최단 거리: %d\n\n",
59 dist[er][ec]);
60 break;
61 }
62
63 for (int d = 0; d < 4; d++) {
64 int nr = cur.r + dr[d];
65 int nc = cur.c + dc[d];
66
67 if (nr >= 0 && nr < ROWS &&
68 nc >= 0 && nc < COLS &&
69 !visited[nr][nc] &&
70 maze[nr][nc] != 1) {
71 visited[nr][nc] = 1;
72 dist[nr][nc] = dist[cur.r][cur.c] + 1;
73 Pos next = {nr, nc};
74 enqueue(next);
75 }
76 }
77 }
78
79 // 거리 출력
80 printf("거리 맵:\n");
81 for (int r = 0; r < ROWS; r++) {
82 for (int c = 0; c < COLS; c++) {
83 if (maze[r][c] == 1)
84 printf(" ## ");
85 else if (dist[r][c] == -1)
86 printf(" -- ");
87 else
88 printf(" %2d ", dist[r][c]);
89 }
90 printf("\n");
91 }
92
93 return 0;
94}

디버깅 연습 -- AI 코드의 흔한 실수

AI 프로그램을 짤 때 자주 하는 실수들을 찾아봐요.

[디버깅 1] 배열 인덱스 오류 -- KNN의 거리 정렬 버그 있는 코드:
// K개 이웃 투표
int votes[2] = {0};
for (int i = 0; i <= K; i++) {  // 버그!
    votes[data[indices[i]].label]++;
}

문제: i <= K 이면 K+1개를 참조해요! 0부터 K까지는 K+1개예요.

수정: i < K로 바꿔야 해요.
for (int i = 0; i < K; i++) {  // 수정!
    votes[data[indices[i]].label]++;
}

배열 범위 실수는 C 프로그래밍에서 가장 흔한 버그예요!
[디버깅 2] 거리 계산 오류 -- 제곱근 빠뜨리기 버그 있는 코드:
double distance(Point a, Point b) {
    double dx = a.x - b.x;
    double dy = a.y - b.y;
    return dx * dx + dy * dy;  // 버그!
}

문제: sqrt()를 빠뜨렸어요! 이렇게 하면 거리의 제곱이 반환돼요.

영향: KNN에서 순서는 같지만, 거리 값이 실제와 달라요.
사실 KNN에서는 비교만 하니까 결과가 같을 수도 있지만,
거리 기반 가중 투표를 할 때는 결과가 달라져요!

수정:
return sqrt(dx * dx + dy * dy);  // 수정!
[디버깅 3] 학습률 문제 -- 퍼셉트론이 배우지 못해요 버그 있는 코드:
double lr = 10.0;  // 학습률이 너무 커요!

// 업데이트
w1 += lr * error * x1;
w2 += lr * error * x2;
b  += lr * error;

문제: 학습률(lr)이 10.0으로 너무 커요!
가중치가 너무 크게 변해서 이리저리 왔다갔다 해요. (발산)

증상: 에폭을 아무리 반복해도 오류가 0이 되지 않아요.

수정: 학습률을 작게 해요 (보통 0.01~0.1).
double lr = 0.1;  // 적절한 학습률

학습률이 너무 크면 발산하고, 너무 작으면 학습이 느려요!
