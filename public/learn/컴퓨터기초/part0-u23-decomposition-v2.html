UNIT 23 분해 (Decomposition)

문제가 너무 커! 어떡하지?

세상의 모든 큰 문제는 사실 작은 문제들의 모음이에요. 100페이지 공부도, 학교 축제 준비도, 게임 만들기도 — 전부 작은 조각으로 나눌 수 있어요. 이렇게 큰 문제를 작은 조각으로 쪼개는 것을 분해(Decomposition)라고 해요.

[핵심] 분해란? 분해(Decomposition) = 크고 복잡한 문제를 작고 간단한 조각으로 나누는 것

왜 분해할까?
1. 작은 문제는 풀기 쉬워요
2. 한 조각씩 차근차근 해결할 수 있어요
3. 여러 사람이 나눠서 할 수 있어요
4. 버그가 생기면 어디서 틀렸는지 찾기 쉬워요

예시 1: 아침 등교 분해하기

"학교에 가기"는 꽤 큰 일이에요. 하지만 분해하면 이렇게 돼요.

단계할 일걸리는 시간필요한 것
1알람 끄기1분알람 시계
2세수하기5분수건, 치약
3옷 입기5분교복 또는 사복
4밥 먹기15분아침 식사
5가방 챙기기3분교과서, 필통

[심화 내용] 분할정복 (Divide and Conquer)

분해의 알고리즘 버전! 문제를 나누고(Divide), 각각 풀고(Conquer), 합치는(Combine) 전략이에요.

[심화 내용] 분할정복의 3단계 1. Divide: 문제를 2개 이상의 하위 문제로 나눈다
2. Conquer: 각 하위 문제를 재귀적으로 풀거나, 충분히 작으면 직접 푼다
3. Combine: 하위 문제의 해답을 합쳐 원래 문제의 해답을 만든다
Cmerge_sort.c
1// 병합 정렬 — 분할정복의 대표 예시
2#include <stdio.h>
3
4void merge(int arr[], int l, int m, int r) {
5 int i = l, j = m + 1, k = 0;
6 int temp[r - l + 1];
7 while (i <= m && j <= r) {
8 if (arr[i] <= arr[j]) temp[k++] = arr[i++];
9 else temp[k++] = arr[j++];
10 }
11 while (i <= m) temp[k++] = arr[i++];
12 while (j <= r) temp[k++] = arr[j++];
13 for (int p = 0; p < k; p++) arr[l + p] = temp[p];
14}
15
16void mergeSort(int arr[], int l, int r) {
17 if (l >= r) return; // 충분히 작으면 멈춤
18 int m = (l + r) / 2;
19 mergeSort(arr, l, m); // 왼쪽 반 정렬
20 mergeSort(arr, m + 1, r); // 오른쪽 반 정렬
21 merge(arr, l, m, r); // 합치기
22}
23
24int main(void) {
25 int arr[] = {5, 2, 8, 1, 9, 3};
26 int n = 6;
27 mergeSort(arr, 0, n - 1);
28 for (int i = 0; i < n; i++) printf("%d ", arr[i]);
29 printf("\n"); // 1 2 3 5 8 9
30 return 0;
31}

1 2 3 5 8 9

용어 정리

[심화 내용] 분할 정복 알고리즘

[심화 내용] 분할 정복 (Divide and Conquer) 3단계:
1. 분할(Divide): 문제를 작은 부분 문제로 나눔
2. 정복(Conquer): 각 부분 문제를 재귀적으로 해결
3. 결합(Combine): 부분 해를 합쳐서 전체 해 구성

대표 알고리즘:
- Merge Sort: 반으로 나누고, 정렬하고, 합치기 O(n log n)
- Quick Sort: 피벗 기준 분할, 재귀 O(n log n) 평균
- Binary Search: 반으로 나누고, 한쪽만 탐색 O(log n)
- Karatsuba 곱셈: 큰 수 곱셈 O(n^1.585)
- Strassen 행렬 곱: O(n^2.807) vs 일반 O(n^3)

마스터 정리: T(n) = aT(n/b) + O(n^d)
C[C 코드] 분할 정복 — 머지 소트

2
3void merge(int arr[], int l, int m, int r) {
4 int temp[100], i = l, j = m+1, k = 0;
5 while (i <= m && j <= r)
6 temp[k++] = (arr[i] <= arr[j]) ? arr[i++] : arr[j++];
7 while (i <= m) temp[k++] = arr[i++];
8 while (j <= r) temp[k++] = arr[j++];
9 for (int t = 0; t < k; t++) arr[l+t] = temp[t];
10}
11
12void merge_sort(int arr[], int l, int r) {
13 if (l >= r) return;
14 int m = (l + r) / 2;
15 merge_sort(arr, l, m); // 왼쪽 정복
16 merge_sort(arr, m+1, r); // 오른쪽 정복
17 merge(arr, l, m, r); // 결합!
18}
