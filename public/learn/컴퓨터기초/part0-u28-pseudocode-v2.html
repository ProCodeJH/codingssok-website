Cpseudo_binary_search.c
1// 의사코드: 이진 탐색
2// 1. INPUT sorted_array A[0..n-1], target
3// 2. low ← 0, high ← n - 1
4// 3. WHILE low ≤ high DO
5// mid ← (low + high) / 2
6// IF A[mid] = target THEN
7// RETURN mid
8// ELSE IF A[mid] < target THEN
9// low ← mid + 1
10// ELSE
11// high ← mid - 1
12// 4. RETURN -1 (못 찾음)
13
14#include <stdio.h>
15
16int binarySearch(int arr[], int n, int target) {
17 int low = 0, high = n - 1;
18 while (low <= high) {
19 int mid = (low + high) / 2;
20 if (arr[mid] == target)
21 return mid;
22 else if (arr[mid] < target)
23 low = mid + 1;
24 else
25 high = mid - 1;
26 }
27 return -1;
28}
29

[심화] 의사코드로 버블 정렬 설계

[심화] 버블 정렬 의사코드 PROCEDURE BubbleSort(A, n)
FOR i ← 0 TO n-2 DO
FOR j ← 0 TO n-2-i DO
IF A[j] > A[j+1] THEN
SWAP A[j], A[j+1]
RETURN A

분석:
• 바깥 루프: n-1번
• 안쪽 루프: 최대 n-1번
• 총 비교: O(n²)

[심화 내용] 의사코드로 복잡도 분석

[심화 내용] 의사코드에서 Big-O 추론 의사코드의 루프 구조를 보면 시간 복잡도를 바로 알 수 있어요!

단일 루프:
FOR i ← 1 TO n → O(n)

중첩 루프:
FOR i ← 1 TO n
FOR j ← 1 TO n → O(n²)

반감 루프:
WHILE n > 0
n ← n / 2 → O(log n)

중첩 + 반감:
FOR i ← 1 TO n
j ← n
WHILE j > 0: j ← j/2 → O(n log n)

[심화 내용] 의사코드와 정형 검증

[심화 내용] 루프 불변식 (Loop Invariant) 루프 불변식 = 반복문 실행 전후에 항상 참인 조건

알고리즘의 정확성을 증명하는 핵심 도구!

예: 선형 탐색의 루프 불변식
SEARCH(A, n, x):
for i = 0 to n-1
if A[i] == x return i
return -1

불변식: "A[0..i-1]에 x가 없다"

초기: i=0, 검사 구간 없음 → 참
유지: A[i]!=x이면 A[0..i]에 x 없음 → 참
종료: i=n, A[0..n-1]에 x 없음 → -1 반환 올바름!
