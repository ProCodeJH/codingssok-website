스택 — 접시 쌓기!

스택(Stack)은 데이터를 쌓아 올리는 구조예요. 마지막에 넣은 것을 가장 먼저 꺼내요. 이걸 LIFO(Last In, First Out)라고 해요.

[핵심] 스택의 두 가지 연산 push(값) = 위에 쌓기 (넣기)
접시 위에 새 접시를 올려요.

pop() = 위에서 꺼내기 (빼기)
맨 위의 접시를 가져가요.

LIFO = Last In, First Out
마지막에 넣은 것이 먼저 나온다!

스택에서는 중간이나 바닥의 데이터를 바로 꺼낼 수 없어요!

큐(Queue)는 줄 서기와 같은 구조예요. 먼저 온 사람이 먼저 나가요. 이걸 FIFO(First In, First Out)라고 해요.

[심화] 스택으로 DFS, 큐로 BFS

[심화] 탐색 알고리즘과 자료구조 매칭 DFS (깊이 우선 탐색):
- 스택 사용 (또는 재귀 = 콜스택)
- 한 방향으로 깊이 파고듦
- 미로에서: 막히면 되돌아가기
- 활용: 연결 요소, 위상정렬, 사이클 검출

BFS (너비 우선 탐색):
- 큐 사용
- 가까운 것부터 탐색 (원 모양 확산)
- 미로에서: 최단 거리 보장!
- 활용: 최단경로, 레벨순 탐색

요약:
스택 → DFS → 깊이 → 되돌아가기
큐 → BFS → 너비 → 최단거리

자료구조만 바꾸면 DFS↔BFS 전환!
[심화 내용] 면접 문제 Q1. 두 스택으로 큐 구현
A: inStack에 push, dequeue 시 outStack이 비면 inStack→outStack으로 이동 후 pop. amortized O(1)!

Q2. 히스토그램 최대 직사각형
A: 모노톤 스택으로 O(n)! 높이가 줄어들면 스택에서 pop하며 넓이 계산.

Q3. 스택 min() O(1)로 구현
A: 메인 스택 + 최소값 스택 병행. push/pop 시 min 스택도 갱신.

Q4. 큐 max() O(1)로 구현
A: 덱(모노톤 큐) 사용. 새 원소보다 작은 건 뒤에서 제거.

[심화] 스택 응용: 중위→후위 변환 (Shunting-yard)

[심화] 다익스트라의 Shunting-yard 알고리즘 중위 표기 → 후위 표기 변환 알고리즘

규칙:
1. 숫자 → 출력
2. ( → 스택에 push
3. ) → ( 나올 때까지 pop하여 출력
4. 연산자:
스택 top의 우선순위 >= 현재 → pop하여 출력
현재 연산자 push
5. 끝나면 스택 전부 pop

예: 3 + 4 * 2
3 → 출력: 3
+ → push + [+]
4 → 출력: 3 4
* → * > + 이므로 push [+, *]
2 → 출력: 3 4 2
끝 → pop: 3 4 2 * +

컴파일러가 수식을 처리하는 핵심 알고리즘!
C[C 코드] 중위→후위 변환
1#include <stdio.h>
2#include <ctype.h>
3
4char ops[100]; int ot = -1;
5void opush(char c) { ops[++ot] = c; }
6char opop() { return ops[ot--]; }
7int prec(char c) { return (c=='+'||c=='-') ? 1 : (c=='*'||c=='/') ? 2 : 0; }
8
9void infix_to_postfix(const char *s) {
10 for (int i = 0; s[i]; i++) {
11 if (isdigit(s[i])) printf("%c ", s[i]);
12 else if (s[i] == '(') opush('(');
13 else if (s[i] == ')') {
14 while (ot >= 0 && ops[ot] != '(') printf("%c ", opop());
15 if (ot >= 0) opop(); // discard (
16 } else if (s[i]=='+'||s[i]=='-'||s[i]=='*'||s[i]=='/') {
17 while (ot >= 0 && prec(ops[ot]) >= prec(s[i])) printf("%c ", opop());
18 opush(s[i]);
19 }
20 }
21 while (ot >= 0) printf("%c ", opop());
22 printf("\n");
23}
