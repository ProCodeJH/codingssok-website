📖 추상화란?

추상화란 복잡한 것에서 핵심(중요한 것)만 뽑아내고, 불필요한 세부사항은 숨기는 것입니다.

  • 핵심 추출 — 중요한 정보만 골라내기
  • 불필요한 것 숨기기 — 지금 필요없는 것은 빼기
  • 단순화 — 복잡한 것을 간단하게 표현하기
  • 일반화 — 여러 것들의 공통점 찾기
지도가 대표적인 추상화! 실제 세상에서 도로·건물만 남기고 나무·사람은 숨김 → 길 찾기에 필요한 것만!

📌 예제: 추상화 과정

실제 고양이→사진→그림→🐱 아이콘→"고양이" 글자

점점 세부사항을 줄이고 핵심만 남기는 과정 = 추상화!
