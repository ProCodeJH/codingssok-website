Level 2 · 데이터 추상화 — 변수·자료형·구조화·표현 방식

📖 데이터 추상화란?

현실의 정보를 컴퓨터가 다룰 수 있는 데이터로 바꾸는 것이 데이터 추상화입니다.

  • 변수 — 데이터에 이름을 붙여 관리
  • 자료형 — 숫자·문자·참/거짓으로 분류
  • 구조화 — 리스트·표·딕셔너리로 정리
  • 인코딩 — 문자→숫자, 이미지→픽셀로 변환
실제 세상 → 데이터로 바꾸려면 '무엇을 남기고 무엇을 버릴지' 결정해야 합니다!

📌 예제: 학생 정보의 데이터 추상화

실제 학생→이름: 김철수→나이: 12→학년: 6→{name, age, grade}
