프로그래밍이 뭘까?

안녕! 나는 코딩이야. 오늘부터 같이 파이썬을 배울 거야. 긴장하지 마, 처음엔 누구나 어려워!

컴퓨터는 우리 말을 못 알아들어요. 대신 프로그래밍 언어라는 특별한 말로 이야기해야 해요. 오늘은 컴퓨터에게 '화면에 글자를 찍어!'라고 시켜볼 거예요!

프로그래밍 언어는 여러 종류가 있어요. C, Java, JavaScript, 그리고 우리가 배울 파이썬(Python)이 있어요. 파이썬은 그중에서 가장 쉽고 읽기 편한 언어예요.

파이썬은 왜 배울까?

파이썬은 1991년에 귀도 반 로섬(Guido van Rossum)이라는 네덜란드 프로그래머가 만들었어요. '누구나 쉽게 읽을 수 있는 코드'를 목표로 만든 언어예요.

1991
파이썬이 처음 세상에 나온 해예요. 이름은 '몬티 파이썬'이라는 영국 코미디 쇼에서 따왔어요. 뱀이 아니에요!
[핵심] 파이썬을 배우면 좋은 이유
1. 쉬워요 — 영어 문장 읽듯이 코드를 읽을 수 있어요.
2. 많이 써요 — 구글, 인스타그램, 유튜브가 파이썬으로 만들어졌어요.
3. 할 수 있는 게 많아요 — 웹사이트, 게임, 인공지능, 데이터 분석, 자동화까지!
4. 취업에 유리해요 — 전 세계에서 가장 인기 있는 프로그래밍 언어예요.

C언어와 뭐가 다를까?

같은 'Hello, World!'를 출력하는 코드를 비교해 볼까요? C언어는 꽤 길었는데, 파이썬은 놀랍도록 짧아요.

✕ 잘못된 코드
#include int main(void) { printf("Hello, World!\n"); return 0; }
C언어: 5줄, #include와 main이 필요해요
✓ 올바른 코드
print("Hello, World!")
파이썬: 딱 1줄! 이게 전부예요
파이썬은 #include도 없고, main 함수도 없고, 세미콜론도 없어요. 진짜 간단하죠?

한 줄씩 뜯어보기

파이썬의 Hello World는 딱 한 줄이지만, 그 안에 중요한 것들이 꽉 차 있어요.

1줄print — 화면에 글자를 출력하는 함수예요. C의 printf와 비슷하지만 훨씬 간단해요.
1줄("Hello, World!") — 괄호 안에 출력할 내용을 넣어요. 이걸 인자(argument)라고 해요.
1줄"Hello, World!" — 큰따옴표로 감싼 글자를 문자열(string)이라고 해요.
[핵심] print() 함수의 구조
print(출력할 내용)

- 괄호 ( ) 안에 출력할 것을 넣어요
- 문자열은 따옴표로 감싸야 해요
- 실행하면 화면에 출력하고 자동으로 줄을 바꿔요
- C의 printf와 달리 \n을 안 써도 자동 줄바꿈!

파이썬은 C언어와 달리 세미콜론(;)이 필요 없어요. 줄이 끝나면 문장이 끝난 거예요. #include도 필요 없고, main 함수도 필요 없어요. 그냥 바로 코드를 쓰면 돼요.

파이썬에서는 줄 끝에 세미콜론을 안 붙여도 돼요! 붙여도 에러는 안 나지만, 파이썬답지 않다고 할 거예요.
[팁] 작은따옴표 vs 큰따옴표
파이썬에서는 'Hello'와 "Hello"가 같아요!
둘 다 문자열이에요. 편한 걸 쓰면 돼요.

다만 문자열 안에 따옴표가 있을 때 유용해요:
print("I'm a student") — 큰따옴표 안에 작은따옴표 OK
print('He said "Hi"') — 작은따옴표 안에 큰따옴표 OK

2print("Hello, World!")
3print('Hello, World!')
4
5# 따옴표 안에 따옴표
6print("I'm a student")

IDLE에서 실행해 보기

파이썬을 설치하면 IDLE이라는 프로그램이 같이 설치돼요. IDLE은 파이썬 코드를 바로 실행해 볼 수 있는 도구예요.

1
IDLE 열기
시작 메뉴에서 'IDLE'을 검색해서 열어요
2
>>> 프롬프트 확인
>>> 표시가 나오면 준비 완료! 코드를 입력할 수 있어요
3
코드 입력
print("Hello, World!")를 타이핑해요
4
Enter 누르기
바로 결과가 나와요!
IDLE 대화형 모드
1>>> print("Hello, World!")
2Hello, World!
3>>> print("안녕하세요!")
4안녕하세요!
5>>> 1 + 1
62
[핵심] 대화형 모드 vs 스크립트 모드
대화형 모드 (Interactive Mode)
- IDLE의 >>> 프롬프트에서 바로 실행
- 한 줄씩 바로 결과 확인
- 계산기처럼 사용하기 좋아요

스크립트 모드 (Script Mode)
- .py 파일에 여러 줄 코드 작성
- 한 번에 전체 실행
- 진짜 프로그램은 이 방식으로 만들어요
대화형 모드는 연습용, 스크립트 모드는 실전용이에요. 처음엔 대화형으로 연습하고, 나중에 .py 파일로 넘어가면 돼요.

대화형 모드에서는 print()를 안 써도 값이 출력돼요. >>> 1 + 1 하면 바로 2가 나와요. 하지만 .py 파일에서는 print()를 써야만 화면에 나와요.

[주의] 대화형 모드의 함정
대화형 모드에서 1 + 1만 쳐도 2가 나와요.
그래서 '아, print 안 써도 되는구나!' 하고 착각하기 쉬워요.

하지만 .py 파일에서 1 + 1만 쓰면 아무것도 안 나와요!
파일에서는 반드시 print(1 + 1)이라고 써야 해요.

C언어는 컴파일(gcc)을 해야 실행할 수 있었는데, 파이썬은 바로 실행돼요! 이걸 인터프리터 방식이라고 해요.
[핵심] 컴파일 vs 인터프리터
C언어 (컴파일 방식)
코드(.c) → 컴파일러(gcc) → 실행파일(.exe) → 실행

파이썬 (인터프리터 방식)
코드(.py) → 파이썬 인터프리터 → 바로 실행!
한 줄씩 읽으면서 바로 실행해요. 실행은 느리지만 편리해요.
C언어 vs 파이썬 실행 과정
hello.c→gcc (컴파일)→hello.exe→실행!
C언어: 번역 → 실행 (2단계)

hello.py→python이 바로 실행!
파이썬: 바로 실행 (1단계)

이스케이프 시퀀스

문자열 안에서 특별한 의미를 가지는 기호를 이스케이프 시퀀스라고 해요. 역슬래시(\)로 시작해요.

escape.py
1print("줄을\n바꿔요") # \n = 줄 바꿈
2print("탭을\t넣어요") # \t = 탭(칸 띄우기)
3print("역슬래시: \\") # \\ = 역슬래시 자체
4print("따옴표: \"안녕\"") # \" = 큰따옴표 자체
실행 결과
줄을 바꿔요 탭을 넣어요 역슬래시: \ 따옴표: "안녕"
[핵심] 자주 쓰는 이스케이프 시퀀스
\n — 줄 바꿈 (New line)
\t — 탭 (Tab)
\\ — 역슬래시(\) 자체
\" — 큰따옴표(") 자체
\' — 작은따옴표(') 자체

print()의 숨겨진 기능

print()에는 몰랐던 기능이 숨어 있어요. 여러 값을 한 번에 출력하거나, 줄바꿈을 없앨 수 있어요.

여러 값 출력 — 쉼표로 구분

print_multi.py
1# 여러 값을 쉼표로 구분하면 자동으로 띄어쓰기
2print("이름:", "코딩이")
3print("나이:", 12, "살")
4print(1, 2, 3, 4, 5)
실행 결과
이름: 코딩이 나이: 12 살 1 2 3 4 5
쉼표로 여러 값을 넣으면 자동으로 공백 한 칸이 들어가요. C의 printf처럼 포맷 지정자(%d, %s)를 안 써도 돼요!

sep와 end — 구분자와 끝문자 바꾸기

sep_end.py
1# sep: 값 사이 구분자 바꾸기 (기본: 공백)
2print("2025", "03", "10", sep="-")
3print("서울", "부산", "대구", sep=" → ")
4
5# end: 줄 끝 바꾸기 (기본: \n)
6print("Hello", end=" ")
7print("World!") # 줄바꿈 없이 이어서 출력
실행 결과
2025-03-10 서울 → 부산 → 대구 Hello World!

숫자도 출력할 수 있어요

print()에는 문자열뿐만 아니라 숫자도 넣을 수 있어요. 숫자는 따옴표 없이 그냥 써요.

print_number.py
1print(42) # 정수
2print(3.14) # 실수
3print(1 + 1) # 계산 결과
4print(10 * 3) # 곱하기
5print(2 ** 10) # 2의 10제곱 = 1024
6print("1 + 1 =", 1 + 1) # 문자열 + 계산
실행 결과
42 3.14 2 30 1024 1 + 1 = 2
[팁] 따옴표 안의 숫자 vs 밖의 숫자
print(1 + 1) → 2 (계산됨!)
print("1 + 1") → 1 + 1 (그냥 글자로 출력!)

따옴표 안에 넣으면 글자(문자열), 밖에 쓰면 숫자예요.

[심화] 파이썬은 어떻게 동작할까?

파이썬이 코드를 바로 실행한다고 했지만, 실제로는 내부에서 몇 단계를 거쳐요.

[심화] 파이썬 실행 과정
1. 소스 코드 읽기 (.py 파일)

2. 바이트코드 컴파일
파이썬이 내부적으로 .pyc 파일(바이트코드)로 변환
C와 달리 사용자가 직접 하지 않아요

3. PVM (Python Virtual Machine) 실행
바이트코드를 한 줄씩 실행
Java의 JVM과 비슷한 개념이에요

즉, 파이썬도 내부적으로는 '컴파일'을 하지만,
사용자가 모르게 자동으로 해줘요!

용어 정리

print()
화면에 글자를 출력하는 파이썬 내장 함수. 자동으로 줄바꿈을 해줘요.
문자열(string)
따옴표로 감싼 글자 데이터. 'Hello'나 "Hello" 모두 문자열이에요.
인터프리터
코드를 한 줄씩 읽으면서 바로 실행하는 프로그램. 파이썬은 인터프리터 방식이에요.
IDLE
파이썬에 포함된 기본 개발 환경. Integrated Development and Learning Environment의 약자.
이스케이프 시퀀스
\n, \t처럼 역슬래시로 시작하는 특수 문자. 줄바꿈, 탭 등을 표현해요.
주석(comment)
# 뒤에 쓰는 설명 글. 파이썬이 무시하고 사람만 읽어요.
바이트코드
파이썬이 내부적으로 변환하는 중간 코드. .pyc 파일로 저장돼요.
PVM
Python Virtual Machine. 바이트코드를 실행하는 가상 기계.
