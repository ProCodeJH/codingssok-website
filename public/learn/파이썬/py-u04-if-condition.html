Lesson 1-4

if 조건문 — 갈림길에서 선택하기

if, elif, else로 조건에 따라 다른 코드를 실행하는 방법을 배워요.

이번에 배울 것

  • if문으로 조건에 따라 코드를 실행할 수 있어요.
  • if-else로 두 갈래 분기를 만들 수 있어요.
  • if-elif-else로 세 갈래 이상 분기를 만들 수 있어요.
  • 중첩 if문을 사용할 수 있어요.
  • 들여쓰기의 중요성을 이해해요.

시작 전 체크

  • 비교 연산자(==, !=, <, >, <=, >=)를 사용할 수 있다
  • 논리 연산자(and, or, not)를 안다
  • True/False(bool)의 의미를 안다
Part 1 진행률: 8%

if문 — 조건이 맞으면 실행해!

드디어 조건문이야! 지금까지는 코드가 위에서 아래로 순서대로 실행됐는데, 이제 갈림길을 만들 수 있어. '점수가 60점 이상이면 합격'처럼!

프로그래밍에서 가장 중요한 개념 중 하나가 조건문이에요. '만약에 ~ 이면 ~ 해라'를 코드로 쓸 수 있어요.

실생활에서도 조건문은 곳곳에 있어요. '비가 오면 우산을 챙겨', '신호가 초록이면 건너', '배터리가 20% 이하면 충전해'... 컴퓨터도 똑같이 동작해요.

들여쓰기 — 파이썬의 생명줄

C언어는 중괄호 { }로 코드 범위를 정했는데, 파이썬은 들여쓰기로 정해요. 들여쓰기가 틀리면 에러가 나요.

✕ 잘못된 코드
if score >= 60: print("합격!") # IndentationError!
들여쓰기를 안 하면 에러!
✓ 올바른 코드
if score >= 60: print("합격!") # 스페이스 4칸!
반드시 들여쓰기 필요
에러 메시지
IndentationError: expected an indented block after 'if' statement on line 1

파이썬이 'if 다음에 들여쓰기된 블록이 있어야 하는데 없어!'라고 알려줘요.
해결법: if 다음 줄에 스페이스 4칸 들여쓰기 추가
✕ 잘못된 코드
if score >= 60: print("합격!") print("축하해요") # 2칸만 들여쓰기!
들여쓰기 칸 수가 다르면 에러!
✓ 올바른 코드
if score >= 60: print("합격!") print("축하해요") # 4칸 통일!
같은 블록은 같은 들여쓰기
[핵심] 들여쓰기 규칙
1. 스페이스 4칸이 표준 (PEP 8)
2. 같은 블록 안에서 들여쓰기 칸 수를 통일
3. Tab과 스페이스 절대 섞지 않기
4. 들여쓰기를 빼면 if문 블록이 끝난 것
C언어는 중괄호를 빠뜨려도 컴파일은 되는 경우가 있었는데, 파이썬은 들여쓰기가 틀리면 바로 에러예요. 더 엄격하지만 덕분에 코드가 항상 깔끔해요.

elif — 세 갈래 이상 나누기

두 갈래가 아니라 세 갈래, 네 갈래로 나누고 싶을 때는 elif를 써요. else + if를 합친 거예요.

[핵심] if-elif-else 규칙
1. if는 반드시 하나, 맨 위에
2. elif는 0개 이상, 원하는 만큼
3. else는 0개 또는 1개, 맨 아래에
4. 위에서부터 순서대로 확인, 처음 True인 것 하나만 실행
5. 하나도 True가 없으면 else 실행

elif 순서가 중요해요!

✕ 잘못된 코드
# 잘못된 순서 if score >= 60: grade = "D" elif score >= 70: grade = "C" # 여기 절대 안 옴! elif score >= 80: grade = "B" # 여기도! else: grade = "F"
score=85면 첫 조건(>=60)에서 True → D등급!
✓ 올바른 코드
# 올바른 순서 (높은 점수부터) if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" else: grade = "F"
높은 조건부터 확인해야 올바른 결과
elif는 위에서부터 순서대로 확인하고, 처음 True인 것 하나만 실행해요. 그래서 범위가 넓은 조건을 아래에, 좁은 조건을 위에 놔야 해요.

용어 정리

if문
조건이 True일 때만 코드를 실행하는 제어문.
else
if의 조건이 False일 때 실행되는 블록.
elif
else + if의 줄임말. 여러 조건을 순서대로 검사.
들여쓰기
코드 앞에 공백을 넣는 것. 파이썬에서 코드 블록 범위를 결정.
중첩 if
if문 안에 또 다른 if문이 있는 것.
코드 블록
같은 들여쓰기 수준의 연속된 코드 묶음.
