Understanding Pointers & Memory Allocation

In C, direct memory manipulation is a powerful feature. Up until now, we've used stack memory for our variables. In this lesson, we will explore the heap and dynamic allocation.

psychology Key Concept: Malloc

The malloc() function allocates a block of memory of a specified size and returns a pointer of type void* to the first byte.

int* ptr = (int*) malloc(5 * sizeof(int));
